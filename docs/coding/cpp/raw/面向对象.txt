1.如果对象有且仅有一个单参数的构造函数可以支持：
type a = 10; <==> type a(10) <==> type a; a = 10;
若有多个单参数的构造函数仍支持自动转化：
接收type参数而给对应参数的类型m
识图返回type而给对应参数的类型
使用explicit关闭该特性(强制转换是仍可用type(m))

2.成员函数后加上const表明该函数不会修改成员的值，可以对const的变量使用该函数

3.设计含指针与动态内存的类时注意
a.构析函数
b.拷贝构造函数 const type&
c.重载为深拷贝的=号

3.

在初始化列表中使用数组时注意应使用大括号{}
classA::classA():arr{1, 2, 3}
