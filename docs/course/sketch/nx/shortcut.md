# NX 快捷键与常用设置
## 常用设置
### 取消自动尺寸
文件->实用工具->用户默认设置->草图->自动判断约束和尺寸->尺寸->取消勾选连续标注尺寸

### 设置图标/快捷键
菜单(最左侧, 导航栏上方)->工具->定制

## 特殊操作
### 参数化
工具->表达式
在建模完成后, 可以指定参数

### 删除约束
1. 右键曲线, 选择删除曲线约束
