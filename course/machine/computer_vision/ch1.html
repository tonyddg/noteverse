                                        默认图片使用 8 位色深, 即灰度值取值范围为 0 ~ 255

                                        # 灰度变换

                                        灰度变换即对图片的每一个像素的灰度 rrr 使用 s=T(r)s=T(r)s=T(r) 代替, 其中 T(r)T(r)T(r) 即灰度变换函数

                                        当 T(r)T(r)T(r) 超过灰度值的取值范围时, 小于 000 的灰度值使用 000 代替, 大于 255255255 的灰度值使用 255255255 代替

                                        直方图均衡化以及图像二值化也属于特殊的灰度变换, 关于其介绍见直方图分析

                                        # 线性变换

                                        当 T(r)T(r)T(r) 为一个关于灰度值 rrr 的线性函数时, 称为线性变换, 常用的线性变换有

                                        • 图像反转 T(r)=255−rT(r)=255 - rT(r)=255−r, 通过该变换可得到图片的底片
                                        • 灰度拉伸 T(r)=255brT(r)=\frac{255}{b}rT(r)=b255​r, 通过该函数, 可将灰度为 0∼b0\sim b0∼b 的像素点的灰度拉伸到 0∼2550\sim 2550∼255, 以此增强被拉伸部分的对比度, 使图像的灰度分布更平坦
                                        • 多段线性变换, 此时 T(r)T(r)T(r) 为一个分段函数, 通过该方法可拉伸 (斜率大于 111) 与压缩特定段的灰度, 强调特定段灰度像素点

                                        # 伽马变换

                                        人眼对于灰度的响应为对数规律, 即对灰度小的像素变化更敏感, 对灰度大的像素更不敏感
                                        因此可通过 T(r)=crγT(r)=cr^\gammaT(r)=crγ 的灰度变换修改图片的亮度

                                        伽马变换的灰度变换函数中 γ\gammaγ 为幂函数的指数, ccc 为归一化系数
                                        归一化系数与 γ\gammaγ 有关, 即对于任意的 γ\gammaγ, 都有 T(255)=255,T(0)=0T(255)=255,T(0)=0T(255)=255,T(0)=0

                                        由幂函数的特性, 对于 x=0∼1x=0\sim 1x=0∼1 范围内

                                        • 当 0<γ<10<\gamma<10<γ<1 曲线上凸, 较小的灰度值被拉伸, 较大的灰度值被压缩, 因此图像亮度增大, 且 γ\gammaγ 越接近 000 图片亮度越被加大

                                        # 图像卷积

                                        图像属于二维的离散系统, 以此定义图像的卷积运算为

                                        F(x,y)=f(x,y)⊗h(x,y)=∑i∑jf(x+i,y+j)h(i,j) F(x,y)=f(x,y)\otimes h(x,y)=\sum_{i}\sum_{j}f(x+i,y+j)h(i,j) F(x,y)=f(x,y)⊗h(x,y)=i∑​j∑​f(x+i,y+j)h(i,j)

                                        实际上, 与一维卷积相比, 图像的卷积运算缺少的翻转过程, 在数学上属于自相关计算, 但在图形学上为了计算方便认为以上运算即卷积运算 (或在图像卷积前翻转卷积核)

                                        其中 f(x,y)f(x,y)f(x,y) 为原图像, h(i,j)h(i,j)h(i,j) 则称为卷积核
                                        一般卷积核可表示为 2n+1×2n+12n+1\times 2n+12n+1×2n+1 的矩阵, 在卷积运算时的求和范围为 −n∼0∼n-n\sim 0\sim n−n∼0∼n, 即以卷积核的中心为锚点

                                        • 图像卷积的本质即遍历整个图像, 遍历时将卷积核的锚点与遍历点重合, 对应点相乘并相加, 以结果作为遍历点的卷积结果
                                        • 通过卷积运算, 原图像中与卷积核相似的区域将得到更大的卷积结果 (响应), 因此卷积的本质即搜索与强调图像中与卷积核相似的部分
                                        • 在边缘点上, 当卷积核覆盖了图像外的区域时, 一般可以忽略这些边缘点或将图像外区域的像素值视为 000
                                        • 通过选择合适的卷积核, 图像卷积可用于图像模糊 (滤波) 与图像锐化 (差分) 处理
                                          • 通常用于滤波的卷积核的元素和为 111 以保持图片的平均像素值
                                          • 通常用于差分的卷积核的元素和为 000 以去除图像的平均分量

                                        • 均值滤波会导致图像的轮廓模糊
                                        • 一般用于去除噪声用于二值化

                                        • 高斯滤波能较好去除白噪声

                                        # 中值滤波

                                        中值滤波不具有严格的卷积核, 中值滤波的在卷积时, 将以卷积范围内的中位数作为卷积结果

                                        • 去除噪声的同时，比较好地保留边缘
                                        • 能够有效去除脉冲噪声
                                        • 能消除孤立的亮度或暗度
                                        • 中值滤波为图像滤波最常用的方法

                                        # 直方图均衡化

                                        直方图均衡化为一种特殊的灰度变换, 其理论目标为使图像的灰度分布均匀, 每个灰度值拥有相同的像素个数, 此时图像的灰度值均匀分布具有最大的对比度

                                        对于均匀分布

                                        • 图像的各个灰度值应当具有相同的概率密度函数, 也即各个灰度值出现的频数相同
                                        • 图像的概率函数 ω(T)=P(T≥x)\omega(T)=P(T\ge x)ω(T)=P(T≥x) 为一条直线

                                        因此对于直方图均衡化中

                                        • 由于图片的离散性, 只能通过使图像的概率函数尽可能接近一条直线, 以达到类似均匀分布的效果
                                        • 但是对于实际图片, 一个灰度值对应了多个像素, 如果修改这写像素间的灰度值就, 图片将失真, 因此一组像素只能一起修改灰度值, 不能单独修改其中几个
                                        • 因此通过灰度变换完成直方图均衡化实质即需要找到一个合适的灰度变换函数 T(r)T(r)T(r) 或灰度变换的映射

                                        # 阈值分析

                                        理想情况下, 对于物体位于环境中时, 物体与环境部分的灰度都呈类正态分布, 因此图像的直方图呈双峰曲线
                                        通过寻找一个合适的阈值 TTT, 该阈值正好位于双峰中间的低谷, 将图像分割为背景与物体两个部分
                                        找到阈值后通常还会将小于阈值的像素值设为 000 大于阈值的像素设为 255255255 完成图像的分隔

                                        根据阈值分割出的两个图像区域 G0G_0G0​ 与 G1G_1G1​, 具有各自的直方图统计参数

                                        • 对于像素值 rrr 满足 r≤Tr\le Tr≤T 的部分 G0G_0G0​, 其统计参数 N0,S0,μ0,ω0N_0,S_0,\mu_0,\omega_0N0​,S0​,μ0​,ω0​ 与对应的统计函数求法相同

                                        T=μ0+μ12=S(T)N(T)+S−S(T)N−N(T)2 T=\frac{\mu_0+\mu_1}{2}=\frac{\frac{S(T)}{N(T)}+\frac{S-S(T)}{N-N(T)}}{2} T=2μ0​+μ1​​=2N(T)S(T)​+N−N(T)S−S(T)​​

                                        # 二值形态学

                                        二值形态学中, 一般以二值化的图片作为输入分析, 将 P=255P=255P=255 的像素点视为 111 (白色像素), 将 P=0P=0P=0 的像素点视为 000 (黑色像素)

                                        # 膨胀运算

                                        对于输入图片 AAA 与掩膜 (结构元) BBB 的膨胀运算表示如下

                                        [A⊕B](x,y)=⋁(i,j)∈BA(x+i,y+j)∧B(i,j) [A\oplus B](x,y)=\bigvee\limits_{(i,j)\in B}A(x+i,y+j)\land B(i,j) [A⊕B](x,y)=(i,j)∈B⋁​A(x+i,y+j)∧B(i,j)

                                        其中

                                        • 符号 ∨\lor∨ 表示或运算, ⋁\bigvee⋁ 则表示对之后的所有表达式间进行或运算, ∧\land∧ 表示与运算
                                        • 对以上数学运算的解释为对于点 P(x,y)P(x,y)P(x,y) 的运算
                                          • 首先将掩膜的锚点与点 PPP 对齐
                                          • 当对齐后, 当掩膜与原图存在任意一对对应点均为 111, 则该点的运算结果为 111
                                          • 当对齐后, 掩膜为 111 的点在原图中均为 000, 则该点的运算结果为 000

                                        由于进过运算后二值图中为 111 的区域一般会向外扩展, 因此称为膨胀

                                        # 腐蚀运算

                                        对于输入图片 AAA 与的掩膜 (结构元) BBB 的腐蚀运算表示如下

                                        [A⊖B](x,y)=⋀(i,j)∈BA(x+i,y+j)∧B(i,j) [A\ominus B](x,y)=\bigwedge\limits_{(i,j)\in B}A(x+i,y+j)\land B(i,j) [A⊖B](x,y)=(i,j)∈B⋀​A(x+i,y+j)∧B(i,j)

                                        其中

                                        • 符号 ∧\land∧ 表示与运算, ⋀\bigwedge⋀ 则表示对之后的所有表达式间进行与运算
                                        • 对以上数学运算的解释为对于点 P(x,y)P(x,y)P(x,y) 的运算
                                          • 首先将掩膜的锚点与点 PPP 对齐
                                          • 当对齐后, 当任一掩膜为 111 的点在原图的对应点不为 111 时, 则该点的运算结果为 000
                                          • 当对齐后, 掩膜为 111 的点在原图中均为 111, 则该点的运算结果为 111

                                        由于进过运算后二值图中为 111 的区域一般会向内收缩, 因此称为腐蚀

                                        # 其他形态学运算

                                        通过组合以上两个形态学运算可实现对图像的处理, 以下为常用的组合运算

                                        • 开运算 A∘B=(A⊖B)⊕BA\circ B=(A\ominus B)\oplus BA∘B=(A⊖B)⊕B
                                          • 开运算中, 先进行腐蚀再进行膨胀
                                          • 用于消除二值图片中细小的区域 (通过先腐蚀消除, 再使用膨胀还原)
                                          • 也可以平滑值为 111 区域的边缘
                                        • 闭运算 A∙B=(A⊕B)⊖BA\bullet B=(A\oplus B)\ominus BA∙B=(A⊕B)⊖B
                                          • 闭运算中, 先进行膨胀再进行腐蚀
                                          • 用于填充值为 111 区域的空洞 (通过先膨胀填充, 再使用腐蚀还原)
                                          • 也可以平滑值为 111 区域的边缘
                                        • 形态学边缘 (A⊕B)−(A⊖B)(A\oplus B)-(A\ominus B)(A⊕B)−(A⊖B)
