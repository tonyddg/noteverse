                                        # 凸轮

                                        # 凸轮传动特点

                                        1. 凸轮设计简单, 结构紧凑, 多用于自动与半自动机械
                                        2. 凸轮机构为高副, 压力较大容易磨损, 一般仅能传递运动

                                        # 凸轮运动规律

                                        # 凸轮基本运动规律

                                        • 将滚子在凸轮基圆上的位置作为基准, 凸轮在转动过程中, 滚子的位移称为推程 sss
                                        • 定义凸轮相对初始位置转动的角度为推程角, 满足 φ=∠BOB′\varphi=\angle BOB'φ=∠BOB′ (几何证明见后)
                                        • 通过函数 s=f(φ)s=f(\varphi)s=f(φ) 来表示凸轮的运动规律, 将 sss 上升段称为推程, 下降段称为回程, 主要研究这两端上凸轮的运动规律, 推程不变部分则称为休止
                                        • 在休止处或推/回程开始处, f(φ)=0/hf(\varphi)=0/hf(φ)=0/h, 为常数
                                        • 由于凸轮匀速转动 (ω=dφdt\omega=\frac{d\varphi}{dt}ω=dtdφ​), 因此滚子的有关运动量可以表示为 (不是直接等于 f(φ)f(\varphi)f(φ) 的导数, 还有系数 ωn\omega^nωn 不能漏)

                                        v=dsdt=ωdsdφ,  a=dvdt=ω2d2sdφ2,  j=dadt=ω3d3sdφ3 v=\frac{ds}{dt}=\omega\frac{ds}{d\varphi},\;a=\frac{dv}{dt}=\omega^2\frac{d^2s}{d\varphi^2},\;j=\frac{da}{dt}=\omega^3\frac{d^3s}{d\varphi^3} v=dtds​=ωdφds​,a=dtdv​=ω2dφ2d2s​,j=dtda​=ω3dφ3d3s​

                                        其中 jjj 为跃动度

                                        • 在休止处或推/回程开始处, f(φ)f(\varphi)f(φ) 的任意阶导数均为 000, 因此对于大部分运动规律, 在推/回程开始处的 v,a,jv,a,jv,a,j 等可能发生突变
                                        • 对于不同运动规律中系数的取值, 可通过以下条件求出推程下的状态

                                        f(0)=0f(Φ)=hdsidiφ∣φ=0/Φ=0(i=1,2,…,n) \begin{split} &f(0)=0\\ &f(\Phi)=h\\ &\frac{ds^i}{d^i\varphi}\bigg|_{\varphi=0/\Phi}=0 (i=1,2,\dots,n) \end{split}​f(0)=0f(Φ)=hdiφdsi​​φ=0/Φ​=0(i=1,2,…,n)​

                                        • 对于推程状态
                                          • nnn 的取值根据运动规律中第一个没有在起止位置发生突变的运动特征决定, 从小到大求导, 可能提前求出参数
                                          • 若回程的运动规律与推程一致, 则可带入 φ′=(Φ−φ)\varphi'=(\Phi-\varphi)φ′=(Φ−φ) 变换得到, 或修改初值条件
                                          • 对于混合规律的运动, 则还有运动特征连续条件, 即各个运动特征需要连续 (可能出现冲激函数)
                                        • 通常要求从动件的各项运动特征尽可能小且没有突变, 以此作为评价运动规律的基本标准

                                        # 多项式运动规律

                                        # 等速运动

                                        等速运动下, 运动规律满足形式

                                        f(φ)=c0+c1φ f(\varphi)=c_0+c_1\varphi f(φ)=c0​+c1​φ

                                        因此图像为一条直线

                                        1. 此时速度则为常数 v=ωc1v=\omega c_1v=ωc1​
                                        2. 在运动开始 / 结束时, 速度从 000 突变到 vvv, 因此加速度 aaa 在这些时刻存在冲激函数, 称为刚性冲击, 滚子将受到巨大的惯性力, 因此仅能用于极低速场合

                                        # 等加速等减速运动

                                        等加速度运动下, 运动规律满足

                                        f(φ)=c0+c1φ+c2φ2 f(\varphi)=c_0+c_1\varphi+c_2\varphi^2 f(φ)=c0​+c1​φ+c2​φ2

                                        1. 注意存在匀加速与匀减速两个部分, 因此 s,vs,vs,v 为分段函数, 并且通常关于 φ=Φ/2\varphi=\Phi/2φ=Φ/2 处对称 / 中心对称 (f(Φ/2)=h/2,f′(Φ/2)=ωvmaxf(\Phi/2)=h/2,f'(\Phi/2)=\omega v_{max}f(Φ/2)=h/2,f′(Φ/2)=ωvmax​)
                                        2. 等减速部分 (后半段) 可通过带入 f(φ)=h/2−s(Φ−φ)f(\varphi)=h/2-s(\Phi-\varphi)f(φ)=h/2−s(Φ−φ)
                                        3. 此时速度为两段折线, 并且没有在起止处发生突变
                                        4. 此时加速度发生共三次突变, 但没有冲激函数
                                        5. 在运动中由于跃动度 jjj 中存在冲激函数, 称为柔性冲击, 在低速下影响小, 但在高速下仍有较大的动载荷

                                        # 三角函数式运动规律

                                        # 余弦加速度运动

                                        由名称可得, 此运动规律下加速度为一个余弦函数, 并在 Φ\PhiΦ 内完成半个周期

                                        a=c1cos⁡(πΦφ) a=c_1\cos(\frac{\pi}{\Phi}\varphi) a=c1​cos(Φπ​φ)

                                        通过积分可得到运动规律的表达式满足

                                        s=c0cos⁡(πΦφ)+c1φ+c2 s=c_0\cos(\frac{\pi}{\Phi}\varphi)+c_1\varphi+c_2 s=c0​cos(Φπ​φ)+c1​φ+c2​

                                        注意到余弦加速度在开始与结束处依然存在突变, 因此跃动度中存在冲激函数, 存在柔性冲击

                                        # 正弦加速度运动

                                        由名称可得, 此运动规律下加速度为一个正弦函数, 并在 Φ\PhiΦ 内完成一个完整周期

                                        a=c1cos⁡(2πΦφ) a=c_1\cos(\frac{2\pi}{\Phi}\varphi) a=c1​cos(Φ2π​φ)

                                        通过积分可得到运动规律的表达式满足

                                        s=c0sin⁡(πΦφ)+c1φ+c2 s=c_0\sin(\frac{\pi}{\Phi}\varphi)+c_1\varphi+c_2 s=c0​sin(Φπ​φ)+c1​φ+c2​

                                        注意到正弦加速度在开始与结束处为 000, 因此跃动度没有冲激函数, 不存在刚性冲击与柔性冲击

                                        # 凸轮结构设计

                                        # 移动从动件凸轮的相对运动瞬心

                                        1. 根据三心定律, 可以确定推杆与凸轮相对运动的瞬心为接触点 AAA 的法线 n−nn-nn−n 与过铰链 OOO 且垂直于推杆的轴线的交点 PPP
                                        2. 在速度瞬心 PPP 上两机构的运动速度相同, 且推杆以 v2v_2v2​ 垂直运动, 凸轮以 ω1\omega_1ω1​ 绕 OOO 转动, 因此可得出 OPω1=v2OP\omega_1=v_2OPω1​=v2​
                                        3. 由于推杆的运动方向垂直, 凸轮通过接触力驱动推杆, 驱动力沿接触点的法线方向, 因此可做出压力角 α\alphaα 即接触点法线与推杆运动方向的夹角, 且有 ∠CAP=α\angle CAP=\alpha∠CAP=α
                                        4. 由于 OPOPOP 长度已知, 且其他长度几何关系确定, 即可通过三角函数得到压力角的表达式

                                        # 移动从动件凸轮偏置方位对压力角的影响

                                        1. 以减小从动件工作行程时的最大压力角为原则选择偏置
                                        2. 由于 ∠CAP=α\angle CAP=\alpha∠CAP=α, 因此当偏置点 CCC 与瞬心 PPP 同侧时, 压力角较小
                                        3. 若不好判断 CCC 与 PPP 相对于 OOO 的位置, 则可以寻找最大 / 最小压力角下的状态, 并在工作行程避开 / 通过这些状态
                                        4. 并且有几何特征可得, 偏置越大, 两侧压力角相差越大, 因此偏置也不可取得过大
                                        5. 通过控制凸轮转向或偏置方位来保证

                                        # 移动从动件凸轮的其他参数对运动的影响

                                        1. 增大基圆半径可以减小压力角, 但增大机构尺寸
                                        2. 当凸轮得推程大时, 压力角将减小
                                        3. 当推杆速度大时, 压力角将增大

                                        # 摆动从动件凸轮的压力角特点

                                        1. 对于摆动从动件, 当凸轮角速度 ω1\omega_1ω1​ 与摆动件角速度 ω2\omega_2ω2​ 异向时压力角较小, 相反则较大, 以相同原则确定凸轮转向
                                        2. 对于任意形状的摆件, 等效为连接接触点与转动铰链的杆

                                        # 尖顶从动件凸轮轮廓设计

                                        # 基本理论

                                        1. 通过固定凸轮, 而是旋转从动件来确定凸轮轮廓, 注意此时从动件的旋转方向应与凸轮转动方向相反
                                        2. 在固定凸轮旋转从动件时, 从动件与旋转中心的偏置关系始终不变, 因此从动件所在直线始终切于直径为偏置 eee 的圆周
                                        3. 由相切点 CCC, 圆心 OOO 与从动件上一点 BBB 组成三角形, 从动件旋转的本质为此三角形的旋转
                                        4. 由于旋转的特点, 因此角 ∠B1OB\angle B_1OB∠B1​OB 即推程角 φ\varphiφ
                                        5. 因此, 从基圆上的运动开始点作为 COCOCO 起始边, 根据推程 s(φ)s(\varphi)s(φ) 确定 BBB 位置, 反向旋转 φ\varphiφ 即可得到凸轮轮廓
                                        6. 对于刚化反转法, 重点即三角形 ΔOCB\Delta OCBΔOCB 的旋转
                                          1. OOO 点即凸轮旋转中心
                                          2. CCC 点即偏置, 始终保持 OC=eOC=eOC=e
                                          3. BBB 点即理论轮廓上的点
                                          4. 三角形中 ∠OCB=90∘\angle OCB=90^\circ∠OCB=90∘, 且 BCBCBC 始终为推杆的移动方向, 并且推杆的移动需要通过比较 BCBCBC 的长度来反映
                                          5. 由于三角形总是绕 OOO 旋转, 因此有角度关系 ∠B1OB2=∠C1OC2\angle B_1OB_2=\angle C_1OC_2∠B1​OB2​=∠C1​OC2​
                                          6. 对于已知状态 111 与理论轮廓上一点 B2B_2B2​, 可先求出 ∠B1OB2\angle B_1OB_2∠B1​OB2​, 再旋转 OC2OC_2OC2​ 相同角度确定 C2C_2C2​
                                          7. 对于已知初始状态 111 与推程 sss, 推程角 φ\varphiφ, 可在初始状态的 C0B0C_0B_0C0​B0​ 上取出 B1′B_1'B1′​, 旋状 OB1′OB_1'OB1′​ 得到 B1B_1B1​

                                        # 尖顶摆动从动件凸轮轮廓设计

                                        1. 注意摆动从动件初始状态下, 接触点在基圆上的行程开始点, 摆杆旋转圆心还需要通过摆杆长度确定

                                        # 滚子凸轮轮廓设计

                                        1. 对于存在滚子的情况, 对于几何法, 可直接令推程 s′=s±rrs'=s\pm r_rs′=s±rr​, 得到近似轮廓线 (对于一般凸轮取 −-−, 外包凸轮则取 +++)

                                        # 偏置凸轮角位移

                                        对于存在偏置的凸轮, 其角位移不是两次接触点 A,DA,DA,D 与圆心 OOO 的夹角, 而是新位置上旧接触点关于 OOO 的圆投影 A′A'A′ 与旧接触点 AAA 关于圆心 OOO 的夹角
