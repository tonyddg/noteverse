                                                                  # 传动特性

                                                                  除有说明, 均针对渐开线齿轮讨论

                                                                  # 一般齿轮传动特性

                                                                  1. 此特性对任何形状的齿轮均成立
                                                                  2. 根据三心定律, 两齿轮对于机架的瞬心为各自圆心, 两齿轮之间的瞬心在其圆心连线上
                                                                  3. 对于啮合点 KKK 认为是分别相对于齿轮圆心转动, 因此对于两个齿轮上的 vkv_kvk​ 分别垂直于 O1KO_1KO1​K 与 O2KO_2KO2​K
                                                                  4. 点 KKK 作为两个物体的接触点, 两个物体上 vkv_kvk​ 沿接触点公法线方向的分量必定相同, 因此可得齿轮的瞬心于其接触点的公法线上, 因此公法线与中心线交点即齿轮瞬心 PPP
                                                                  5. 对于瞬心 PPP, 两齿轮上在瞬心位置的速度大小方向均相同, 有 vp=O1Pω1=O2Pω2v_p=O_1P\omega_1=O_2P\omega_2vp​=O1​Pω1​=O2​Pω2​
                                                                  6. 定义传动比 i12=ω1ω2=O2PO1Pi_{12}=\frac{\omega_1}{\omega_2}=\frac{O_2P}{O_1P}i12​=ω2​ω1​​=O1​PO2​P​, 因此, 只要 PPP 点位置不变就可以保证传动比不变

                                                                  # 渐开线齿轮的恒定传动比

                                                                  1. 渐开线齿轮的啮合点即两齿轮渐开线的切点 KKK
                                                                  2. 根据渐开线性质, KKK 分别位于两齿轮基圆的一条切线上, 并且两条切线共线, 为 KKK 点处的公法线
                                                                  3. 对于两个位置固定的基圆与固定方向的齿面上的任意啮合点, 公法线唯一, 并且即两个基圆的一条公切线 N1N2N_1N_2N1​N2​
                                                                  4. 因此渐开线齿轮满足传动比不变的条件, 具有稳定的传动比, 并且啮合点只会出现在公切线上, 因此也称为啮合线

                                                                  # 中心距可变性

                                                                  1. 当中心距变化, 两个基圆的公切线随之变化, 因此 PPP 点的位置并不固定
                                                                  2. 任意中心距始终有 ΔO2N2P∼ΔO1N1P\Delta O_2N_2P\sim \Delta O_1N_1PΔO2​N2​P∼ΔO1​N1​P
                                                                  3. 因此传动比 i12=ω1ω2=O2PO1P=r2r1i_{12}=\frac{\omega_1}{\omega_2}=\frac{O_2P}{O_1P}=\frac{r_2}{r_1}i12​=ω2​ω1​​=O1​PO2​P​=r1​r2​​, 对于任何中心距, 当基圆不变, 传动比不变

                                                                  # 啮合角可变性

                                                                  1. 定义啮合齿轮的节圆的公切线 (即中心线过 PPP 的垂线) 与接触点的公法线的夹角为啮合角 α′\alpha'α′, 啮合角即节线上的压力角
                                                                  2. 根据角度关系可得 ∠N1PO1=∠N2PO2=α′\angle N_1PO_1=\angle N_2PO_2=\alpha'∠N1​PO1​=∠N2​PO2​=α′, 即渐开线在瞬心 PPP 点的压力角 (不是分度圆)
                                                                  3. 可得啮合角满足 cos⁡α′=rb1O1P=rb2O2P\cos\alpha'=\frac{r_{b1}}{O_1P}=\frac{r_{b2}}{O_2P}cosα′=O1​Prb1​​=O2​Prb2​​, 由于 O1PO_1PO1​P 与 O2PO_2PO2​P 随中心距变化, 因此啮合角也仅随中心距变化
                                                                  4. 对于标准齿轮的啮合, 分度圆与压力角重合, 数值上 α′=α\alpha'=\alphaα′=α, 并且对于两齿轮啮合角相同

                                                                  1. 当前一对轮齿于 K1K_1K1​ 点接触时, 后一对轮齿的渐开线必须在啮合线上的另一点接触 (不是实际接触), 否则将产生干涉或传动比不连续

                                                                  # 连续传动条件

                                                                  2. 当两个齿轮啮合时, 始终存在接触点, 保证动力始终能通过接触点传输, 则称齿轮能够连续传动

                                                                  1. 正变位
                                                                    • 正变位中, 齿根圆靠外 (范成齿顶线远离圆心), 因此实际啮合线较短, 需要校核重合度 ε\varepsilonε

                                                                  2. 负变位
                                                                    • 负变位中, 范成法齿顶线靠近圆心, 需要校核是否发生根切

                                                                  1. 模数越大, 齿轮越厚大, 齿轮抗弯强度越大
                                                                  2. 模数越小, 重合度越大, 传动越平稳, 在满足抗弯强度的条件下, 模数越小越好

                                                                  # 齿数选择

                                                                  1. 减小机构尺寸, 需要选择较小的齿数
                                                                  2. 增大齿轮传动的平稳性, 需要选择较多齿数
                                                                  3. 齿数最好选择不可互约的质数, 以获得更好的传动效果

                                                                  # 啮合条件

                                                                  # 当量齿数

                                                                  1. 使用仿形法铣齿时与斜齿啮合 (范成法) 时, 主要受力方向平行于法面, 为了研究这些问题, 首先需要研究以法面为截面上的齿形
                                                                  2. 由于法面实际为螺旋面 (圆柱展开面上的法面为直线) , 无法直接研究, 通过寻找一个齿形与法面齿形相同的直圆柱齿轮来等效替代

                                                                  4. 可得分度圆半径即法面与圆柱面相交得到的螺旋线的曲率
                                                                    1. 取斜齿轮分度圆柱面上的点 ccc, 法面螺旋面在 ccc 处的位面为一斜截面, 截面从分度圆柱面上截出一椭圆
                                                                    2. 可得, ccc 点位于椭圆短半轴端点上, 螺旋线在 ccc 点的曲率等于 ccc 点在此椭圆上的曲率

                                                                  # 收缩顶隙传动

                                                                  • 当啮合两圆锥的齿顶圆锥, 分度圆锥, 节圆锥, 齿根圆锥, 基圆锥的锥顶均重合与一点时, 定义为收缩顶隙传动
                                                                  • 此时两齿轮的顶隙线最终也将交于一点, 因此顶隙从大端到小端不断缩小, 将影响齿轮强度
                                                                  • 齿根圆与齿顶圆的锥顶角满足 (注意啮合两齿的齿顶高, 分度圆锥顶角等参数不同)

                                                                  # 等顶隙传动

                                                                  • 当两啮合圆锥齿轮的顶隙不变时, 定义为等顶隙传动
                                                                  • 由于顶隙不变, 因此两啮合圆锥之间的齿顶锥与齿根锥投影母线平行
                                                                  • 通过改变两齿轮的齿顶圆锥来达到这一效果, 齿顶圆锥的锥度减小, 使小端上的齿顶圆减小, 防止小端齿廓变尖并且降低齿高增加了齿轮的强度
