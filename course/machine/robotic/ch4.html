                                  # 操作臂雅可比矩阵的意义

                                  在操作臂运动学中, 通常将关节空间中的关节变量矢量 q\bm{q}q 作为变量矢量, 将操作空间中的元素作为矢量函数, 将两者的雅可比矩阵称为操作臂雅可比矩阵
                                  由雅可比矩阵的基本定义可知, 操作比雅可比矩阵描述了关节变量相对时间的导数 (即关节转动或平动速度) 相对于操作臂末端执行器的位姿相对时间的变化 (对于三维空间, 一般将末端执行器视为刚体, 使用末端执行器的速度线矢量) 的映射, 满足 (DDD 为微分运动矢量)

                                  D=J(q)dq→V=[vω]=J(q)q˙ D=\bm{J}(\bm{q})\mathrm{d}\bm{q}\to V=\begin{bmatrix}\bm{v}\\\bm{\omega}\end{bmatrix}=\bm{J}(\bm{q})\dot{\bm{q}} D=J(q)dq→V=[vω​]=J(q)q˙​

                                  当雅可比矩阵 J(q)\bm{J}(\bm{q})J(q) 行满秩时, 表明总是存在一组特定的关节速度 q˙\dot{\bm{q}}q˙​ 使当前操作臂的末端执行器以任意速度运动
                                  反之, 当行不满秩时, 操作臂末端只能以特定形式的速度运行, 称此时的操作臂 q\bm{q}q 处于奇异状态

                                  综上可知

                                  • 操作臂的雅可比矩阵有形状 m×nm\times nm×n, 其中
                                    • mmm 为操作空间的维数, 对于三维空间一般为 666
                                    • nnn 为操作臂的关节数, 因此三维空间中的雅可比矩阵行数一定为 666, 但列数不固定
                                  • 以三维空间为例, 当操作臂的关节数 nnn 少于 666 时, 雅可比矩阵行必定不满秩, 因此总是处于奇异状态
                                  • 对于三维空间的 666 关节操作臂, 此时雅可比矩阵为 6×66\times 66×6 的方阵, 因此可通过行列式 ∣J(q)∣=0\begin{vmatrix}\bm{J}(\bm{q})\end{vmatrix}=0​J(q)​​=0 判断操作臂当前是否处于奇异状态
                                  • 对于不同的观察坐标系, 速度线矢量不同, 同样的, 这些不同观察坐标系下对应的雅可比矩阵也不同
                                    通常情况下, 总是将速度线矢量 VVV 的原点固连在手腕坐标系原点 pOn\bm{p}_{O_n}pOn​​ 上, 但坐标轴的方向可能不同
                                    • 使用 J(q)\bm{J}(\bm{q})J(q) 表示基座标 {B}\{\bm{B}\}{B} 下的雅可比矩阵, 此时得到的速度线矢量的观察坐标系的坐标轴平行于基坐标系 (注意原点依然在手腕坐标系原点 pOn\bm{p}_{O_n}pOn​​ 上), 定义该坐标系为 {W0}\{\bm{W_0}\}{W0​}
                                    • 使用 TJ(q){}^{T}\bm{J}(\bm{q})TJ(q) 表示手腕坐标系 {W}\{\bm{W}\}{W} (即末端连杆 {n}\{n\}{n}) 下的雅可比矩阵, 此时得到的速度线矢量以工具坐标系 {W}\{\bm{W}\}{W} 为观察坐标系

                                  # 雅可比矩阵的四个基本子空间

                                  通过雅可比矩阵的四个基本子空间中, 行空间与零空间为关节速度空间 Rn\R^{n}Rn 的子空间, 列空间与左零空间为操作速度空间 Rm=6\R^{m=6}Rm=6 的子空间
                                  雅可比矩阵子空间的形状能一定程度上反应操作臂当前状态 q\bm{q}q 下的运动特点

                                  # 雅可比矩阵的列空间

                                  对于雅可比矩阵的列空间满足 (教材上也称为域空间 R[J(q)]R[\bm{J}(\bm{q})]R[J(q)])

                                  C[J(q)]={V∣V=J(q)q˙} C[\bm{J}(\bm{q})]=\{V|V=\bm{J}(\bm{q})\dot{\bm{q}}\} C[J(q)]={V∣V=J(q)q˙​}

                                  因此列空间包含了操作臂在当前状态下, 所能达到的所有速度 VVV
                                  易得列空间维数即雅可比矩阵的秩 rrr, 满足 dim(C[J(q)])=rdim(C[\bm{J}(\bm{q})])=rdim(C[J(q)])=r

                                  列空间的维数反映了操作臂当前状态下所具有的自由度

                                  • 当雅可比矩阵行满秩时, r=m=6r=m=6r=m=6, 列空间张满整个操作速度空间 R6\R^{6}R6
                                    此时对于任意指定的末端执行器速度 VVV, 总是存在至少一种关节速度组合 q˙\dot{\bm{q}}q˙​ 可得达到该速度
                                  • 当行不满秩时 r<m=6r<m=6r<m=6, 操作臂则处于奇异状态, 存在无法达到的速度

                                  # 雅可比矩阵的零空间

                                  对于雅可比矩阵的零空间满足

                                  N[J(q)]={q˙∣0=J(q)q˙} N[\bm{J}(\bm{q})]=\{\dot{\bm{q}}|\bm{0}=\bm{J}(\bm{q})\dot{\bm{q}}\} N[J(q)]={q˙​∣0=J(q)q˙​}

                                  因此零空间包含了一系列关节速度的组合 q˙\dot{\bm{q}}q˙​, 这些速度组合对于末端执行器的速度没有任何效果
                                  易得零空间维数满足 dim(N[J(q)])=n−rdim(N[\bm{J}(\bm{q})])=n-rdim(N[J(q)])=n−r

                                  当行满秩时, 零空间的维数反映了操作臂冗余度, 冗余的自由度可用于躲避障碍

                                  • 对于六关节六自由度操作臂, 显然其冗余度 dim(N[J(q)])=0dim(N[\bm{J}(\bm{q})])=0dim(N[J(q)])=0, 无法实现避障, 只能在开阔空间中移动
                                  • 对于七关节以上的操作臂, 其具有额外冗余度用于避障, 反解具有无限多的可能性

                                  # 雅可比矩阵的左零空间

                                  对于雅可比矩阵的左零空间满足

                                  N[JT(q)]={F∣0=JT(q)F} N[\bm{J}^T(\bm{q})]=\{F|\bm{0}=\bm{J}^T(\bm{q})F\} N[JT(q)]={F∣0=JT(q)F}

                                  因此零空间包含了一系列外载荷 FFF, 当末端执行器受到该载荷时, 关节的约束力几乎为 000, 载荷由机器人的基座承担
                                  易得左零空间维数满足 dim(N[J(q)])=m−rdim(N[\bm{J}(\bm{q})])=m-rdim(N[J(q)])=m−r

                                  当左零空间维数不为 000 时, 也即行不满秩, 操作臂处于奇异状态
                                  因此奇异状态下, 操作臂除了存在无法到达的速度 V′V'V′, 还存在可完全通过基座承担的外载荷 FFF
                                  由于列空间与左零空间互为正交补, 因此奇异状态下, 操作臂所能达到的速度 VVV 与可通过基座承担的外载荷 FFF 满足 VTF=0V^TF=0VTF=0
