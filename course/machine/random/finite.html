                                参考教程 现代设计方法实用教程 孙新民等编著

                                # 单元与节点

                                有限元分析中将结构划分为一系列单元与节点

                                # 单元与节点的特性

                                节点具有位移与受力

                                • 节点上各方向的位移与受力均使用 n×1n\times 1n×1 的向量表示, 其中 nnn 为节点的自由度 (不是维度)
                                • 通常节点即表示物体上的一个实际点 (三维) 或连杆之间的铰链 (平面刚架)
                                • 对于固定支座, 可视为一个无法位移的节点
                                • 注意, 节点只是一个依附在物体上的几何点

                                单元即系统种分析的最小结构, 具有形状, 弹性模量等物理系数

                                • 单元之间通过节点连接, 一般一个单元上的节点数固定
                                • 认为单元仅接收来自节点的反力与变形, 除此之外单元没有受到其他外界作用
                                • 通常单元即表示物体上的一个微小物体 (三维) 或具有弹性的连杆 (平面刚架)
                                • 单元具有方向性与单元坐标系, 需要在划分单元时注意

                                # 单元与节点的编码

                                • 单元一般使用字母编码
                                  • 一般在上标加上括号使用单元编码, 可用于表示单元的物理性质如 A(e)A^{(e)}A(e) 等
                                  • 在矩阵中使用时, 可在矩阵的上标标上 (e)(e)(e), 表示矩阵中所有元素都具有此上标
                                • 节点一般使用数字编码, 且具有两种编码方式
                                  • 局部码, 表示该节点为单元 aaa 中的第 iii 各节点, 则有编码 i(a)i^{(a)}i(a), 对于杆单元, 一般认为杆单元的方向为节点 1(a)1^{(a)}1(a) 指向 2(a)2^{(a)}2(a)
                                  • 全局码, 使用一个数字 iii 唯一表示该节点在物体上的编码
                                  • 因此一个节点可能具有多个局部码, 但仅有一个全局码, 且一般默认使用全局码表示节点
                                  • 一般在下标中使用节点编码

                                # 有关符号约定汇总

                                • 对于单元刚度矩阵
                                  • 使用 [k](e)[\bm{k}]^{(e)}[k](e) 表示单元 eee 在单元坐标系下的刚度矩阵
                                  • 使用 K(e)\bm{K}^{(e)}K(e) 表示单元 eee 在全局坐标系下的刚度矩阵
                                • 对于刚度系数 (对于多维情况一般为矩阵, 其中节点编号使用全局码)
                                  • 使用 [k]ij(e)[\bm{k}]_{ij}^{(e)}[k]ij(e)​ 表示单元 eee 下, 以单元坐标观察的节点 iii 到 jjj 方向的刚度系数
                                  • 使用 Kij(e)\bm{K}_{ij}^{(e)}Kij(e)​ 表示单元 eee 下, 以全局坐标观察的节点 iii 到 jjj 方向的刚度系数
                                  • 当 i≠ji\neq ji=j 刚度矩阵表示当节点 iii 移动单位距离时, 节点 jjj 受到的外力
                                  • 当 i=ji=ji=j 则表示其他节点固定, 驱动节点 iii 移动单位距离所需的外力
                                • 对于节点参数 (一般为 n×1n\times 1n×1 的向量, 其中节点编号使用全局码)
                                  • 使用 fi(e)\bm{f}_i^{(e)}fi(e)​ 表示节点 iii 在单元坐标系下的节点受力参数, 即单元间的内力
                                  • 使用 Fi\bm{F}_iFi​ 表示的是全局坐标系下节点 iii 上的外载荷
                                  • 使用 Φi\bm{\Phi}_iΦi​ 表示节点 iii 在全局坐标系下的位移参数, 在单元坐标下则为 ϕi(e)\bm{\phi}_i^{(e)}ϕi(e)​
                                  • 对于多个节点参数堆叠而成的参数, 使用多下标表示, 如节点 i,ji, ji,j 的位移参数为 ϕi,j(e)=[ϕi,jTϕi,jT](e)T\bm{\phi}_{i,j}^{(e)}=\begin{bmatrix}\bm{\phi}_{i,j}^T&\bm{\phi}_{i,j}^T\end{bmatrix}^{(e)T}ϕi,j(e)​=[ϕi,jT​​ϕi,jT​​](e)T
                                • 对于全局刚度矩阵
                                  • 使用 K\bm{K}K 表示全局刚度矩阵
                                  • 使用 Kij\bm{K}_{ij}Kij​ 表示节点 iii 到 jjj 方向的全局刚度系数
                                • 使用 T(e)\bm{T}^{(e)}T(e) 表示全局坐标系映射到单元坐标系的坐标转换矩阵

                                # 单元与节点特性

                                # 一维杆件的单元

                                • 一维杆件的单元通常为具有弹性的一维杆, 或着一维弹簧
                                • 单元具有刚度属性 k(e)k^{(e)}k(e), 对于一维杆则有 k(e)=E(e)A(e)L(e)k^{(e)}=\frac{E^{(e)}A^{(e)}}{L^{(e)}}k(e)=L(e)E(e)A(e)​
                                • 一个一维杆单元与两个节点连接
                                • 单元的坐标系只有一个方向, 且通常与世界坐标系平行
                                  此时所有节点的坐标系同向, 因此不区分单元坐标系与世界坐标系

                                # 一维杆件的节点

                                • 一维杆件的节点通常为两个杆的连接处, 或者单平动自由度的质点如小车
                                • 一维杆件的节点具有 xxx 方向的一个自由度
                                • 节点的受力参数为标量 FiF_iFi​, 位移参数为标量 Φi\Phi_iΦi​, 以节点坐标系正方向为正
                                • 与单元关联的力参数向量写为 Fi,j=[FiFj]T\bm{F}_{i,j}=\begin{bmatrix}F_{i}&F_{j}\end{bmatrix}^{T}Fi,j​=[Fi​​Fj​​]T
                                • 与单元关联的位移参数向量写为 Φi,j=[ΦiΦj]T\bm{\Phi}_{i,j}=\begin{bmatrix}\Phi_{i}&\Phi_{j}\end{bmatrix}^{T}Φi,j​=[Φi​​Φj​​]T

                                # 节点间的刚度矩阵

                                其中的 ki,j(e)k_{i,j}^{(e)}ki,j(e)​ 即节点间的刚度矩阵 (系数), 满足

                                • 认为节点 iii 的一个方向发生了单位位移, 除此外的其他节点固定, 除 eee 外的其他单元为刚体, 且以单元坐标系为位移与受力参数的观察坐标系
                                • 当 i=ji=ji=j 表示在以上约束下驱动节点 iii 移动单位距离所需的外力
                                • 当 i≠ji\neq ji=j 表示此时单元在节点 jjj 受到的约束反力
                                • 对于包含多个方向的位移参数, 方向 mmm 的单位位移产生的反力代表了矩阵 [k]ij(e)[\bm{k}]_{ij}^{(e)}[k]ij(e)​ 的第 mmm 列 (该条用于具有多个位移参数的情况, 对于一维杆件不需要)

                                上述讨论的为单元坐标系下的节点间的刚度矩阵 [k]i,j(e)[\bm{k}]_{i,j}^{(e)}[k]i,j(e)​
                                在全局坐标系下有 Ki,j(e)\bm{K}_{i,j}^{(e)}Ki,j(e)​, 性质相同

                                # 一维杆件的总体刚度矩阵

                                由于没有坐标系转换, 因此 [k](e)=K(e)[\bm{k}]^{(e)}=\bm{K}^{(e)}[k](e)=K(e)

                                与单元坐标系下的单元刚度矩阵的集成类似, 全局坐标系下的单元刚度矩阵也可进行分解, 有

                                [k](e)=K(e)=[Ki,iKj,iKi,jKj,j](e) [\bm{k}]^{(e)}=\bm{K}^{(e)}=\begin{bmatrix} K_{i,i}&K_{j,i}\\ K_{i,j}&K_{j,j}\\ \end{bmatrix}^{(e)}[k](e)=K(e)=[Ki,i​Ki,j​​Kj,i​Kj,j​​](e)

                                根据叠加法, 假设其他单元均为刚体, 则此时的全局刚度矩阵除 (i,i),(i,j),(j,i),(j,j)(i,i),(i,j),(j,i),(j,j)(i,i),(i,j),(j,i),(j,j) 处的元素与对应的节点间单元刚度外, 均为 000, 由此可通过单元刚度矩阵进行集成得到全局刚度矩阵

                                首先定义节点 iii 到节点 jjj 的刚度矩阵 (系数)满足

                                Ki,j=∑单元Ki,j(e) K_{i,j}=\sum_{\text{单元}} \bm{K}_{i,j}^{(e)} Ki,j​=单元∑​Ki,j(e)​

                                最后将各个全局坐标系下的所有 mmm 个节点间刚度矩阵进行集成, 即等效为将所有单元单独作用的刚度矩阵进行叠加, 可得到全局刚度矩阵满足

                                K=[K1,1…K1,m⋮⋱⋮Km,1…Km,m] \bm{K}=\begin{bmatrix} K_{1,1}&\dots&K_{1,m}\\ \vdots&\ddots&\vdots\\ K_{m,1}&\dots&K_{m,m} \end{bmatrix}K=​K1,1​⋮Km,1​​…⋱…​K1,m​⋮Km,m​​​

                                # 求解处理

                                最终求解即通过全局位移平衡方程, 求出系统各个节点的受力与位移

                                KΦ=F \bm{K}\bm{\Phi}=\bm{F} KΦ=F

                                根据受力平衡易得单元刚度矩阵 [k](e)[\bm{k}]^{(e)}[k](e) 与总体刚度矩阵 K\bm{K}K 必定为不可逆的对称矩阵 (该性质对任何单元均成立)
                                因此仅当有至少一个节点被约束时, 才能求解方程

                                # 单元划分方法

                                • 通常将杆件中具有相同属性的部分划分为一个单元
                                • 当杆件的属性连续变化时, 则只能划分有限个单元进行离散化, 划分的越密精度越高
                                • 对于系统的约束点, 受载点通常也作为一个节点, 如果位于杆件中间时则将以此为界划分单元

                                # 约束与载荷的处理

                                • 对于外载荷 F\bm{F}F
                                  • 当节点没有受力也没有受约束时, 对应的外载荷 Fi=0F_i=0Fi​=0
                                  • 当节点没有受约束但具有外载荷 PPP 作用时, 对应的外载荷 Fi=PF_i=PFi​=P
                                    (注意外载荷正方向是否与全局坐标系相同)
                                  • 当节点受约束时, 节点的外载荷未知
                                • 对于被约束的节点
                                  • 节点被约束自由度下的位移参数为约束变形 Φi=u\Phi_i=uΦi​=u, 对于固定端约束 u=0u=0u=0
                                  • 约束位移参数对应的外载荷未知
                                  • 通常使用以下方法处理
                                    • 假设被约束的全局位移参数为第 mmm 个参数
                                    • 令平衡方程中的刚度矩阵 K\bm{K}K 的第 mmm 行与第 mmm 列整列设为 000, 但 (m,m)(m,m)(m,m) 元素为 111
                                    • 令对应第 mmm 个全局外载荷参数为 uuu (即约束量)
                                    • 解出位移参数 Φ\bm{\Phi}Φ 后代回刚度矩阵 K\bm{K}K 得到约束反力
                                    • 相当于去除约束参数的影响, 通过删除特定行与列再补充等式可得到相同效果

                                # 平面刚架

                                用于由可视为一维直线的杆件 (单个杆内截面, 材料性质不变) 之间, 使用固连约束 (刚架) 或铰链约束 (桁架) 链接为的平面结构

                                # 平面刚架单元与节点特性

                                # 平面刚架的单元

                                • 平面刚架单元通常为具有弹性的一维梁, 因此也称为梁单元
                                • 单元具有属性: 弹性模量 E(e)E^{(e)}E(e), 面积 A(e)A^{(e)}A(e), 长度 L(e)L^{(e)}L(e), 截面矩 J(e)J^{(e)}J(e)
                                • 一个梁单元在两端与两个节点连接
                                • 平面刚架单元的坐标系具有 x,yx,yx,y 方向与一个平面旋转, 具体见平面刚架单元的坐标系介绍
                                  • 同样世界坐标系也具有 x,yx,yx,y 方向, 但不一定与单元坐标平行
                                  • 通常所有坐标系均以逆时针旋转为旋转的正方向, 因此不做区分

                                # 平面刚架的节点

                                • 平面刚架的节点通常为梁之间的固连约束 (不是铰链)
                                • 平面刚架的节点具有 x,yx,yx,y 方向与平面旋转共三个自由度
                                • 节点的力参数为向量
                                  • 力参数包含 x,yx,yx,y 方向的力以及平面力矩
                                  • 全局坐标系下的力参数写为 Fi=[FxiFyiMi]\bm{F}_i=\begin{bmatrix}F_{xi}&F_{yi}&M_i\end{bmatrix}Fi​=[Fxi​​Fyi​​Mi​​]
                                  • 单元坐标系下的力参数写为 fi(e)=[TiQiMi](e)\bm{f}_i^{(e)}=\begin{bmatrix}T_{i}&Q_{i}&M_i\end{bmatrix}^{(e)}fi(e)​=[Ti​​Qi​​Mi​​](e) (字母分别代表拉力, 剪力与弯矩)
                                  • 与单元关联的力参数向量写为 fi,j(e)=[fiTfjT](e)T\bm{f}_{i,j}^{(e)}=\begin{bmatrix}\bm{f}_{i}^T&\bm{f}_{j}^T\end{bmatrix}^{(e)T}fi,j(e)​=[fiT​​fjT​​](e)T
                                • 节点位移参数为标量

                                对于作用在单元上的载荷, 由于单元不能承受载荷, 因此需要使用虚功原理将单元载荷移动到单元节点上, 等效为作用在节点上的载荷
                                以下为两种常见情况的等效方法 (对于 yyy 方向结论类似)

                                • 作用在单元一边上任意点的 xxx 方向集中力 QQQ 可等效为

                                # 总体刚度矩阵的存储

                                对于任何有限元系统, 总体刚度矩阵具有对称性, 稀疏性, 奇异性
                                为此, 可使用特定方法存储刚度矩阵, 以减小内存空间的消耗

                                根据节点间刚度矩阵可知, 仅当两个节点 i,ji,ji,j 在至少同一个单元上 (相邻) 时, 节点间刚度矩阵 Ki,j≠0\bm{K}_{i,j}\neq 0Ki,j​=0

                                NB=(相邻节点编码最大差值+1)×节点自由度 N_B=(\text{相邻节点编码最大差值}+1)\times\text{节点自由度} NB​=(相邻节点编码最大差值+1)×节点自由度
