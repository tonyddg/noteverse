                                            # 特征方程与稳态响应

                                            系统传递函数的分母代表了系统的特征方程, 假设特征方程的根为 λi=ai+bij\lambda_i=a_{i}+b_{i}jλi​=ai​+bi​j
                                            则系统有自由稳态响应 (与输入无关的稳态响应) xo=∑i=1nAieλitx_o=\sum_{i=1}^{n}A_ie^{\lambda_{i}t}xo​=∑i=1n​Ai​eλi​t

                                            因此仅当所有特征根的实部 ai<0a_{i}<0ai​<0 才能保证 xox_oxo​ 快速衰减, 仅留下与输入有关的强迫相应
                                            此外认为 λi=0\lambda_i=0λi​=0 时, 系统的稳态响应为常数, 也属于稳定

                                            综上所述, 系统稳定的条件为特征根 λi=0\lambda_i=0λi​=0 或特征根的实部 ai<0a_i<0ai​<0
                                            但出现不满足以上条件的特征根, 则系统临界稳定或不稳定

                                            详细见书 P155

                                            # 系统稳定的必要条件

                                            习惯上处理特征方程时, 通常会对其乘以 −1-1−1 以保证尽量多的系数为正数 (以下默认处理)

                                            易得系统稳定的必要条件为 (证明见书 P157)

                                            • 特征方程各项系数 aia_iai​ 都不等于零
                                            • 特征方程各项系数 aia_iai​ 的符号相同 (按习惯处理后, 各项系数均为正数)

                                            # Bode 判据

                                            根据 Nyquist 曲线与 Bode 图幅频 / 相频曲线之间的联系, 推广可得到 Bode 判据

                                            # 基于正负穿越概念的 Nyquist 判据

                                            # 正负穿越判断

                                            沿 ω\omegaω 增大的方向, 将以下情况记为一次穿越

                                            • 在 ω=0∼∞\omega=0\sim\inftyω=0∼∞ 的开环 Nyquist 曲线上穿过 (−1,0)(-1,0)(−1,0) 点左侧的实轴

                                              • 穿越方向为自下而上时, 记为负穿越
                                              • 来自 (−∞,0)(-\infty,0)(−∞,0) 的渐近线向下移动时 (通常为 ω=0+\omega=0^+ω=0+ 的对应点), 记为正半次穿越, 反之则为负半次穿越

                                              • 穿越方向为自上而下时, 记为负穿越
                                              • 来自 φ=−180°\varphi=-180\degreeφ=−180° 的渐近线向上移动时 (通常为 ω=0+\omega=0^+ω=0+ 的对应点), 记为正半次穿越, 反之则为负半次穿越

                                            # 正负穿越的使用

                                            • 正穿越次数减去负穿越次数再乘以 222 即开环 NyqistNyqistNyqist 曲线绕 (−1,0)(-1,0)(−1,0) 点逆时针旋转次数, 可以此判断系统是否稳定
