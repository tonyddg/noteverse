                                  # 基准类型

                                  # 设计基准

                                  即在设计图上采用的基准
                                  对于几何公差, 即图纸上规定的公差基准
                                  对于定位尺寸的尺寸界限, 一侧为被约束的平面, 另一侧即该平面的设计基准
                                  对于直径尺寸, 该直径所过的圆心所在的中心线即设计基准

                                  # 工序基准

                                  即在工序图上采用的基准, 是加工工序的起点
                                  获取方法与设计基准相同

                                  # 定位基准

                                  即被夹具确定的几何要素
                                  如 V 型块确定接触圆弧的圆心, 点接触确定接触点的位置

                                  # 定位误差分析

                                  同批零件在夹具中定位时, 工序基准位置在工序尺寸方向或沿加工要求方向上的最大变动量 因此在分析后, 要将误差投影到加工方向才能得到要求的定位误差
                                  通过比较特定尺寸取上下极限的方法确定误差值

                                  定位误差根据成因可分为基准不重合导致的定位误差 Δbc\Delta_{bc}Δbc​ 与基准位置误差导致的定位误差 Δjw\Delta_{jw}Δjw​

                                  # 基准不重合导致的定位误差

                                  当定位基准与工序基准不重合时, 两基准相对位置的最大变动量为 Δbc\Delta_{bc}Δbc​
                                  误差的大小即定位基准与工序基准距离的公差

                                  对于图示左侧的情况, 两基准距离为半径 d2\frac{d}{2}2d​, 因此有定位误差 Δbc=Td2\Delta_{bc}=\frac{T_d}{2}Δbc​=2Td​​
                                  对于图示右侧的情况, 两基准距离为即零件上的尺寸 LLL, 因此有定位误差 Δbc=TL\Delta_{bc}=T_LΔbc​=TL​

                                  # 基准位置误差导致的定位误差

                                  当定位基准随工件某一特征变动时, 定位基准的最大变动量为 Δjw\Delta_{jw}Δjw​
                                  分别计算该特征取最大值与最小值时基准相对特定绝对位置的距离, 将两个距离相减得到 Δjw\Delta_{jw}Δjw​

                                    计算时可以 V 型块的下顶点 MMM 为参考绝对位置, 分别计算两个基准相对 MMM 的距离

                                    计算时可以点 MMM 在定位点在水平线的垂足 HHH 为参考绝对位置, 分别计算两个基准相对 HHH 的距离

                                  ∠MN1N2=αmin⁡∠MN2N1=180∘−αmax⁡N1H‾=hcot⁡αmin⁡N2H‾=−hcot⁡αmax⁡N1N2‾=N1H‾−N2H‾=h(cot⁡αmin⁡+cot⁡αmax⁡) \begin{split} \angle MN_{1}N_{2}=\alpha_{\min}\quad&\angle MN_{2}N_{1}=180^\circ-\alpha_{\max}\\ \overline{N_1H}=h\cot\alpha_{\min}\quad&\overline{N_2H}=-h\cot\alpha_{\max}\\ \overline{N_1N_2}=\overline{N_1H}-\overline{N_2H}&=h(\cot\alpha_{\min}+\cot\alpha_{\max }) \end{split}∠MN1​N2​=αmin​N1​H​=hcotαmin​N1​N2​​=N1​H​−N2​H​​∠MN2​N1​=180∘−αmax​N2​H​=−hcotαmax​=h(cotαmin​+cotαmax​)​

                                  # 定位误差计算

                                  计算定位误差时, 不能直接将 Δbc\Delta_{bc}Δbc​ 与 Δjw\Delta_{jw}Δjw​ 相加
                                  当 Δbc\Delta_{bc}Δbc​ 与 Δjw\Delta_{jw}Δjw​ 相关时, 还需要判断两者的方向, 当两者方向相反时则需要相减

                                  如图所示, 分析时将 d=dmin⁡d=d_{\min}d=dmin​ 的情况视为理想位置, d=dmax⁡d=d_{\max}d=dmax​ 的情况视为最大误差位置, 此时
                                  Δbc\Delta_{bc}Δbc​ 的方向随工艺基准的位置改变 (保持为定位基准指向工艺基准), Δjw\Delta_{jw}Δjw​ 方向与大小不变 (理想基准指向最大误差基准)
                                  因此对于图示的三个基准, 定位误差 Δdw\Delta_{dw}Δdw​ 不同

                                  计算注意

                                  • 当 Δbc\Delta_{bc}Δbc​ 与 Δjw\Delta_{jw}Δjw​ 有多个独立的来源时, 则可以分开分别计算 Δdw\Delta_{dw}Δdw​, 然后相加
                                  • 计算时不要混淆 Δbc\Delta_{bc}Δbc​ (定位与工序基准之间距离的公差, 如上例中的半径公差) 与 Δjw\Delta_{jw}Δjw​ (两极限位置下定位基准的距离, 如上例中的圆心距离)
                                  • 注意计算 Δbc\Delta_{bc}Δbc​ 与 Δjw\Delta_{jw}Δjw​ 时, 还应投影到工序尺寸方向上, 如下图所示, Δjw\Delta_{jw}Δjw​ 与工序尺寸方向垂直, 因此对于定位误差没有影响

                                  # 正态分布曲线

                                  • 分布曲线的拐点

                                  • 正态分布曲线的绘制

                                    1. 标出顶点 (μ,0.4σ)(\mu,\frac{0.4}{\sigma})(μ,σ0.4​)
                                    2. 标出两个拐点 (μ±σ,0.24σ)(\mu\pm\sigma,\frac{0.24}{\sigma})(μ±σ,σ0.24​)
                                    3. 认为曲线端点为 (μ±3σ,≈0)(\mu\pm 3\sigma,\approx 0)(μ±3σ,≈0)

                                  # 工序能力与不合格率

                                  # 工序能力

                                  定义工序能力 CpC_pCp​ 评价工序能满足加工精度的程度

                                  Cp=T6σ C_p=\frac{T}{6\sigma} Cp​=6σT​

                                  仅当 Cp>1C_p>1Cp​>1 表明工序中绝大部分的工件均能满足公差 TTT 的要求 (加工误差范围小于公差范围), 工序能力足够
                                  当 Cp<1C_p<1Cp​<1 表明工序能力不足, 不合格品无法避免

                                  # 加工质量改进措施

                                  # 控制分散中心

                                  当尺寸均值与公差带中心相差过大时, 表明保持不变的系统误差占主导

                                  μ≈xˉ≠es+ei2 \mu\approx\bar{x}\neq\frac{es+ei}{2} μ≈xˉ=2es+ei​

                                  可通过控制分散中心 μ\muμ, 通过调整机床使其靠近公差带中心 es+ei2\frac{es+ei}{2}2es+ei​ 提升加工质量

                                  # 减小分散范围

                                  当尺寸变动范围 6σ6\sigma6σ 大于公差时, 表明变化的系统误差或随机误差占主导

                                  σ≈S>T \sigma\approx S> T σ≈S>T

                                  可通过控制分散程度 σ\sigmaσ, 通过调整工艺参数与使用更高级的机床提升加工质量
