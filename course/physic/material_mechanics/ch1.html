                                                  # 截面法

                                                  刚体中的力或力偶可移性原理不适用于变形体, 为了研究变形体, 对变形体取截面研究截面的内力

                                                  # 截面法过程

                                                  1. 对变形体取一个任意截面, 使变形体分为两个部分并取任意一个部分研究
                                                  2. 规定截面上内力与内力矩集中于截面的形心, 并以截面指向研究部分外的法线为 xxx 轴正方向, 截面形心为原点, 建立右手坐标系
                                                  3. 由于物体静止, 因此截面上的内力及内力矩与研究部分上的外力平衡, 即合力为 000
                                                  4. 计算内力时注意
                                                    1. 截面内力与外力平衡, 不是等于外力合力, 有 ∑F=−F内\sum F=-F_{\text{内}}∑F=−F内​
                                                    2. 计算内力时可以将研究部分视为刚体, 并且使用力平移定律与力矩平移定律 (力平移后, 将产生一个力矩, 其力臂为平移后点指向平移前点)
                                                    3. 对于不过形心的轴力, 将产生两个扭矩 (非对心压损 / 斜齿轮接触点轴向力)
                                                    4. 注意截面形心坐标系的 xxx 轴方向为截面向外的法线, 当截面方向不同时, 坐标系不同
                                                    5. 截面可以任取, 但只能有一个 (薄壁圆桶问题中两个截面实际在同一个平面上)
                                                    6. 对于同一个截面, 取不同的研究部分, 结果不一定相同
                                                  5. 根据内力计算得到各个位置截面内力的函数, 并画出内力图

                                                  # 截面内力的正方向

                                                  1. 对于截面内力
                                                    1. 轴力, 扭矩的正方向与截面法线方向重合
                                                    2. 在纸面 (x,yx,yx,y 面) 内, 使梁段产生顺时针转动的剪力为正 (截面法线顺时针旋转 90∘90^\circ90∘ 到达剪力正方向)
                                                    3. 在纸面 (x,yx,yx,y 面) 内, 使梁段产生向内 yyy 轴负方向弯曲的弯矩为正 (yyy 轴正方向一侧梁受压, 因此此时的弯矩图 MMM 轴向下)
                                                  2. 已知分析方向时, 产生正内力的外力方向
                                                    1. 轴力, 扭矩的正外力与分析方向相反
                                                    2. 剪力的正外力为分析方向逆时针旋转 90∘90^\circ90∘ 得到
                                                    3. 产生正弯矩的 正外剪力 为杆件向外的方向 (对于水平杆即 yyy 轴方向)
                                                    4. 产生正弯矩的 正外弯矩 的旋向箭头与正外剪力相反, 弧线包裹分析方向

                                                  # 内力图画法

                                                  • 确定截面形心位置与主轴方向 (对于弯曲变形特别注意)
                                                  • 将各个位置的力集中到轴线上, 并按照三个坐标轴方向分解
                                                    • 对于轴线外的力则平移到轴线上, 并产生一个力矩
                                                    • 对于斜方向的力, 需要向坐标轴投影
                                                    • 对于一个截面上的多个力, 可能产生的扭矩相互抵消, 但剪力效果相互叠加, 要具体分析
                                                  • 确定约束力 (对于单个约束如固定端约束, 可作为最后分析点, 从而避免求约束力)
                                                  • 确定内力的分析方向, 截取截面沿此方向移动, 同时也是截面的 xxx 轴正方向
                                                  • 根据分析方向确定在此参考方向上, 产生正内力的外力方向, 便于分析
                                                  • 沿分析方向, 从截面一端分析至另一端, 从 000 开始, 经过存在外力的截面时, 内力根据界面的值发生突变
                                                  • 分析完成后, 轴两侧的内力必定为 000
                                                  • 对于弯矩图与剪力图, 还具有以下特点
                                                    1. 弯矩与剪力存在积分关系, 即 dMdx=Fs\frac{dM}{dx}=F_sdxdM​=Fs​ 因此可先求出剪力, 并由剪力积分得到弯矩
                                                    2. 此积分关系无法反应集中弯矩, 需要具体求解
                                                    3. 对于复杂的弯矩情况, 可通过叠加法求解, 即仅求出各个外力单独作用下的弯矩, 最后再进行叠加
                                                    4. 为了是梁受拉部分在坐标轴上方, 规定弯矩图中, 竖轴正方向朝下

                                                  # 特殊内力图

                                                  1. 钢架内力图
                                                    1. 将钢架拆分为独立杆件分别分析
                                                    2. 在杆的交点, 需要将钢架在角点截开, 然后将没有被分析杆部分视为刚体然后将其外力 (包括约束力) 全部移动到交点上 (移动力可能产生附加力矩)
                                                    3. 在分析各个杆的时候仍需要规定好分析方向
                                                    4. 在画钢架内力图时, 要求弯矩出现在受拉一侧 (有弯矩曲线的一侧受拉), 当采用一般正方向时, 弯矩的正值在钢架内侧
                                                    5. 交点处的力对于一侧的杆为剪力, 对于另一侧的杆可能为轴力
                                                  2. 钢架内约束特点
                                                    1. 钢架内的铰链约束
                                                      1. 需要先求出约束力再进行分析
                                                      2. 由于铰链约束上无约束力矩, 因此内力图在铰链约束处的弯矩为 000
                                                    2. 钢架内的转角
                                                      1. 需要通过截面法截开转角, 将其它力平移到转角上分析
                                                      2. 由于力偶无方向性, 因此转角处的力偶在两个杆之间连续
                                                  3. 弧形钢架

                                                  # 内力图快速画法

                                                  1. 通过叠加法实现内力图的快速画法

                                                    1. 即在每个受力状态变化点后使函数叠加上该点的变化 (可能一次叠加多个力)
                                                    2. 叠加只对之后的值有效果, 对之前的值无效, 因此要分段
                                                    3. 通过 (x−a)(x-a)(x−a) 改变坐标系, 从而使用 x=0x=0x=0 时力单独作用的结论
                                                    4. 需要先计算得到约束力才能开始

                                                  2. 一般内力图

                                                    1. 用于扭矩, 轴力与剪力
                                                    2. 内力在集中力点上突变, 即在 该点后 函数叠加常数 FaF_aFa​
                                                    3. 对于分布力系, 则为 该点后 函数叠加 q(x−a)q(x-a)q(x−a), aaa 为分布力开始位置, 分布力系结束后, 将其在结束点之后的值视为常数, 常数大小为结束点上的叠加效果

                                                  3. 弯矩内力图

                                                    1. 根据 dMdx=Fs\frac{dM}{dx}=F_sdxdM​=Fs​, 在剪力处, 该点后 函数叠加 Fsa(x−a)F_{sa}(x-a)Fsa​(x−a)
                                                    2. 对于分布剪力系, 则为叠加 12q(x−a)2\frac{1}{2}q(x-a)^221​q(x−a)2, 结束后处理相同
                                                    3. 对于集中弯矩则直接叠加常数

                                                  4. 钢架内力图

                                                    1. 此结论仅在使用相同规则的弯矩正方向并且按一个分析方向走完整个钢架时成立 (以钢架向外为正)
                                                    2. 由于弯矩在拐角处弯矩连续的特点, 因此在下一根杆上开始时的弯矩叠加上常数, 等于上一根杆在此处的弯矩
                                                    3. 通过此方法避免求外力移动产生的矩 (相等), 仅需将外力移动到弯折点即可 (注意方向转变使力的性质转变)

                                                  # 拉压正应变的计算

                                                  取轴上的微段 dxdxdx, 在拉力的作用下发生了变形, 长度变为 dx′dx'dx′, 定义轴向线应变为 (注意不是变化前后长度之比, 而是长度变化量与原长之比)

                                                  ε=d(Δl)dx=dx′−dxdx \varepsilon=\frac{d(\Delta l)}{dx}=\frac{dx'-dx}{dx} ε=dxd(Δl)​=dxdx′−dx​

                                                  # 拉压正应力的计算

                                                  认为轴向的正应力在垂直于轴线的轴截面 (横截面) 上均匀分布, 因此横截面上正应力的合力即截面上的轴力 (属于内力, 不是外力) 满足

                                                  σ=FN(x)A \sigma=\frac{F_N(x)}{A} σ=AFN​(x)​

                                                  # 拉压变形的计算

                                                  • 根据胡克定律可得, 在材料的线弹性阶段, 拉压变形物体的应变与应力成正比, 满足

                                                  σ=Eε \sigma=E\varepsilon σ=Eε

                                                  • 根据应变与应力的关系可得拉压伸长变形满足

                                                  d(Δl)=FN(x)dxEA d(\Delta l)=\frac{F_N(x)dx}{EA} d(Δl)=EAFN​(x)dx​

                                                  • 积分可得到拉伸变形量与轴向内力 FNF_NFN​ 满足关系

                                                  Δl=∫LFN(x)dxEA \Delta l=\int_L \frac{F_N(x)dx}{EA} Δl=∫L​EAFN​(x)dx​

                                                  其中 FN(x)F_N(x)FN​(x) 为截面轴力关于截面位置的函数, 定义 EAEAEA 为拉压刚度

                                                  1. 取微段 dxdxdx, 微段端面上有两点 ABABAB, 在经过扭转变形后, BBB 点变形至 CCC 点, 定义切应变为 ABABAB 变形的角度 ∠BAC=γ\angle BAC=\gamma∠BAC=γ
                                                  2. 根据切应变定义可得扭转切应变表达式为, 可得切应变的大小还与截面上点的径向位置 ρ\rhoρ 有关 (几何条件)

                                                  γ≈tan⁡∠BAC=BCAB=ρdφdx \gamma\approx\tan\angle BAC=\frac{BC}{AB}=\frac{\rho d\varphi}{dx} γ≈tan∠BAC=ABBC​=dxρdφ​

                                                  # 扭转应力的计算

                                                  • 根据胡克定律可得, 应力与应变满足关系 τ=Gγ\tau=G\gammaτ=Gγ
                                                  • 扭转变形中, 切应力的合力即截面上的扭矩 (属于内力, 不是外力), 因此可得到切应力满足 (物理条件)

                                                  τ=T(x)ρIρ \tau=\frac{T(x)\rho}{I_\rho} τ=Iρ​T(x)ρ​

                                                  • 其中 IρI_{\rho}Iρ​ 为极惯性矩, 为一个与截面形状有关的物理量, 其定义为

                                                  Iρ=∫Aρ2dA I_\rho=\int_A \rho^2 dA Iρ​=∫A​ρ2dA

                                                  • 对于厚圆筒截面满足 Iρ=π32(D14−D24)I_{\rho}=\frac{\pi}{32}(D_1^4-D_2^4)Iρ​=32π​(D14​−D24​)
                                                  • 可得, 在圆柱的表面切应力达到最大, 最能反映圆柱面强度, 因此定义抗扭截面系数 Wρ=Iρ/RW_{\rho}=I_{\rho}/RWρ​=Iρ​/R, 单位为 m3m^3m3

                                                  # 扭转变形的计算

                                                  1. 根据切应力的表达式与胡克定律可得扭转变形中, 微段 dxdxdx 的扭转角 dφd\varphidφ 满足

                                                  1. 积分可得到扭转变形的扭转角

                                                  # 弯曲变形

                                                  在本节讨论中, 做出以下定义

                                                  1. 定义 xxx 轴为梁的轴线, 弯曲变形发生在 yzyzyz 截面, 截面几何量均相对于 yzyzyz 截面讨论
                                                  2. 定义 yyy 轴正方向为梁受拉侧指向受压侧的方向, 即弯曲方向
                                                  3. 定义 zzz 轴为梁的中性轴, 即平行且通过梁中性层的轴线 (梁中未发生拉伸变形层, 同时也是受压与受拉的分界层), 同时弯矩也为 zzz 轴方向
                                                  4. 公式在其他截面情况下依然成立, 但弯曲方向与中性轴需满足右手定则, 否则要取负号

                                                  # 纯弯曲正应变的计算

                                                  1. 纯弯曲中, 梁仅受弯矩作用, 没有剪力, 因此梁上仅有正应力, 没有剪应力, 每个截面均匀绕中性轴偏转, 梁弯曲成为一条圆弧线
                                                  2. 对变形后的圆弧取一个段 dθd\thetadθ, 由于中性层不变的性质有 dx=ds=ρdθdx=ds=\rho d\thetadx=ds=ρdθ, 其中 ρ\rhoρ 为弯曲微段的曲率半径, 根据线应变的定义可得, 线应变满足

                                                  ε=ds′−dsds=−yρ \varepsilon=\frac{ds'-ds}{ds}=\frac{-y}{\rho} ε=dsds′−ds​=ρ−y​

                                                  # 纯弯曲正应力的计算

                                                  1. 根据胡克定律将应变转为应力, 并且带入物理条件截面上应力的合力矩与弯矩相同, 且合力为 000 (截面上没有轴力)
                                                  2. 根据截面无轴力条件可得, 中性轴上满足条件 Sz=∫AydA=0S_z=\int_A ydA=0Sz​=∫A​ydA=0, 即中性轴 zzz 位于截面关于 yyy 轴的形心上 (SzS_zSz​ 为截面静矩)
                                                  3. 根据截面上应力的合力矩与弯矩相同可得, 弯曲正应力与弯矩满足关系

                                                  σ=−M(x)yIz \sigma=\frac{-M(x)y}{I_z} σ=Iz​−M(x)y​

                                                  1. 其中 IzI_zIz​ 为截面相对于 zzz 轴的惯性矩 (相对于 zzz 的距离即 yyy 坐标), 单位为 m4m^4m4, 具体计算见后

                                                  # 弯曲剪应力的计算

                                                  • 一般弯曲中, 由于剪力的存在, 因此弯曲变形还存在剪应力与剪切变形 (变形效果为使梁不均匀弯曲或上下面错位), 但通常极小, 可忽略, 并假设弯曲截面上的剪力均平行于剪力 (不均匀分布)
                                                  • 先取微段 dxdxdx (即图中的 mnmnmn), 再对微段 yyy 处取截面
                                                  • 由于受剪力时, 弯矩不为常数, 因此微段两侧的正应力合力为 dFN=FsSz∗/IzdF_N=F_sS^*_z/I_zdFN​=Fs​Sz∗​/Iz​ (注意弯矩与剪力的微分关系), 因此截面上也将存在切应力, 并且使微段的轴向力为 000
                                                  • 根据切应力互等原理, 下表面无切应力并且两侧面与截面的交界处切应力与截面互等 (不是整个侧面), 由此可得弯曲切应力满足

                                                  τb=FSSz∗Iz \tau b=\frac{F_S S_z^*}{I_z} τb=Iz​FS​Sz∗​​

                                                  其中 bbb 为截面 zzz 轴方向的宽度, Sz∗S_z^*Sz∗​ 为被截微段的侧面相对于 zzz 轴的静矩

                                                  • 可得剪应力的分布特点为两侧为 000, 中性轴上剪应力最大
                                                    1. 对于矩形截面, τmax=1.5FSA\tau_{max}=1.5\frac{F_S}{A}τmax​=1.5AFS​​
                                                    2. 对于圆截面, τmax=43FSA\tau_{max}=\frac{4}{3}\frac{F_S}{A}τmax​=34​AFS​​

                                                  # 弯曲变形的计算

                                                  • 联立弯曲正应变与弯曲正应力可得
                                                  • 注意在小变形条件下将 dxdxdx 近似为微圆弧段, 有关系 dx≈ρdθdx\approx \rho d\thetadx≈ρdθ

                                                  # 刚化叠加法

                                                  1. 刚化叠加法为将复杂结构分段, 将每一段视为弹性体其他视为刚体分别分析, 最后叠加
                                                  2. 通常根据受力分段, 在集中力处分段, 在分布力作为一段等, 使之与简单情况相同便于分析
                                                  3. 对于刚体满足以下性质
                                                    1. 刚体上的力或力偶可以任意移动至刚体与弹性体的交界 (可能产生附加力矩)
                                                    2. 刚体上 θ=C,y=Cx\theta=C,y=Cxθ=C,y=Cx, 但与弹性体交界处仍满足约束的边界条件 (见上)
                                                    3. 刚体与弹性体接触时的受力可认为有垂直于接触方向的约束力
                                                    4. 对于远离弹性体确认不会变形的刚体可以不分析

                                                  # 弯矩面坐标轴的选择

                                                  1. 以上的弯曲公式仅在以下条件下成立
                                                    1. 选择截面的形心主惯性轴作为坐标系 (查看题目要求, 用于非对称截面)
                                                    2. 将截面的对称轴作为 yyy 或 zzz 轴, 且另一轴过形心 (根据此要求, 矩形只能以两个对称轴作为 yzyzyz 轴, 圆形则可任意选择)
                                                  2. 为了使用弯曲公式, 分析弯矩时, 必须采用以上要求的坐标轴
                                                  3. 注意当中性轴为 yyy 轴, 弯曲方向为 zzz 轴时, 右手关系不满足, 因此公式需要取负号 (原来负号被抵消)
                                                  4. 对于不平行于坐标轴的剪力或弯矩, 需要分解投影到坐标轴上分别计算变形
                                                  5. 计算两个方向的变形时, 注意坐标轴两个方向上的惯性矩不同
                                                  6. 求总挠度时, 可以使用平行四边形法则将两个方向上的挠度叠加

                                                  # 复合梁的弯曲变形

                                                  1. 复合梁为截面上由两种材料组成的梁, 两种材料弹性模量与形状不同
                                                  2. 对于复合梁, 其应变连续但应力不连续, 需要对弯曲公式进行修改
                                                  3. 通过等效截面法计算 zzz 轴的位置 (不是形心)
                                                    1. 复合梁中性轴满足条件 E1SZ1+E2SZ2=0E_1S_{Z1}+E_2S_{Z2}=0E1​SZ1​+E2​SZ2​=0
                                                    2. 选择其中一种材料 E1E_1E1​ 为基础, 另一种材料在 zzz 轴方向长度变化 E2E1\frac{E_2}{E_1}E1​E2​​, 得到等效截面
                                                    3. 计算等效截面的形心即 zzz 轴位置
                                                  4. 梁的应变连续, 因此复合梁的刚度为 E1IZ1+E2IZ2E_1I_{Z1}+E_2I_{Z2}E1​IZ1​+E2​IZ2​, 但应力不连续, 依然通过虎克定律求出有

                                                  # 偏心拉伸与压缩

                                                  • 当轴力 (内力) 不过截面形心时, 需要将其平移至截面形心处, 同时对 y,zy,zy,z 轴产生弯矩, 因此截面将沿 z,yz,yz,y 方向发生弯曲变形 (注意弯曲公式的符号)
                                                  • 截面的应力将由均匀分布的拉伸应力与两个弯曲应力叠加而成, 定义截面上拉应力与压应力的交界线为中性轴 (正应力为 000), 因此可得中性轴满足 (提出 FFF 与 AAA)

                                                  1+yFy0(IZ/A)+zFz0(IY/A)=0 1+\frac{y_Fy_0}{(I_Z/A)}+\frac{z_Fz_0}{(I_Y/A)}=0 1+(IZ​/A)yF​y0​​+(IY​/A)zF​z0​​=0

                                                  设中心轴所在点的坐标为 (y0,z0)(y_0,z_0)(y0​,z0​), 偏心压力作用点为 (yF,zF)(y_F,z_F)(yF​,zF​)

                                                  • 由中心轴条件可得, 当偏心压力坐标越大, 中性轴越靠近形心
                                                    1. 为了保证截面仅受压应力 (脆性材料), 需要保证在最偏心的压力作用下, 中性轴仅与截面相切, 这个偏心压力作用范围称为截面核心
                                                      • 计算截面核心时, 可假设中心轴过截面最外侧交点, 并带入 (y0,z0)(y_0,z_0)(y0​,z0​), 得到一条截面核心的边界线
                                                      • 对于圆形截面, 截面核心为圆, 带入圆边缘一点即可得到其半径
                                                      • 对于矩形截面等多边形, 危险点为多边形的顶点, 叠加各个顶点对应的边界线即可得到截面核心
                                                    2. 在校核截面是否受拉应力时, 危险点位置与上同
