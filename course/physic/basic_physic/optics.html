                                                                      # 光波性质

                                                                      1. 光是电磁波, 其在真空中的传播速度最快, 且为常量 ccc, 因此有

                                                                      c=λν c=\lambda\nu c=λν

                                                                      1. 将光速与光波在透明介质传播速率的比值定义为折射率

                                                                      1. 光波中 E⃗\vec{E}E 与 H⃗\vec{H}H 垂直, 使用 E⃗\vec{E}E 表征光波, 将 E⃗\vec{E}E 的振动称为光振动
                                                                      2. 沿 xxx 方向的平面光波可表示为

                                                                      E⃗(x,t)=E0⃗cos(ωt−2πλx+φ0) \vec{E}(x,t)=\vec{E_0}cos(\omega t-\frac{2\pi}{\lambda}x+\varphi_0) E(x,t)=E0​​cos(ωt−λ2π​x+φ0​)

                                                                      # 光波叠加

                                                                      同频率光波叠加类似同频率简谐波的叠加, 满足旋转矢量的平行四边形叠加的余弦定律

                                                                      # 相干叠加

                                                                      当叠加光波相干(如来自同一个光源), 此时两列波 φ0\varphi_0φ0​ 相同, 波程差来自光程 xxx, 因此对于一个点, cos⁡Δφ‾=C\overline{\cos\Delta\varphi}=CcosΔφ​=C, 并在平面上显现出明暗相间的干涉条纹

                                                                      Δφ={±2kπ,明条纹±(2k+1)π,暗条纹(k=±0,1,2,… ) \Delta\varphi=\begin{cases}\pm 2k\pi,&\text{明条纹}\\\pm (2k+1)\pi,&\text{暗条纹}\end{cases}(k=\pm 0,1,2,\dots) Δφ={±2kπ,±(2k+1)π,​明条纹暗条纹​(k=±0,1,2,…)

                                                                      # 相干条件

                                                                      1. 各光波频率相同
                                                                      2. 各光波有平行分量(能够投影到一个平面上)
                                                                        • 由于光为横波, 两列光波的光矢量在空间中不一定平行, 需要将其中一个光矢量投影到另一个上计算
                                                                        • 垂直分量的光矢量构成背景光
                                                                        • 当光矢量垂直, 则不构成相干光
                                                                      3. 光振动的相位差保持稳定(从同一光源分光或使用激光源)

                                                                      # 物象间等光程性

                                                                      1. 光学仪器 (透镜, 球镜) 的物点 (光源) 和像点 (焦点) 之间的各个光线光程相同
                                                                      2. 对于平行光源, 在垂直于光线的平面内相位相同, 来自无穷远处的光源
                                                                      3. 认为平行光聚焦于无穷远处, 来自无穷远处的光源

                                                                      # 时间相干性

                                                                      # 晶体双折射现象

                                                                      1. 各向异性介质 (晶体) 中, 一束入射光将产生两束折射光, 将符合折射定律的折射光称为 ooo 光, 另一束则称为 eee 光
                                                                      2. 晶体中存在一个方向, 沿此方向不发生双折射, 称为光轴 (存在无数条, 相互平行)
                                                                      3. 对于晶体中的点光源, 经过一段时间后, ooo 光产生普通的球面光, eee 光则产生椭球面光, 且在光轴上于 ooo 光产生的球面相切 这是由于 eee 光中, 光沿垂直于光轴的方向传播速度 ve≠vov_e\neq v_ove​=vo​
                                                                        1. 当 vo>vev_o>v_evo​>ve​ 称为正晶体, 球面在椭球面外 (此时 no<ne=cven_o<n_e=\frac{c}{v_e}no​<ne​=ve​c​)
                                                                        2. 当 vo<vev_o<v_evo​<ve​ 称为负晶体, 球面在椭球面内 (此时 no>nen_o>n_eno​>ne​) 当入射面与光轴平行时, 产生的 ooo 光与 eee 光的光振动垂直

                                                                      # 波片

                                                                      • 当入射光垂直于光轴, 产生的 ooo 光与 eee 光的光振动垂直, 并且由于垂直波速不同, 当光线出射后重新叠加, 出射光的光振动关于 e,oe,oe,o 方向的分量相位被改变, 根据此原理制成波片
                                                                      • 波片厚度 ddd 满足 Δφ=2πλ(ne−no)d\Delta\varphi=\frac{2\pi}{\lambda}(n_e-n_o)dΔφ=λ2π​(ne​−no​)d, 只对特定的 λ\lambdaλ 有效
                                                                      • 对于自然光, 波片无法改变其状态
                                                                      # 四分之一波片
                                                                      1. 能够产生 π/2\pi/2π/2 的相位差
                                                                      2. 当入射光为线偏振光
                                                                        1. 当与 ooo 光或 eee 光的光振动方向相同, 则出射仍为线偏振光
                                                                        2. 当与 ooo 光或 eee 光的光振动夹角为 45∘45^{\circ}45∘, 则初射后, 正交分量振幅相同, 相位差 π/2\pi/2π/2, 为圆偏振光
                                                                        3. 其他情况下, 由于正交分量振幅不同, 为椭圆偏振光
                                                                      3. 当入射光为圆偏振光, 圆偏振光向任意方向分解, 相位差均为 π/2\pi/2π/2, 再次改变后, 变为 π\piπ, 因此变为线偏振光
                                                                      4. 当入射光为椭圆偏振光, 如果其 ooo 光与 eee 光的分量为 π/2\pi/2π/2 (主轴), 则出射线偏振光, 否则仍为椭圆偏振光
                                                                      # 半波片
                                                                      1. 能够产生 π\piπ 的相位差
                                                                      2. 入射光为线偏振光时, 能够改变其偏振方向
                                                                      3. 入射光为圆偏振光时, 能够改变其旋转方向

                                                                      # 偏振光的干涉

                                                                      • 线偏振光通过波片, 改变其 ooo 光与 eee 光方向分量的相位差
                                                                      • 再通过偏振片, 获得其 ooo 光与 eee 光方向分量在偏振片方向的投影
                                                                      • 此时可将 ooo 光与 eee 光方向分量视为两束叠加的相干光
                                                                      • 两束光有相位差, 只与入射光波长与波片厚度有关 (只考虑相位差, 不考虑振幅大小)

                                                                      Δφ=2πλ(ne−ne)d \Delta\varphi=\frac{2\pi}{\lambda}(n_e-n_e)d Δφ=λ2π​(ne​−ne​)d

                                                                      • 如果投影后, ooo 光与 eee 光方向相反, 则要附加光程差 π/2\pi/2π/2, 此时转动偏振片, 可能使其投影后方向相同, 导致干涉结果变化
