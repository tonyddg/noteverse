                                                        # 阻尼振动

                                                        阻尼振动中, 物体还受到一个阻碍运动的阻力, 其方向与速度方向相反, 大小与速度成正比

                                                        F=−γdxdt F=-\gamma\frac{dx}{dt} F=−γdtdx​

                                                        γm=2β \frac{\gamma}{m}=2\beta mγ​=2β

                                                        # 弱阻尼

                                                        1. 当阻尼较小时, 物体依然保持往复运动
                                                        2. 弱阻尼下, 物体的角速度减小, 固有周期增长, 振动被延缓

                                                        ω=ω02−β2 \omega=\sqrt{\omega_0^2-\beta^2} ω=ω02​−β2​

                                                        # 过阻尼/临界阻尼

                                                        1. β>ω0\beta>\omega_0β>ω0​ 时, 为过阻尼, 物体回到平衡位置后不再运动
                                                        2. β=ω0\beta=\omega_0β=ω0​ 时, 为临界阻尼, 物体回到平衡位置速度最快

                                                        # 受迫振动

                                                        1. 在周期性外力的作用下振动, 称为受迫振动
                                                        2. 振动的角频率, 振幅与初相位取决于驱动力
                                                        3. 当物体的固有频率与驱动力的频率接近时, 受迫振动的振幅达到最大, 称为共振

                                                        # 机械波的条件

                                                        1. 波源
                                                        2. 能够传播机械振动的介质

                                                        # 机械波的描述

                                                        1. 介质中振动相位相同的点构成的面为波面
                                                        2. 沿传播方向上的线为波线

                                                        # 机械波的波长

                                                        1. 符号表示 λ\lambdaλ
                                                        2. 同一波线上相位相邻 2π2\pi2π 的两个质元之间的距离

                                                        # 波的周期和频率

                                                        1. 周期符号 TTT, 频率符号 ν\nuν
                                                        2. 波的周期与波源的振动周期相同

                                                        # 波速

                                                        1. 相位传播的速度, 又称为相速度, 符号表示 uuu

                                                        2. u=λT=νλ u=\frac{\lambda}{T}=\nu\lambda u=Tλ​=νλ

                                                        # 波函数

                                                        用于描述波线上不同点的振动状态

                                                        y(x,t)=Acos⁡[ω(t−xu)+φ]=Acos⁡[2π(tT−xλ+φ)] y(x,t)=A\cos[\omega(t-\frac{x}{u})+\varphi]=A\cos[2\pi(\frac{t}{T}-\frac{x}{\lambda}+\varphi)] y(x,t)=Acos[ω(t−ux​)+φ]=Acos[2π(Tt​−λx​+φ)]

                                                        1. xxx 为波传播的正方向, 如果沿负方向传播, 则要取负数
                                                        2. 如果已知任意点 x0x_0x0​ 的振动的振动状态(初相等), 则可带入 x−x0x-x_0x−x0​, 得到波函数
                                                        3. 求波函数时
                                                          1. 最好先使用旋转矢量法确定一个点 (原点) 的振动方程
                                                          2. 再确定波传播的方向
                                                          3. 使用 t±x/ut\pm x/ut±x/u 代换 ttt

                                                        # 波的能量

                                                        假设初相位 φ=0\varphi=0φ=0, 波中任一点的能量为(每一点能量不同)

                                                        E=Ep+Ek=ρΔVA2ω2sin⁡2[ω(t−xu)] E=E_p+E_k=\rho\Delta V A^2\omega^2\sin^2[\omega(t-\frac{x}{u})] E=Ep​+Ek​=ρΔVA2ω2sin2[ω(t−ux​)]

                                                        1. Ep∝ΔyE_p\propto\Delta yEp​∝Δy, 在波峰与波谷处, yyy 大, 但 Δy\Delta yΔy 小, EpE_pEp​ 小; 在平衡位置 EpE_pEp​ 大
                                                        2. Ek∝vE_k\propto vEk​∝v, 在波峰与波谷处 EkE_kEk​ 小 , 在平衡位置处 EkE_kEk​ 大
                                                        3. 因此波的能量在波峰波谷处最小, 能量流入, 在平衡位置能量最大, 能量流出; 不同于振动, 波中任一点的能量随时间不断变化
                                                        4. 整个波的能量是连续的, 但波上的单个点作简谐振动, 其能量不变

                                                        ω‾=1T∫EΔVdt=12ρA2ω2 \overline{\omega}=\frac{1}{T}\int \frac{E}{\Delta V}dt=\frac{1}{2}\rho A^2\omega^2 ω=T1​∫ΔVE​dt=21​ρA2ω2

                                                        • 平均能流密度通常又称为波的强度

                                                        # 波的干涉

                                                        # 波干涉条件

                                                        1. 相遇的两列波频率相同, 振动方向相同(不是传播方向), 对振幅无要求
                                                        2. 在相遇区域两列波的相位差保持不变

                                                        # 干涉的极大与极小

                                                        1. 当相位差 Δφ=(φ2−φ2)=2kπ(k=0,±1,±2,...)\Delta\varphi=(\varphi_2-\varphi_2)=2k\pi(k=0,\pm 1,\pm 2,...)Δφ=(φ2​−φ2​)=2kπ(k=0,±1,±2,...) 时, 两列波同相, 为干涉极大, 和振幅为 A=A1+A2A=A_1+A_2A=A1​+A2​
                                                        2. 当相位差 Δφ=(φ2−φ2)=2kπ(k=0,±1,±2,...)\Delta\varphi=(\varphi_2-\varphi_2)=2k\pi(k=0,\pm 1,\pm 2,...)Δφ=(φ2​−φ2​)=2kπ(k=0,±1,±2,...) 时, 两列波反相, 为干涉极小, 和振幅为 ∣A∣=∣A1−A2∣|A|=|A_1-A_2|∣A∣=∣A1​−A2​∣
                                                        3. 当两列波初相位相同, 相位差只取决于波程差(两列波的波速相同)

                                                        Δφ2πλ=Δr=r2−r1 \frac{\Delta\varphi}{2\pi}\lambda=\Delta r=r_2-r_1 2πΔφ​λ=Δr=r2​−r1​

                                                        1. 两个波源在一点的位相差要综合考虑波源初位相与波程导致的位相差

                                                        Δφ=(φ2−φ1)−2πλ(r2−r1) \Delta\varphi = (\varphi_2-\varphi_1)-\frac{2\pi}{\lambda}(r_2-r_1) Δφ=(φ2​−φ1​)−λ2π​(r2​−r1​)

                                                        # 驻波

                                                        1. 当两列振幅相同的相干波在同一直线上沿反方向传播时, 在叠加区域形成驻波
                                                        2. 合成的驻波有方程 y=2Acos⁡(2πxλ)cos⁡(ωt)y=2A\cos(\frac{2\pi x}{\lambda})\cos(\omega t)y=2Acos(λ2πx​)cos(ωt)
                                                        3. 由方程得驻波的振幅为 2Acos⁡(2πxλ)2A\cos(\frac{2\pi x}{\lambda})2Acos(λ2πx​) 与时间无关, 且成余弦函数形式
                                                        4. 当 ∣cos⁡(2πxλ)∣=1|\cos(\frac{2\pi x}{\lambda})|=1∣cos(λ2πx​)∣=1 (注意取绝对值) 即 x=kλ2(k∈Z)x=k\frac{\lambda}{2}(k\in Z)x=k2λ​(k∈Z), 振幅最大, 为波腹
                                                        5. 当振幅最小时为波节
                                                        6. 当驻波两端固定时, 两端必为波节, 则要形成稳定的驻波, 要求绳长满足 l=nλ2l=n\frac{\lambda}{2}l=n2λ​ (两波节间长度的整数倍)

                                                        # 半波损失

                                                        1. 当入射点不移动, 则相当于反射波与入射波形成的驻波的一个波节, 此时入射波与反射波反相, 即有相位差 π\piπ, 称为半波损失
                                                        2. 当波从波疏介质入射到波密介质, 反射波将产生半波损失(折射波不存在半波损失), 否则不会有半波损失
                                                        3. 波密与波疏取决于介质中的 ρ⋅u\rho\cdot uρ⋅u
                                                        4. 半波损失指入射波与反射波在入射点处的相位差为 π\piπ, 不是波函数的相位差为 π\piπ
                                                        5. 入射波与反射波的方向相反, 即波函数中要取 −x-x−x
                                                        6. 通过求出入射波在入射点的振动方程, 得到反射波在入射点的振动方程, 带入 −(x−x0)-(x-x_0)−(x−x0​)(传播方向相反, 特定点的振动), 从而求出反射波的波函数
