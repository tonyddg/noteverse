                                    # 运动学

                                    # 质点运动学

                                    # 自然坐标系

                                    设质点轨迹的指向运动方向的切向为 τ⃗\vec{\tau}τ, 指向曲率圆心的法向 n⃗\vec{n}n, 将 τ⃗,n⃗\vec{\tau},\vec{n}τ,n 构成的坐标系称为自然坐标系

                                    # 速度表示

                                    自然坐标系下, τ⃗\vec{\tau}τ 即速度的方向, 因此 v⃗=vτ⃗\vec{v}=v\vec{\tau}v=vτ

                                    # 加速度分解

                                    自然坐标系下, 加速度可分解为切向加速度 ata_tat​ 与法向加速度 ana_nan​, 根据微分关系可表示为

                                    a⃗=atτ⃗+ann⃗=dvdtτ⃗+v2ρn⃗ \vec{a}=a_t\vec{\tau}+a_n\vec{n}=\frac{dv}{dt}\vec{\tau}+\frac{v^2}{\rho}\vec{n} a=at​τ+an​n=dtdv​τ+ρv2​n

                                    其中 vvv 为速度的大小, 不是矢量; ρ\rhoρ 为轨迹的曲率半径, 可通过此方法求出运动轨迹的曲率半径

                                    # 点运动的合成

                                    将点的运动向静系和动系分解, 可得到三种运动

                                    1. 绝对运动 (absolution) 动点相对于静系的运动
                                    2. 相对运动 (relation) 动点相对于动系的运动
                                    3. 牵连运动 (entrainment) 动系相对于静系的运动

                                    解题时必须明确指出所选择的动系与计算的动点

                                    # 速度合成定律

                                    va⃗=vr⃗+ve⃗ \vec{v_a}=\vec{v_r}+\vec{v_e} va​​=vr​​+ve​​

                                    # 加速度合成定律

                                    加速度合成时, 如果动系不是平动 ω=0\omega=0ω=0 , 还要考虑科氏加速度 ak⃗=2ω⃗×vr⃗\vec{a_k}=2\vec{\omega}\times\vec{v_r}ak​​=2ω×vr​​ 的影响

                                    a⃗=ar⃗+ae⃗+2ω⃗×vr⃗ \vec{a}=\vec{a_r}+\vec{a_e}+2\vec{\omega}\times\vec{v_r} a=ar​​+ae​​+2ω×vr​​

                                    # 动系动点的选择

                                    1. 动点为两个框体的交点时, 分别以两个框体为动系, 联立方程
                                    2. 没有稳定接触点时, 使用刚体的平面运动
                                    3. 有稳定接触点时, 动系的选择
                                      1. 动点不能在动系上, 否则无意义
                                      2. 动系最好与题目有关 (求动系的 ω\omegaω )
                                      3. 动点轨迹易求
                                      4. 动系平动 (避免求科氏加速度)
                                    4. 动点选择
                                      1. 套筒
                                      2. 铰链
                                      3. 圆心
                                      4. 杆的一端

                                    # 加速度分析

                                    按原先速度分析的顺序对加速度进行分析

                                    由于杆 DEDEDE 匀速运动, 因此动点 DDD 的绝对加速度满足

                                    aD=0 a_D=0 aD​=0

                                    杆 ADADAD 上的点 DDD 绕铰链 BBB 旋转, 因此需要将牵连加速度 aDe⃗\vec{a_D^e}aDe​​ 按自然坐标系分解为法向 aDen⃗\vec{a_D^{en}}aDen​​ 与切向 aDeτ⃗\vec{a_D^{e\tau}}aDeτ​​, 其中

                                    • 切向加速度的方向与速度方向相同, 垂直于杆 ADADAD, 大小满足

                                    aDeτ=BDαAD=2rαAD a_D^{e\tau}=BD\alpha_{AD}=2r\alpha_{AD} aDeτ​=BDαAD​=2rαAD​

                                    • 法向加速度的方向指向旋转中点, 平行于杆 ADADAD, 满足

                                    aDen=BDωAD2=3v28r a_D^{en}=BD\omega_{AD}^2=\frac{3v^2}{8r} aDen​=BDωAD2​=8r3v2​

                                    同相对速度, 套筒仅会沿杆运动, 因此动点 DDD 的相对加速度 aDr⃗\vec{a_D^r}aDr​​ 依然为沿杆 ADADAD 方向

                                    注意, 由于动系杆 ADADAD 的角速度 ωAD≠0\omega_{AD}\neq 0ωAD​=0, 因此还存在科氏加速度

                                    aDk=2ωAD×vDr=3v24r a_D^k=2\omega_{AD}\times v_D^r=\frac{\sqrt{3}v^2}{4r} aDk​=2ωAD​×vDr​=4r3​v2​

                                    其中科氏加速的方向需要具体分析. ωAD\omega_{AD}ωAD​ 为顺时针方向, 因此矢量方向为垂直纸面向下, 与 vDr⃗\vec{v_D^r}vDr​​ 通过右手定则得到 aDk⃗\vec{a_D^k}aDk​​ 方向

                                    根据加速度合成定律

                                    aD⃗=aDeτ⃗+aDen⃗+aDr⃗+aDk⃗ \vec{a_D}=\vec{a_D^{e\tau}}+\vec{a_D^{en}}+\vec{a_D^{r}}+\vec{a_D^{k}} aD​​=aDeτ​​+aDen​​+aDr​​+aDk​​

                                    一个矢量方程能够分解为两个独立的方程, 解出两个未知量
                                    对于此题, 仅未知量 αAD\alpha_{AD}αAD​ 对于求出待求量有帮助, 因此仅求出 aDeτ⃗\vec{a_D^{e\tau}}aDeτ​​ 即可, 避免加速度分析中不必要的计算 (由于速度分析较简单, 且与加速度分析关联强, 因此能求出的未知量尽量求出)
                                    为了最大程度避免额外未知量参与, 可沿待求未知量 aDeτ⃗\vec{a_D^{e\tau}}aDeτ​​ 方向分解矢量, 得到方程

                                    aDeτ=−aDk=−3v24r a_D^{e\tau}=-a_D^k=-\frac{\sqrt{3}v^2}{4r} aDeτ​=−aDk​=−4r3​v2​

                                    αAD=−3v28r2 \alpha_{AD}=-\frac{\sqrt{3}v^2}{8r^2} αAD​=−8r23​v2​

                                    对动点 AAA 进行加速度分析

                                    由于杆 ADADAD 绕铰链 BBB 转动, 因此动点 AAA 的绝对加速也由切向 aAτ⃗\vec{a_A^\tau}aAτ​​ 与法向 aAn⃗\vec{a_A^n}aAn​​ 两部分组成, 满足

                                    aAτ=ABαAD=−3v24r a_A^{\tau}=AB\alpha_{AD}=-\frac{\sqrt{3}v^2}{4r} aAτ​=ABαAD​=−4r3​v2​

                                    aAn=ABωAB2=3v28r a_A^{n}=AB\omega_{AB}^2=\frac{3v^2}{8r} aAn​=ABωAB2​=8r3v2​

                                    aAeτ=AOαAO a_A^{e\tau}=AO\alpha_{AO} aAeτ​=AOαAO​

                                    # 刚体的平面运动

                                    # 刚体转动

                                    在刚体上选取任意基点, 刚体上其他点相对于基点的角速度与角加速度相同

                                    # 基点法

                                    将刚体上的点的运动分解为:

                                    • 相对于基点的转动 (刚体上两点距离不变, 只能做转动运动)
                                    • 基点作为动系的平动

                                    选取基点 AAA, 求刚体上的质点 BBB 的运动有

                                    # 基点法求速度

                                    vB⃗=vA⃗+vBA⃗ \vec{v_B}=\vec{v_A}+\vec{v_{BA}} vB​​=vA​​+vBA​​

                                    其中

                                    vBA⃗=ω⃗⋅AB⃗ \vec{v_{BA}}=\vec{\omega}\cdot\vec{AB} vBA​​=ω⋅AB

                                    # 基点法求加速度

                                    aB⃗=aA⃗+aBAn⃗+aBAe⃗ \vec{a_B}=\vec{a_A}+\vec{a_{BA}^n}+\vec{a_{BA}^e} aB​​=aA​​+aBAn​​+aBAe​​

                                    其中

                                    aBAn⃗=∣AB∣ω2=vBA2AB \vec{a_{BA}^n}=|AB|\omega^2=\frac{v_{BA}^2}{AB} aBAn​​=∣AB∣ω2=ABvBA2​​

                                    aBAt⃗=∣AB∣α \vec{a_{BA}^t}=|AB|\alpha aBAt​​=∣AB∣α

                                    # 速度投影法

                                    将基点法向 AB⃗\vec{AB}AB 投影可得, 对于刚体上任意两点, 满足

                                    vA⃗⋅eAB⃗=vB⃗⋅eAB⃗ \vec{v_A}\cdot\vec{e_{AB}}=\vec{v_B}\cdot\vec{e_{AB}} vA​​⋅eAB​​=vB​​⋅eAB​​

                                    反映了刚体上任意两点距离不变的特性

                                    # 速度瞬心法

                                    • 刚体上必定存在一点 CCC, 满足 vC=0v_C=0vC​=0
                                    • 通过对刚体上两点速度方向垂线的交线得到 CCC
                                    • 如果垂线平行, 则能判断刚体平动, ω=0\omega=0ω=0
                                    • 对于刚体上任意一点 MMM, vM=MC⋅ωv_M=MC\cdot\omegavM​=MC⋅ω
                                    • 速度瞬心的加速度一般不为 000

                                    # 加速度投影法

                                    • 当刚体平动 ω=0\omega=0ω=0 时使用
                                    • 此时没有法向加速度, 只有切向加速度, 表现出类似速度的性质
                                    • 因此有加速度投影法

                                    aA⃗⋅eAB⃗=aB⃗⋅eAB⃗ \vec{a_A}\cdot\vec{e_{AB}}=\vec{a_B}\cdot\vec{e_{AB}} aA​​⋅eAB​​=aB​​⋅eAB​​

                                    # 加速度瞬心

                                    • 一般情况下加速度瞬心无法直接求得
                                    • 如果刚体 v≠0v\neq 0v=0, 由于法向加速度, 加速度瞬心难以使用
                                    • 当 v=0v=0v=0, 与速度瞬心性质类似, 加速度瞬心为两点加速度垂线的交点

                                    # 纯滚动

                                    • 当圆柱体 (齿轮) 等于地面没有相对滑动, 则称为纯滚动
                                    • 设圆心 OOO, 接触点 AAA

                                    # 纯滚动的运动特性

                                    • 当纯滚动的接触面为地面时 (静止物体, 圆心水平运动)
                                      • 纯滚动中, 接触点 vA=0v_A=0vA​=0 为速度瞬心
                                      • 根据速度瞬心法可得

                                      vO=rω v_O=r\omega vO​=rω

                                      • 接触点不是加速度瞬心
                                      • 圆心的切向加速度满足

                                      aOt=rα a_O^t=r\alpha aOt​=rα

                                    • 当纯滚动的接触面为弧面时, 圆心切向加速度不变, 还有法向加速度(注意与法向加速度公式区分)

                                    aOn=vO2r+R=r2ω2r+R a_O^n=\frac{v_O^2}{r+R}=\frac{r^2\omega^2}{r+R} aOn​=r+RvO2​​=r+Rr2ω2​

                                    • 当相对于运动物块纯滚动时, 则以物块为动系, 当物块平动时, 满足

                                    aOr=rα a_O^r=r\alpha aOr​=rα

                                    # 纯滚动的其他特性

                                    1. 对于绳子绕在圆柱上, 认为绳子与圆柱的接触点纯滚动
                                    2. 纯滚动的位置通常存在一个拉力 / 滑动摩擦力

                                    # 齿轮问题

                                    • 齿轮齿数 zzz 满足

                                    z1z2=r1r2=i12 \frac{z_1}{z_2}=\frac{r_1}{r_2}=i_{12} z2​z1​​=r2​r1​​=i12​

                                    其中 i12i_{12}i12​ 为齿轮的传动比

                                    • 齿轮 / 齿条的啮合处 CCC 为纯滚动, 且对于两齿轮速度相同, 满足

                                    r1ω1=−r2ω2=vC r_1\omega_1=-r_2\omega_2=v_C r1​ω1​=−r2​ω2​=vC​

                                    # 刚体绕平行轴转动

                                    对于二维动系中的刚体, 满足

                                    ωa=ωr+ωe \omega_a=\omega_r+\omega_e ωa​=ωr​+ωe​

                                    αa=αr+αe \alpha_a=\alpha_r+\alpha_e αa​=αr​+αe​

                                    # 曲柄固定齿轮问题

                                    • 对于曲柄固定齿轮的角速度问题, 可以设曲柄为动系, 其他齿轮相对曲柄运动, 此时齿轮圆心相对静止, 易于计算
                                    • 其中固定齿轮 ωa=ωr+ωe=0\omega_a=\omega_r+\omega_e=0ωa​=ωr​+ωe​=0
