                                    # 辐射换热

                                    # 基本概念

                                    # 辐射过程

                                    # 辐射换热特点

                                    1. 辐射换热是指物体之间相互辐射和吸收过程的总效果
                                    2. 热辐射不依赖介质传播
                                    3. 辐射换热时伴随着热力学能与电磁能的转化
                                    4. 温度高于 0K0K0K 的物体就能辐射能量

                                    # 简单辐射过程

                                    热射线与光的特性相同, 所以辐射同样存在投射, 反射, 折射等规律

                                    定义来到物体表面的辐射能为 GGG, 其中吸收部分 (absorb) 为 GαG_\alphaGα​, 反射部分 (reflect) 为 GρG_\rhoGρ​, 透射部分 (transmission) 为 GτG_\tauGτ​

                                    以此定义吸收比 α=GαG\alpha=\frac{G_\alpha}{G}α=GGα​​, 反射比 ρ=GρG\rho=\frac{G_\rho}{G}ρ=GGρ​​, 透射比 τ=GτG\tau=\frac{G_\tau}{G}τ=GGτ​​

                                    对于一般物体, 始终有 α+ρ+τ=1\alpha+\rho+\tau=1α+ρ+τ=1
                                    对于固体与液体, 认为热辐射无法穿过, 有 τ=0,α+ρ=1\tau=0, \alpha+\rho=1τ=0,α+ρ=1
                                    对于气体, 认为热辐射不会被反射, 有 ρ=0,τ+α=1\rho=0, \tau+\alpha=1ρ=0,τ+α=1

                                    # 理想物体与现象定义

                                    注意以下定义均对热辐射而言, 而不是可见光

                                    • 漫反射 / 镜面反射
                                      物体表面是漫反射还是镜反射取决于物体表面相对于辐射波长的表面粗糙程度
                                      当反射的辐射在各个方向均匀分布, 则称为漫反射 (具体见兰贝特定律)
                                      一般情况下认为物体对镜面辐射近似漫反射
                                    • 黑体
                                      当热辐射被完全吸收, 则称为黑体, 满足 α=1\alpha=1α=1
                                      在相同温度的物体中, 黑体的辐射能力最大
                                    • 白体 / 镜体
                                      当物体能完全反射投射到其表面的辐射, 则称为白体 (漫反射) / 镜体 (镜面反射), 满足 ρ=1\rho=1ρ=1
                                    • 透明体
                                      投射到物体表面的辐射被完全透过, 则称为透明体, 满足 τ=1\tau=1τ=1

                                    # 总辐射力

                                    定义单位时间单位辐射面积向半球空间一切方向辐射出去的所有波长的辐射能量为总辐射力

                                    单位为 W/m2W/m^2W/m2

                                    # 单色辐射力

                                    定义单位时间单位辐射面积的辐射能量中, 关于特定波长 λ\lambdaλ 的分布密度为单色辐射力

                                    Eφ=dEdωˉ=d2ΦdωˉdA E_\varphi=\frac{\mathrm{d}E}{\mathrm{d}\bar{\omega}}=\frac{\mathrm{d}^2\Phi}{\mathrm{d}\bar{\omega}\mathrm{d}A} Eφ​=dωˉdE​=dωˉdAd2Φ​

                                    # 定向辐射强度

                                    发射面 dA\mathrm{d}AdA 在垂直于 φ\varphiφ 方向上的接收面上的投影为 dA′\mathrm{d}A'dA′, 同时也是接收面实际接收到的辐射面积
                                    根据几何关系, 两个微元量满足 dAcos⁡φ=dA′\mathrm{d}A\cos\varphi=\mathrm{d}A'dAcosφ=dA′

                                    定义单位时间内垂直于 φ\varphiφ 方向的单位被接收面积 (也称为单位可见辐射面积) 上接收到的辐射能量, 关于接收面所在空间角 ωˉ\bar{\omega}ωˉ 的分布密度为定向辐射强度

                                    # 辐射基本定律

                                    # 黑体辐射

                                    根据能量守恒, 黑体除了能吸收所有能量, 还是在同一温度下, 有最大辐射力的物体

                                    使用下标 bbb 表示来自黑体的辐射力

                                    # 普朗克定律

                                    普朗克定律表示的是黑体的辐射能按波长的分布规律
                                    公式给出了黑体的单色辐射力与热力学温度 TTT, 波长 λ\lambdaλ 之间的 函数关系

                                    Ebλ=c1λ−5ec2/(λT)−1 E_{b\lambda}=\frac{c_1\lambda^{-5}}{e^{c_2/(\lambda T)}-1} Ebλ​=ec2​/(λT)−1c1​λ−5​

                                    c1c_1c1​ 为第一辐射常数， c1=3.743×10−16W⋅m2c_1=3.743\times 10^{-16}W\cdot m^2c1​=3.743×10−16W⋅m2c2c_2c2​ 为第二辐射常数， c2=1.4387×10−2m⋅Kc_2=1.4387\times 10^{-2}m\cdot Kc2​=1.4387×10−2m⋅K

                                    单色辐射力随着波长的增加而增加, 达到某一最大值后又随着波长的增加而慢慢减小, 定义这个最大值为 λm\lambda_mλm​

                                    # 维恩位移定律

                                    λm\lambda_mλm​ 与温度 TTT 成反比, 温度越高, λm\lambda_mλm​ 越小, 辐射中短波热射线含量也随之增大

                                    # 玻尔兹曼定律

                                    对普朗克定律从 λ=(0,∞)\lambda=(0,\infty)λ=(0,∞) 积分, 可得到黑体辐射的总辐射能满足

                                    Eb=σ0T4 E_b=\sigma_0 T^{4} Eb​=σ0​T4

                                    其中

                                    • TTT 为黑体的绝对温度, 单位为 KKK
                                    • σ0=5.67×10−8W/(m2K4)\sigma_0=5.67\times 10^{-8} W/(m^2K^4)σ0​=5.67×10−8W/(m2K4) 为玻尔兹曼常数

                                    # 波段辐射函数

                                    定义波段辐射函数 Fb(λ1−λ2)F_{b(\lambda_1-\lambda_2)}Fb(λ1​−λ2​)​ 为辐射在波长范围 λ1∼λ2\lambda_1\sim\lambda_2λ1​∼λ2​ 的能量占总辐射能的比值, 满足

                                    Fb(λ1−λ2)=E(λ1∼λ2)Eb=∫λ1λ2EbλdλEb F_{b(\lambda_1-\lambda_2)}=\frac{E_{(\lambda_1\sim\lambda_2)}}{E_b}=\frac{\int_{\lambda_1}^{\lambda_2}E_{b\lambda}\mathrm{d}\lambda}{E_b} Fb(λ1​−λ2​)​=Eb​E(λ1​∼λ2​)​​=Eb​∫λ1​λ2​​Ebλ​dλ​

                                    根据积分的特性可得, 波段辐射函数满足

                                    Fb(λ1−λ2)=Fb(0−λ2)−Fb(0−λ1) F_{b(\lambda_1-\lambda_2)}=F_{b(0-\lambda_2)}-F_{b(0-\lambda_1)} Fb(λ1​−λ2​)​=Fb(0−λ2​)​−Fb(0−λ1​)​

                                    波段辐射函数可通过黑体辐射函数求出
                                    注意黑体辐射函数参数为 λT\lambda TλT, 此也表明波段辐射函数会随温度 TTT 变化

                                    Fb(0−λ1)=f(λ1T) F_{b(0-\lambda_1)}=f(\lambda_1 T) Fb(0−λ1​)​=f(λ1​T)

                                    # 兰贝特定律

                                    辐射表面表面沿半球空间各方向上, 定向辐射强度 IφI_\varphiIφ​ 均相等
                                    也将满足这种条件的表面称为漫辐射表面

                                    Iφ=C I_\varphi=C Iφ​=C

                                    根据方向辐射力 EφE_\varphiEφ​ 与定向辐射强度IφI_\varphiIφ​ 的关系可得

                                    Eφ=Iφcos⁡φ E_\varphi=I_\varphi\cos\varphi Eφ​=Iφ​cosφ

                                    公式表明, 对于相同的被接收面积, 各个方向的辐射能均相同; 但对于相同的发射面积, 随俯仰角 φ\varphiφ 增大, 实际接收面积减小, 导致辐射能减小

                                    通过积分可以推导得到黑体的定向辐射强度 IbφI_{b\varphi}Ibφ​ 与黑体辐射力之间满足

                                    Eb=πIbφ E_b=\pi I_{b\varphi} Eb​=πIbφ​

                                    黑体辐射力等于其定向辐射强度 IbφI_{b\varphi}Ibφ​ 的 π\piπ 倍

                                    对于黑体属于漫反射表面, 其对一切方向, 一切波长的辐射吸收率均为 111

                                    α=αλ=αφ=1 \alpha=\alpha_\lambda=\alpha_\varphi=1 α=αλ​=αφ​=1
