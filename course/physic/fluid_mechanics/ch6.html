                          # 量纲分析与相似性原理

                          # 量纲分析

                          在流体力学中, 涉及到的基本量纲为参数 LLL, 时间 TTT, 质量 MMM 与温度 Θ\ThetaΘ (可压缩流体), 其他的参数则可通过这些基本量纲导出

                          # 常用导出参数的量纲

                          对于一些常用的参数, 有导出量纲

                          符号量纲单位
                          几何参数仅包含 LLL
                          SSS 面积L2L^2L2m2m^2m2
                          VVV 体积L3L^3L3m3m^3m3
                          运动参数包含 L,TL,TL,T
                          vvv 速度LT−1LT^{-1}LT−1m/sm/sm/s
                          aaa 加速度LT−2LT^{-2}LT−2m/s2m/s^2m/s2
                          ν\nuν 运动粘度L2T−1L^2T^{-1}L2T−1m2/sm^2/sm2/s
                          动力参数包含 L,T,ML,T, ML,T,M
                          ρ\rhoρ 密度ML−3ML^{-3}ML−3kg/m3kg/m^3kg/m3
                          FFF 力MLT−2MLT^{-2}MLT−2kg⋅m/s2=Nkg\cdot m/s^2=Nkg⋅m/s2=N
                          PPP 功率ML2T−3ML^2T^{-3}ML2T−3kg⋅m2/s3=Wkg\cdot m^2/s^3=Wkg⋅m2/s3=W

                          μ\muμ 动力粘度ML−1T−1ML^{-1}T^{-1}ML−1T−1kg/(s⋅m)kg/(s\cdot m)kg/(s⋅m)

                          # Π 定理

                          假设物理现象与 nnn 个参数有关, 并且有方程表示 (注意方程右侧为 000, 通常公式的结果也属于一个参数)

                          f(q1,q2,…,qn)=0 f(q_1,q_2,\dots,q_n)=0 f(q1​,q2​,…,qn​)=0

                          假设方程涉及 mmm 个上述基本量纲, 则可将方程转换为由 n−mn-mn−m 个无量纲参数表示的方程

                          F(Π1,Π2,…,Πn−m)=0 F(\Pi_1,\Pi_2,\dots,\Pi_{n-m})=0 F(Π1​,Π2​,…,Πn−m​)=0

                          # 无量纲化方法

                          1. 从 nnn 个中选取 mmm 个参数作为循环量, 这 mmm 个参数必须包含 mmm 个基本参数, 且最好从相似性参数中选择
                          2. 将循环量与剩余的参数 n−mn-mn−m 个参数, 分别组成无量纲综合参数 Π\PiΠ
                          3. 对于非循环量参数 xix_ixi​, 可以设其无量纲综合参数可表示为

                          Πi=xi∏k=1n−mmkak \Pi_i=x_i\prod_{k=1}^{n-m} m_k^{a_k} Πi​=xi​k=1∏n−m​mkak​​

                          # 无量纲综合参数求解

                          假设方程涉及基本量纲 L,T,ML,T,ML,T,M
                          循环量 yi=[LliTtiMmi],(i=1,2,3)y_i=[L^{l_i}T^{t_i}M^{m_i}],(i=1,2,3)yi​=[Lli​Tti​Mmi​],(i=1,2,3)
                          对于参数 x=[LuTvMw]x=[L^{u}T^{v}M^{w}]x=[LuTvMw]
                          有无量纲综合参数 Π=x⋅y1a⋅y2b⋅y3c=[L0T0M0]\Pi=x\cdot y_1^a\cdot y_2^b\cdot y_3^c=[L^0T^0M^0]Π=x⋅y1a​⋅y2b​⋅y3c​=[L0T0M0], 因此可得到线性方程组 (注意右侧的负号与排列方向)

                          [l1l2l3t1t2t3m1m2m3][abc]=[−u−v−w] \begin{bmatrix} l_1&l_2&l_3\\ t_1&t_2&t_3\\ m_1&m_2&m_3\\ \end{bmatrix} \begin{bmatrix} a\\ b\\ c \end{bmatrix}= \begin{bmatrix} -u\\ -v\\ -w \end{bmatrix}​l1​t1​m1​​l2​t2​m2​​l3​t3​m3​​​​abc​​=​−u−v−w​​

                          对于同一组循环量, 系数矩阵 AAA 始终相同, 仅需根据参数的量纲改变方程左侧即可

                          # 相似性原理

                          通常为了测试物体在流场中的状态, 需要根据实际情况放大或缩小比例建立模型
                          模型与实际之间在不同参数之间需要存在比例, 以保证实际与模型的相似性

                          # 比例尺

                          定义对于参数 xxx, 根据模型与参数之间的比例定义比例尺 CxC_xCx​ 有

                          Cx=xmxp C_x=\frac{x_m}{x_p} Cx​=xp​xm​​

                          其中
                          mmm 表示模型 model, ppp 表示原型 prototype

                          对于流场中的三个基本量纲, 长度 LLL, 时间 TTT, 质量 MMM 可以根据其他参数的量纲推导出各自的比例尺

                          # 相似性

                          在实际中通常选择以下参数的比例尺作为基本比例尺

                          1. 长度比例尺 CLC_LCL​
                            当这一比例尺满足时, 模型与实际的流场中的各对应长度成同一比例
                            对应基本量纲中的长度 LLL, 称此时称模型与实际有几何相似性
                          2. 速度比例尺 CVC_VCV​
                            当这一比例尺满足时, 模型与实际的流场中的各对应速度方向相同, 且成同一比例
                            对应基本量纲中的时间 TTT, 称此时称模型与实际有运动相似性
                          3. 密度比例尺 CρC_\rhoCρ​
                            当这一比例尺满足时, 模型与实际的流场中的各对应类型的力方向相同, 且成同一比例
                            对应基本量纲中的质量 MMM, 称此时称模型与实际有动力相似性

                          # 参数比例尺的导出

                          注意, 仅在明确了试验与模型满足某种相似性的情况下, 才可以根据相似性对应的比例尺导出有关参数的比例尺 (与下文的动力相似准则做区别)

                          1. 常量
                            对于常量如 ggg, 一般不会随实验条件而改变, 因此常量的比例尺均为 Cc=1C_c=1Cc​=1
                          2. 一般参数
                            1. 根据三个基本比例尺可以分别表达出基本量纲的比例尺
                              CL=CL,CT=CL/CV,CM=CρCL3C_L=C_L,C_T=C_L/C_V,C_M=C_{\rho}C_{L}^3CL​=CL​,CT​=CL​/CV​,CM​=Cρ​CL3​
                            2. 根据参数的量纲即可得到其比例尺对应基本量纲的表达式
                            3. 带入基本量纲关于基本比例尺的表达式即可得到其比例尺

                          将不可压缩粘性流体的动力方程无量纲化后有

                          f⃗−(p0ρl0)∇⃗pˉ+(νv0l02)∇ˉ⃗2vˉ⃗=(v0t0)∂vˉ⃗∂tˉ+(v02t0)(vˉ⃗⋅∇ˉ⃗)vˉ⃗ \vec{f}-(\frac{p_0}{\rho l_0})\vec{\nabla}\bar{p}+(\frac{\nu v_0}{l_0^2})\vec{\bar{\nabla}}^2\vec{\bar{v}}=(\frac{v_0}{t_0})\frac{\partial\vec{\bar{v}}}{\partial \bar{t}}+(\frac{v_0^2}{t_0})(\vec{\bar{v}}\cdot\vec{\bar{\nabla}})\vec{\bar{v}} f​−(ρl0​p0​​)∇pˉ​+(l02​νv0​​)∇ˉ2vˉ=(t0​v0​​)∂tˉ∂vˉ​+(t0​v02​​)(vˉ⋅∇ˉ)vˉ

                          f⃗\vec{f}f​外力f⃗\vec{f}f​

                          由此也可得, 运动方程中的每一项分别对应了一种类型的力

                          # 动力相似准则

                          根据无量纲化的运动方程可得, 只要模型与实际的流场在方程各项的系数成比例, 就一定有同样的解, 因此也将显现出相同的物理规律.

                          然而通常情况下所有系数成比例无法实现, 但是可以忽略影响小的力, 仅要求几个发挥主要作用的力对应的量级成同一比例也能使模型近似地反应实际情况.

                          当要求两种类型力的量级 A1,A2A_1,A_2A1​,A2​ 成同一比例时, 有

                          A1mA1p=A2mA2p→A1mA2m=A1pA2p=C \frac{A_{1m}}{A_{1p}}=\frac{A_{2m}}{A_{2p}}\to\frac{A_{1m}}{A_{2m}}=\frac{A_{1p}}{A_{2p}}=C A1p​A1m​​=A2p​A2m​​→A2m​A1m​​=A2p​A1p​​=C

                          因此将这两种类型力的量级之间的比称为准则数, 当模型与实际具有相同的准则数, 即表示其在准则数对应的两种力之间的关系具有相似性

                          # 动力形似性准则的应用

                          根据比例尺的性质易得, 对于任意参数 xxx 的基准 x0x_0x0​ 在模型与实际之间的比值满足

                          Cx=x0mx0p C_x=\frac{x_{0m}}{x_{0p}} Cx​=x0p​x0m​​

                          模型与实际之间满足的动力相似性的实质即其关于基本比例尺的约束
                          因此可根据模型与实际之间满足的动力相似准则来计算模型与实际之间应当采用的比例尺
                          注意与相似性不同, 动力形似性准则仅能得到有关比例尺之间的关系, 不能推导其他参数的比例尺

                          通常一个试验仅能有一个或两个相似性准则, 过多的准则将导致比例尺难以选择且试验难以设计

                          由于准则数为两个无量纲化项系数之比, 因此还能根据准则数的大小判断两种力在流体运动中发挥作用的大小

                          # 常用相似准则

                          在实际情况下人们要研究的都是流场在某个因素下的运动, 因此反应流场运动的对流惯性力始终是一个重要的研究对象
                          在使用这些准则时, 首先根据准则考虑的因素记忆, 然后在草稿纸上将准则数由两个因数组成的量级分式推导到表达式. 不要直接记忆, 容易出错

                          根据试验因素的不同, 有以下常用的相似性准则

                          参考文献 https://zhuanlan.zhihu.com/p/649931767open in new window

                          # 雷诺准则

                          用于研究流场的粘性对于流场运动的影响

                          Re=对流惯性力粘性力=v02/l0νv0/l02=v0l0ν Re=\frac{\text{对流惯性力}}{\text{粘性力}}=\frac{v_0^2/l_0}{\nu v_0/l_0^2}=\frac{v_0l_0}{\nu} Re=粘性力对流惯性力​=νv0​/l02​v02​/l0​​=νv0​l0​​

                          通常用于计算粘性流体在管道中的流动以及物体在流体中的阻力

                          # 欧拉准则

                          用于研究流场的压力对于流场运动的影响

                          Eu=压力对流惯性力=p0/(ρl0)v02/l0=p0ρv02 Eu=\frac{\text{压力}}{\text{对流惯性力}}=\frac{p_0/(\rho l_0)}{v_0^2/l_0}=\frac{p_0}{\rho v_0^2} Eu=对流惯性力压力​=v02​/l0​p0​/(ρl0​)​=ρv02​p0​​

                          其中的压力基准 p0p_0p0​ 可以是相对压力差或绝对压力
                          当流体的压力在流动中有影响时需要考虑, 如空泡效应; 或者作为辅助准则以导出模型与实际之间的受力关系 (压力在流场中普遍存在且, p0=F0/l02p_0=F_0/l_0^2p0​=F0​/l02​)

                          # 傅劳德准则

                          用于研究流场的重力对于流场运动的影响
                          此时认为流场仅受重力, 因此 ∣f⃗∣=g|\vec{f}|=g∣f​∣=g

                          Fr2=对流惯性力重力=v02/l0g=v02gl0 Fr^2=\frac{\text{对流惯性力}}{\text{重力}}=\frac{v_0^2/l_0}{g}=\frac{v_0^2}{g l_0} Fr2=重力对流惯性力​=gv02​/l0​​=gl0​v02​​

                          通常用于研究船舶受到的波浪阻力以及大坝溢流

                          # 斯特罗哈准则

                          用于研究流场的局部惯性力对于流场运动的影响

                          St=局部惯性力对流惯性力=v0/t0v02/l0=l0v0t0 St=\frac{\text{局部惯性力}}{\text{对流惯性力}}=\frac{v_0/t_0}{v_0^2/l_0}=\frac{l_0}{v_0 t_0} St=对流惯性力局部惯性力​=v02​/l0​v0​/t0​​=v0​t0​l0​​

                          斯特罗哈准则在非时变流动中没有意义
                          通常用于时变流动的研究, 如阀门开关过程, 周期性力作用下的流动等

                          # 马赫准则

                          用于研究可压缩流场的弹性力 (对应不可压缩流场中的压力) 对于流场运动的影响
                          根据可压缩流体的运动可得, 参数 ppp 与 ρ\rhoρ 不再独立, 因此使用声速 c=dρdpc=\sqrt{\frac{\mathrm{d}\rho}{\mathrm{d}p}}c=dpdρ​​ 来替换这两个参数之间的关系

                          Ma2=对流惯性力弹性力=v02/l0c2/l0=v02c2 Ma^2=\frac{\text{对流惯性力}}{\text{弹性力}}=\frac{v_0^2/l_0}{c^2/l_0}=\frac{v_0^2}{c^2} Ma2=弹性力对流惯性力​=c2/l0​v02​/l0​​=c2v02​​

                          马赫准则在不可压缩流体流动中没有意义
                          通常用于可压缩流体流动的研究
