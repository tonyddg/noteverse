                                    # 流体静力学

                                    当流体之间没有相对运动时, 认为流体静止, 此时可以将流体视为刚体分析, 且流体中仅存在压力, 而不体现黏性

                                    认为相对非惯性系静止的流体依然属于静止流体, 但根据达朗贝尔原理可得, 流体中的点受到惯性力

                                    本节中均以静止的不可压缩流体为研究对象

                                    # 相对压强与绝对压强

                                    定义相对压强 pgp_gpg​ (测量 gauge), 即测压器读数得到的压强, 因此也称为测量压强

                                    当 pg<0p_g<0pg​<0 时, 压力值需要使用真空计测量, 因此有真空度定义 pv=−pgp_v=-p_gpv​=−pg​ (真空 vacuum)

                                    定义大气压强 pap_apa​ (大气 atmosphere); 当压力表的参考对象不是大气时也使用 pbp_bpb​ (背景 background)

                                    因此物体受到的实际压强 ppp 为

                                    p=pa+pg p=p_a+p_g p=pa​+pg​

                                    # 压强分布中的等压面

                                    将流体中压强相等的面定义为等压面, 等压面一般为平面, 但也可能是曲面

                                    等压面具有以下性质

                                    1. 等压面上任一点所受的质量力方向与等压面垂直
                                    2. 不同密度的液体之间, 液体与空气的分界面为等压面
                                    3. 认为空气的压强为定值, 此时液体与空气的分界面上的压强大小等于空气压强
                                    4. 仅受重力时, 在相同介质中的任意水平面均为等压面

                                    eg.

                                    截面 A,BA,BA,B 在同一介质的统一水平面上, 因此为等压面
                                    但截面 CCC 在另一介质中, 根据压强计算显然 PC≠PAP_C\neq P_APC​=PA​ 因此截面 CCC 与 A,BA,BA,B 不在同一等压面上

                                    • VVV 为曲面与液面围成的体积, 称为控制体, 忽略 p0p_0p0​ 项有取 zzz 轴以向上为正
                                      • 当壁面上方与液体接触时, FzF_zFz​ 的方向为垂直向下, 取 VVV 为负
                                      • 当壁面下方与液体接触时, FzF_zFz​ 的方向为垂直向上, 取 VVV 为正
                                      • 对于复杂曲面, 可分解为多段, 并对控制体叠加 V=∑ViV=\sum V_iV=∑Vi​
                                    • FxF_xFx​ 不能忽略, 总是从液体指向壁面
