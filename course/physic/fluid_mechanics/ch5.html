                                  # 可压缩流体的一元流动

                                  一般在研究气体动力学时需要考虑流体的压缩性
                                  空气动力学常以空气为研究对象, 同样在没有特殊说明的情况下, 均以空气作为本章的研究对象.

                                  # 热力学基础

                                  # 理想气体状态方程

                                  通常将空气作为理想气体, 此时满足理想气体状态方程

                                  pV=nRT pV=nRT pV=nRT

                                  对于流体分析, 密度参数 ρ\rhoρ 比体积参数 VVV 以及物质的量 nnn 更常用
                                  因此对上式变形有

                                  p=ρRgT(5-1) p=\rho R_g T\tag{5-1} p=ρRg​T(5-1)

                                  其中 RgR_gRg​ 为气体常数, 单位 J/(kg⋅K)J/(kg\cdot K)J/(kg⋅K), 满足

                                  Rg=RM=cp−cv R_g=\frac{R}{M}=c_p-c_v Rg​=MR​=cp​−cv​

                                  # 等熵过程的状态方程

                                  # 可压缩流体的连续性方程

                                  对于定常可压缩流体一元流动, 其连续性方程为

                                  ∇⃗⋅(ρv⃗)=0 \vec{\nabla}\cdot(\rho\vec{v})=0 ∇⋅(ρv)=0

                                  可得可压缩流体的连续性还与质量有关
                                  以此定义的 qmq_mqm​ 即质量流量, 为单位时间内通过流场特定面积 AAA 的流体质量, 满足

                                  qm=∫A(ρv⃗)dA⃗ q_m=\int\limits_A (\rho\vec{v})\mathrm{d}\vec{A} qm​=A∫​(ρv)dA

                                  使用类似不可压缩流体的处理方式可得

                                  ∮S(ρv⃗)dS⃗=∫Vdiv⁡(ρv⃗)dV=0 \oint\limits_S (\rho\vec{v})\mathrm{d}\vec{S}=\int\limits_{V}\operatorname{div}(\rho\vec{v})\mathrm{d}V=0 S∮​(ρv)dS=V∫​div(ρv)dV=0

                                  对于一条流管, 取流管两端截面 A1,A2A_1,A_2A1​,A2​, 流体仅通过这两个截面, 因此有定常可压缩流体的连续性方程

                                  ρ1v1A1=ρ2v2A2=qm(5-3) \rho_1 v_1A_1=\rho_2 v_2A_2=q_m\tag{5-3} ρ1​v1​A1​=ρ2​v2​A2​=qm​(5-3)

                                  # 绝热流动的能量方程

                                  由于空气流动中, 重力 ggg 与热交换 Δq\Delta qΔq 的影响都极小可以忽略, 以此为条件联立理想流体的运动微分方程与理想气体状态方程可得到可压缩流体绝热流动的能量方程

                                  cpT+12v2=C(5-4) c_p T+\frac{1}{2}v^2=C\tag{5-4} cp​T+21​v2=C(5-4)

                                  根据方程可得, 当知道气瓶与外界的温度差即可求出气瓶出口气流的速度 (认为气瓶内 v=0v=0v=0)

                                  # 声速

                                  假设气流收到一个微小的扰动, 由于气流可压缩, 因此扰动区域的压强变为 p+dpp+\mathrm{d}pp+dp
                                  并且扰动区域随时间不断扩大, 定义这个扰动扩大 (传播) 的速度即声速
                                  在扰动区的分界线上建立坐标系, 根据气体运动方程可推导得到声速 ccc 满足 (证明省略)

                                  c=dpdρ c=\sqrt{\frac{\mathrm{d}p}{\mathrm{d}\rho}} c=dρdp​​

                                  由于认为空气流动属于等熵过程, 因此带入有关方程可解出声速满足

                                  c=kRgT c=\sqrt{kR_gT} c=kRg​T​

                                  # 马赫数

                                  定义气流速度与当地声速的比值为马赫数

                                  Ma=vc=vkRgT Ma=\frac{v}{c}=\frac{v}{\sqrt{kR_gT}} Ma=cv​=kRg​T​v​

                                  当马赫数已知时, 有能量方程 (将滞止状态作为参考), 注意速度前的 12\frac{1}{2}21​ 系数

                                  cpT+12Ma2kRgT=cpT0T=cpT0cp+12Ma2kRg \begin{split} c_pT+\frac{1}{2}Ma^2kR_gT&=c_pT_0\\ T&=\frac{c_pT_0}{c_p+\frac{1}{2}Ma^2kR_g} \end{split}cp​T+21​Ma2kRg​TT​=cp​T0​=cp​+21​Ma2kRg​cp​T0​​​

                                  注意

                                  • 当温度不同, 相同的气流速度 vvv 对应的马赫数不同 (当地声速与温度有关)
                                  • 由可压缩流体的运动方程可得, 对于同一流体, 其温度 TTT 与速度 uuu 是一一对应的, 因此当地声度的含义是流体在速度 vvv 下的对应温度 TTT 计算得到的声速 ccc

                                  当 Ma>1Ma>1Ma>1, 流体的速度大于声速, 此时扰动仅在一个锥形区域内传播

                                  # 参考状态

                                  使用可压缩流体的运动方程时, 需要选择一些特殊的状态作为参考状态

                                  根据参考状态的定义, 可以计算出参考状态下的温度
                                  再根据等熵过程的状态方程可推导出滞止状态下的其他参数

                                  以下为两种常用的参考状态的定义

                                  # 滞止状态

                                  在滞止状态下, 气流的速度 v0=0v_0=0v0​=0, 该状态使用 000 作为下标, 此时有

                                  cpT0=cpT+12u2 c_pT_0=c_pT+\frac{1}{2}u^2 cp​T0​=cp​T+21​u2

                                  通常认为容器内的气体 v=0v=0v=0, 处于滞止状态

                                  # 临界状态

                                  在临界状态下, 气流的马赫数 Ma=1,v∗=cMa=1,v_*=cMa=1,v∗​=c, 该状态使用 ∗*∗ 作为下标, 此时有

                                  cpT∗+12v∗2=T∗(cp+12kRg)=cpT+12u2 c_pT_*+\frac{1}{2}v_*^2=T_*(c_p+\frac{1}{2}kR_g)=c_pT+\frac{1}{2}u^2 cp​T∗​+21​v∗2​=T∗​(cp​+21​kRg​)=cp​T+21​u2

                                  # 流动参数变化规律

                                  联立连续性方程与流体的运动微分方程可得到可压缩流体速度随管道截面变化满足微分方程

                                  (Ma2−1)dvv=dAA (Ma^2-1)\frac{\mathrm{d}v}{v}=\frac{\mathrm{d}A}{A} (Ma2−1)vdv​=AdA​

                                  因此流体速度与管道截面的变化规律还与马赫数有关

                                  根据可压缩流体的能量方程与绝热过程参数间关系还可得, 当速度 vvv 增大时温度 TTT 减小, 相应的压强 ppp, 密度 ρ\rhoρ 也都将减小

                                  综上可得流动参数变化规律为

                                  • 当 Ma<1Ma<1Ma<1, Ma2−1<0,dv∝−dAMa^2-1<0, \mathrm{d}v\propto -\mathrm{d}AMa2−1<0,dv∝−dA
                                    当管道截面积减小, 速度 vvv 增大; 压强 ppp, 密度 ρ\rhoρ, 温度 TTT 均减小
                                    亚声速气流在收缩管内逐渐加速, 在扩张管内逐渐减速
                                  • 当 Ma>1Ma>1Ma>1 , Ma2−1>0,dv∝dAMa^2-1>0, \mathrm{d}v\propto \mathrm{d}AMa2−1>0,dv∝dA
                                    当管道截面积减小, 速度 vvv 减小; 压强 ppp, 密度 ρ\rhoρ, 温度 TTT 均增大
                                    超声速气流在收缩管内逐渐减速, 在扩张管内逐渐加速

                                  # 缩放喷管

                                  根据流体速度变化规律, 设计合理的缩放喷管能够产生超音速气流, 也成为拉伐尔喷管.
