                                # 黏性流体总流的伯努利方程

                                # 水头形式的伯努利方程

                                由于黏性摩擦的存在, 随着流体的流动能量将不断损失, 因此伯努利方程不再成立
                                为此在伯努利方程中还需要添加一个摩擦损失项用于修正
                                对于黏性流体的研究通常使用水头形式的伯努利方程, 以方便地反映出流体在黏性摩擦下的能量损失

                                z1+p1ρg+αv1ˉ22g=z2+p2ρg+αv2ˉ22g+hw z_1+\frac{p_1}{\rho g}+\alpha\frac{\bar{v_1}^2}{2g}=z_2+\frac{p_2}{\rho g}+\alpha\frac{\bar{v_2}^2}{2g}+h_w z1​+ρgp1​​+α2gv1​ˉ​2​=z2​+ρgp2​​+α2gv2​ˉ​2​+hw​

                                其中项 hwh_whw​ 即水头损失, 反映了流体流动时损失的能量
                                如图所示, hwh_whw​ 可通过管道上两处测压管高度差直观地测量出来

                                # 水头损失

                                水头损失 hwh_whw​ (waste) 有两种形式, 分别为沿程水头损失 hfh_fhf​ (friction) 与局部水头损失 hjh_jhj​ (局部)

                                对于截面 1,21,21,2 之间的水头损失为两截面之间所有水头损失之和 (单向管)

                                hw=∑hf+∑hj h_w=\sum h_{f}+\sum h_{j} hw​=∑hf​+∑hj​

                                # 沿程水头损失

                                沿程水头损失为来自管壁与流体之间摩擦导致的损失, 其大小与管道长度 lll 成正比, 与管道直径 ddd 成反比, 满足

                                hf=λldvˉ22g h_f=\lambda\frac{l}{d}\frac{\bar{v}^2}{2g} hf​=λdl​2gvˉ2​

                                其中 vˉ22g\frac{\bar{v}^2}{2g}2gvˉ2​ 项为动能参照, 反映了损失的能量与动能大小成正比
                                λ\lambdaλ 为沿程损失系数, 无量纲, 与流体的状态, 特别是速度有关

                                # 局部水头损失

                                局部水头损失为来自流动遇到的阻碍导致的损失, 通常出现在管道的连接处, 与管道形状有关, 满足

                                hf=ξldvˉ22g h_f=\xi\frac{l}{d}\frac{\bar{v}^2}{2g} hf​=ξdl​2gvˉ2​

                                其中 ξ\xiξ 为局部损失系数, 无量纲, 与管道形状有关, 且对于同一处管道, 流动方向不同, ξ\xiξ 也不同

                                # 雷诺数

                                对于不同流动状态, 流速在管道中的分布有很大的差别, 通过无量纲的参数雷诺数 ReReRe 来区分这些状态

                                Re=vˉdν Re=\frac{\bar{v}d}{\nu} Re=νvˉd​

                                雷诺数越小, 黏性力的作用越大, 流动为层流状态, 流体根据速度梯度 dvdh\frac{\mathrm{d}v}{\mathrm{d}h}dhdv​ 分层
                                雷诺数足够大时, 粘性力作用减小, 对流惯性力发挥主要作用, 各流层之间存在剧烈扰动, 速度趋于平均

                                # 层流流动

                                当 Re<2300Re<2300Re<2300, 认为流动为层流, 此时管截面上的速度分布是旋转抛物面

                                取截面上的流体速度满足

                                vˉ=vmax⁡2 \bar{v}=\frac{v_{\max}}{2} vˉ=2vmax​​

                                根据总流的伯努利方程中动能修正系数 α\alphaα 的定义可得, 对于层流流体满足

                                α=2 \alpha=2 α=2

                                根据泊肃叶定理可推导得到, 层流流动的沿程损失系数为

                                λ=64Re \lambda=\frac{64}{Re} λ=Re64​

                                此时沿程水头损失与流速的一次方成正比, hf∝vˉh_f\propto \bar{v}hf​∝vˉ

                                # 湍流流动

                                当 2300<Re<40002300<Re<40002300<Re<4000, 认为流动为层流向湍流过渡, 没有明确流动的规律

                                当 4000<Re4000<Re4000<Re, 流动为湍流, 其速度分布具有如图所示的规律

                                • 黏性底层的速度成线性分布, 并且随着雷诺数 ReReRe 的增大而减小
                                • 湍流核心区的速度分布有对数函数的形式, 也可认为此处的速度平均分布, 因此 α≈1\alpha\approx 1α≈1

                                # 管壁粗糙度

                                根据试验可得, 定义管壁的粗糙度 Δd\frac{\Delta}{d}dΔ​, 其中 Δ\DeltaΔ 为管壁粗糙物的平均凸出高度

                                管壁粗糙物对与层流以及近似层流的黏性底层影响较小, 但对于湍流核心区有显著影响
                                现根据黏性底层厚度与管壁的粗糙度将湍流流动分为湍流水力光滑管与湍流水力粗糙管

                                因此此时的沿程水头损失与速度的 1.751.751.75 次方成正比, hf∝vˉ1.75h_f\propto \bar{v}^{1.75}hf​∝vˉ1.75

                                # 湍流水力粗糙管

                                当 4160(d2Δ)0.85<Re4160(\frac{d}{2\Delta})^{0.85}<Re4160(2Δd​)0.85<Re, 认为是湍流水力粗糙管, 粗糙物露出黏性底层, 类似湍流在粗糙管道中, 此时沿程损失系数满足

                                # 水泵问题

                                水泵的工作原理为在入水口处产生一个真空压强的方式对流体做功, 从而将水运送到高处
                                此时介质不连续, 因此不能直接使用伯努利方程
                                使用水泵扬程 HpH_pHp​ 来衡量水泵产生并补充到下游的能量, 经此修正后上下游水池的伯努利方程为

                                paρg+Hp=paρg+H+hw \frac{p_a}{\rho g}+H_p=\frac{p_a}{\rho g}+H+h_w ρgpa​​+Hp​=ρgpa​​+H+hw​

                                其中

                                • HHH 为上方水池相对于下方水池的高度差, 也称为扬水高程
                                • hwh_whw​ 为水泵在传输过程中的总水头损失
                                • HpH_pHp​ 即水泵扬程

                                根据公式也可得

                                • 当流体经过水泵时, 连续性条件不满足, 需要对下游部分的伯努利方程补充扬程, 且扬程满足

                                Hp=H+hw H_p=H+h_w Hp​=H+hw​

                                • 扬程本质即水泵做功, 将水头折算为能量, 并带入流量后可得水泵的功率为

                                P=ρgQHpη P=\frac{\rho g Q H_p}{\eta} P=ηρgQHp​​
