                              # 正弦稳态与三相电路分析

                              # 正弦稳态电路

                              # 相量法

                              # 正弦电量

                              u=Umcos(ωt+ϕ) u=U_m cos(\omega t+\phi) u=Um​cos(ωt+ϕ)

                              1. uuu 瞬时值
                              2. UmU_mUm​ 最大值
                              3. ω\omegaω 角频率
                              4. ϕ\phiϕ 初相位
                              5. U=Um2U=\frac{U_m}{\sqrt{2}}U=2​Um​​ 有效值(电表测量结果)

                              # 相量法

                              U˙=U∠ϕ \dot{U}=U\angle\phi U˙=U∠ϕ

                              1. 相量中实部使用的是有效值, 转成瞬时值需要化为最大值
                              2. 使用相量的实部体现瞬时值
                              3. 基尔霍夫定律依然满足, 但需要使用相量计算
                              # 相量分析法
                              1. 通过画图, 结合相量之间的几何关系解题
                              2. 分析前选择一个相量为参考, 令其相位角为 0∘0^\circ0∘
                              3. 参考向量选择
                                1. 电路已知条件最多的部分
                                2. 在电路最内部
                                3. 电阻上的 I˙\dot{I}I˙(串) 或 U˙\dot{U}U˙(并)

                              # 阻抗

                              # 元件阻抗

                              # 电阻

                              Z=R Z=R Z=R

                              # 电感

                              Z=jωL Z=j\omega L Z=jωL

                              # 电容

                              Z=1jωC Z=\frac{1}{j\omega C} Z=jωC1​

                              # 阻抗性质

                              U˙=I˙Z \dot{U}=\dot{I}Z U˙=I˙Z

                              1. 阻抗依然满足串联与并联
                              2. 感性阻抗 ϕ>0\phi>0ϕ>0, U˙\dot{U}U˙ 超前
                              3. 容性阻抗 ϕ<0\phi<0ϕ<0, U˙\dot{U}U˙ 滞后
                              4. 由 1 得阻抗角度也体现 U˙\dot{U}U˙ 与 I˙\dot{I}I˙ 的夹角

                              # 功率

                              1. 功率计算所用到的均为有效值
                              2. 由功率求得 UUU, III 后还需转化为相量
                              3. 公式中均使用参考方向
                              4. 除视在功率均满足功率守恒

                              # 有功功率

                              P=UIcos(ϕU−ϕI) P=UIcos(\phi_U-\phi_I) P=UIcos(ϕU​−ϕI​)

                              单位 WWW

                              # 无功功率

                              Q=UIsin(ϕU−ϕI) Q=UIsin(\phi_U-\phi_I) Q=UIsin(ϕU​−ϕI​)

                              单位 varvarvar

                              # 复功率

                              Sˉ=P+jQ=ZI2=U˙I˙‾ \bar{S}=P+jQ=ZI^2=\dot{U}\overline{\dot{I}} Sˉ=P+jQ=ZI2=U˙I˙

                              单位为 V⋅AV\cdot AV⋅A
                              元件复功率相位角即阻抗相位角

                              # 视在功率

                              S=UI S=UI S=UI

                              单位为 V⋅AV\cdot AV⋅A

                              S=P2+Q2 S=\sqrt{P^2+Q^2} S=P2+Q2​

                              视在功率不满足功率守恒

                              # 功率因数

                              λ=PS=cos(ϕU−ϕI) \lambda=\frac{P}{S}=cos(\phi_U-\phi_I) λ=SP​=cos(ϕU​−ϕI​)

                              功率因数即阻抗角的余弦值
                              阻抗角可能取正或负, 通过成感性(正)/容性(负)确定

                              # 功率因数矫正

                              1. 前提: 不改变负载的工作状态(电压)
                              2. 通过并联电容, 提高端口的功率因数
                              3. 分析时令 U˙\dot{U}U˙ 的相位为 000, 则 I˙\dot{I}I˙ 的相位为 −ϕ-\phi−ϕ
                              4. 向 yyy 轴方向投影 I˙\dot{I}I˙, 比较其与目标功率因数下投影大小的差值(仅增加电容, xxx 方向投影不会改变)
                              5. 由于并联分流, 因此此差值即矫正电容分压大小(端口电压不变)

                              # 最大功率传输

                              • 当负载阻抗与网络戴维南支路的等效阻抗满足共轭关系时, 负载获得的功率最大

                              ZL=Zeq‾ Z_L=\overline{Z_{eq}} ZL​=Zeq​​

                              • 最大有功功率为

                              PLmax=Uoc24Req P_{Lmax}=\frac{U_{oc}^2}{4R_{eq}} PLmax​=4Req​Uoc2​​

                              其中 UocU_{oc}Uoc​ 为等效电路的电压源有效值, ReqR_{eq}Req​ 为等效阻抗的实部

                              • 当负载为纯电阻时, 获得最大功率的条件为

                              RL=∣Zeq∣ R_L=|Z_{eq}| RL​=∣Zeq​∣

                              # 易错知识点

                              1. 阻抗 XXX 一定大于零, ZZZ 相位角一定在 −90∘<θ<90∘-90^{\circ}<\theta<90^{\circ}−90∘<θ<90∘ 范围内
                              2. 结合 111, 当仅电量的模长已知时, 可通过 KCLKCLKCL/KVLKVLKVL 结合相量图, 提前求出电量的方向
                              3. 相量的模长为有效值, 直接使用其 xxx 方向投影无意义

                              # 三相电路

                              1. 对称三相电压中, 满足 U˙A+U˙B+U˙C=0\dot{U}_A+\dot{U}_B+\dot{U}_C=0U˙A​+U˙B​+U˙C​=0
                              2. U˙A,U˙B,U˙C\dot{U}_A,\dot{U}_B, \dot{U}_CU˙A​,U˙B​,U˙C​ 按顺时针顺序以 120∘120^{\circ}120∘ 夹角排列
                              3. 相电压 UxpU_{xp}Uxp​ 仅含 xxx 与中性点 NNN 的电压, 因此也可写成 UANU_{AN}UAN​
                              4. 线电压 UxlU_{xl}Uxl​ xxx 与 yyy(下一相) 对应节点之间的电压, 因此也可写成 UABU_{AB}UAB​
                              5. 相电流 IxpI_{xp}Ixp​ 线电流在 UxlU_{xl}Uxl​ xxx 与 yyy(下一相) 的 xxx 分流电流中
                              6. 线电流 IxlI_{xl}Ixl​ xxx 端口上的总电流

                              # 星形联结(Y形)

                              • 相线电压满足关系

                              U˙xl=3U˙xp∠30∘ \dot{U}_{xl}=\sqrt{3}\dot{U}_{xp}\angle{30^{\circ}} U˙xl​=3​U˙xp​∠30∘

                              • 相线电流相同(区分无意义)

                              # 三角形联结(Δ\DeltaΔ形)

                              • 相线电压相同(区分无意义)
                              • 相线电流满足关系

                              I˙xl=3I˙xp∠−30∘ \dot{I}_{xl}=\sqrt{3}\dot{I}_{xp}\angle{-30^{\circ}} I˙xl​=3​I˙xp​∠−30∘

                              • 三角形联结中, 电压源正负极首尾相接, 按顺时针顺序排列

                              # 分相计算

                              此处公式仅适用于对称电路, 此时三相的电流电压关系完全相同, 负载大小相同

                              # Y-Y

                              1. 取出其中一相与中性点构成回路, 计算此回路中的电量
                              2. 无论中性点上电阻取何值(短路或开路), 中性点上电流为 0, 且为等势点
                              3. 单相中与中性点之间的电压 UaNU_{aN}UaN​ 即单相上结点的相电压
                              4. 通过公式转换即可得到线电压(不同相之间对应节点的电压) UabU_{ab}Uab​(或已知线电压, 计算相电压)
                              5. 单相中的电流即线电流, 对于 Y形 电路, 区分无意义
                              6. 等效回路中两个非中性点之间的电压无法转换(无意义)

                              # Y-Δ\DeltaΔ

                              • 通过将 Δ\DeltaΔ 形负载转化为等效的 YYY 形负载
                              • 由于负载对称, 满足关系

                              ZLΔ=13ZLY Z_{L\Delta}=\frac{1}{3}Z_{LY} ZLΔ​=31​ZLY​

                              • 通过单个负载的电流计算公式等价于相电流的转换公式, 对于 YYY 型电源, 其相电流等于线电流

                              eg.

                              此电路中, 等效单相回路为两个负载并联, 且 Δ\DeltaΔ 负载上的电流为线电流 Ial1I_{al1}Ial1​ , 单个负载上通过的实际电流为相电流 Iap1I_{ap1}Iap1​, 计算时需要转换

                              # Δ\DeltaΔ-Δ\DeltaΔ

                              1. 将Δ\DeltaΔ 形负载转化为等效的 YYY 形负载, 再将 Δ\DeltaΔ 形电压源转化为等效的 YYY 形电压源
                              2. Δ\DeltaΔ 形电压源中的单相电压即线电压(电压源均在两个端口之间), 将其转为相电压即可使用分相法计算

                              # 对称三相电路的功率

                              # 功率公式

                              • 与一般正弦电路功率不同, 三相电路的总功率为

                              P=3UpIpcosϕ=3UlIlcosϕ P=3U_{p}I_{p}cos\phi=\sqrt{3}U_{l}I_{l}cos\phi P=3Up​Ip​cosϕ=3​Ul​Il​cosϕ

                              • 其他类型功率的计算类似, 均使用 3UpIp3U_{p}I_{p}3Up​Ip​ 或 3UlIl\sqrt{3}U_{l}I_{l}3​Ul​Il​
                              • 功率因数, 其中 ϕ\phiϕ 为负载阻抗角

                              λ=cosϕ=PS \lambda=cos\phi=\frac{P}{S} λ=cosϕ=SP​

                              • 由于三相对称, 因此单相功率即

                              P单=UpIpcosϕ P_{\text{单}}=U_{p}I_{p}cos\phi P单​=Up​Ip​cosϕ

                              # 计算注意

                              • 由于 YYY 型电路中 IlI_lIl​ 与 IpI_pIp​ 等价, 因此分相法中可直接使用 P=3UpIlcosϕP=3U_{p}I_{l}cos\phiP=3Up​Il​cosϕ(不能直接对 Δ\DeltaΔ 负载使用)
                              • 分相法中的欧姆定律

                              U˙p=ZI˙l \dot{U}_p=Z\dot{I}_l U˙p​=ZI˙l​

                              • 题目中的电机功率均为三相总功率, 注意区分总功率(默认, 需要乘上系数)与单相功率

                              # 三相电路功率测量

                              瓦特表无星号线测量的是电流流出端 / 电压负极

                              # 三瓦特表法
                              1. 用于测量四线制三相电路 P=P1+P2+P3P=P_1+P_2+P_3P=P1​+P2​+P3​, 如果对称则 P1=P2=P3P_1=P_2=P_3P1​=P2​=P3​
                              2. 瓦特表的引出线接在 NNN 上
                              # 两瓦特表法
                              • 用于测量三线制三相电路, 或对称线制三相电路, P=P1+P2P=P_1+P_2P=P1​+P2​, P1≠P2P_1\neq P_2P1​=P2​
                              • 瓦特表的引出线接在无瓦特表的相上

                              • P1=Re[U˙acI˙‾a]≠P2=Re[U˙bcI˙‾b] P_1=Re[\dot{U}_{ac}\overline{\dot{I}}_{a}]\neq P_2=Re[\dot{U}_{bc}\overline{\dot{I}}_{b}] P1​=Re[U˙ac​I˙a​]=P2​=Re[U˙bc​I˙b​]

                              # 非对称三相电路

                              电源对称时可使用以下方法:

                              1. 有无负载的中性线 NNN 时, 仍可用分相法, 但三相要单独计算
                              2. 其他情况可用节点分析法, 以 NNN 为接地点
                              3. 线相电压转换关系依然满足

                              # 易错知识点

                              • 两表法测功率中

                              P1≠P2 P_1\neq P_2 P1​=P2​

                              P1=Re[U˙acI˙‾a]≠P2=Re[U˙bcI˙‾b] P_1=Re[\dot{U}_{ac}\overline{\dot{I}}_{a}]\neq P_2=Re[\dot{U}_{bc}\overline{\dot{I}}_{b}] P1​=Re[U˙ac​I˙a​]=P2​=Re[U˙bc​I˙b​]

                              当公式中出现 U˙ac\dot{U}_{ac}U˙ac​ 其值与 U˙ca\dot{U}_{ca}U˙ca​ 相反, U˙ac=−U˙ca=−U˙ab∠120∘\dot{U}_{ac}=-\dot{U}_{ca}=-\dot{U}_{ab}\angle 120^{\circ}U˙ac​=−U˙ca​=−U˙ab​∠120∘
