                                    # 基础电路分析

                                    # 基尔霍夫定律

                                    对于任何复杂的电路 (半导体, 电容, 电感), 基尔霍夫定律依然成立

                                    # KVL

                                    对于任意回路, 回路上各元件的电压之和为 000

                                    # KCL

                                    对于任意结点, 包括广义节点, 流入结点的电流等于流出结点的电流

                                    # 等效电路

                                    # 等效的含义

                                    1. 等效电路并不是替代, 而是将一个复杂的二端口的电路等效为一个由电源与电阻组成的电路
                                    2. 等效前后的电路的端口具有完全相同伏安特性
                                    3. 但两个等效电路的内部没有任何关系, 不能用于计算

                                    # 纯电阻等效

                                    1. 串联电阻等效为电阻之和
                                    2. 并联电导等效为电导之和, 即导数关系

                                    # 理想电源

                                    1. 与理想电流源串联的任何元件均可等效为单个理想电流源, 即串联电路电流相同
                                    2. 与理想电压源并联的任何元件均可等效为单个理想电压源, 即并联电路电压相同
                                    3. 理想电压源与电阻串联可等效为理想电流源与电阻并联, 满足

                                    IsR=Us,  Ri=Ru=R I_sR=U_s,\;R_i=R_u=R Is​R=Us​,Ri​=Ru​=R

                                    # 开路的处理

                                    1. 将开路部分视为一个元件, 由于导线被截断, 不可能有电流传输, 因此可将开路视为一个 I=0I=0I=0 的理想电流源
                                    2. 对外分析时, 可将开路所在支路完全等效为断开
                                    3. 对内分析时
                                      1. 由于没有电流, 串联电阻上电压必定为 000
                                      2. 当支路内有电压源, 则需要使用 KVLKVLKVL 或结点分析法具体分析开路两端的电压
                                    4. 对于以下情况可以视为开路
                                      1. 计算电路的输出电压
                                      2. 二极管反向截止
                                      3. 直流稳态电路中电容
                                      4. 电压表 / 示波器端口

                                    # 分析实例

                                    假设二极管 D2D2D2 反向截止, 求示波器 AAA 的电压

                                    1. 由于二极管 D2D2D2 反向截止, 视为开路
                                    2. ACACAC 段与示波器无关, 以此可以整条支路视为开路
                                    3. 剩余回路中, 三个元件为串联关系, 示波器 AAA 为开路, 以此电阻上分压为 000
                                    4. 使用 KVLKVLKVL 可得, 示波器 AAA 上的电压与电源相同

                                    # 短路的处理

                                    1. 将短路部分视为一个元件, 可以认为是一个 0Ω0\Omega0Ω 电阻, 不可能有电压, 因此可将短路视为一个 U=0U=0U=0 的理想电压源
                                    2. 对外分析时, 可将与短路并联的支路同样视为短路
                                    3. 对内分析时
                                      1. 由于没有电流, 并联电阻上电流必定为 000
                                      2. 当并联支路内有电流源, 则需要使用 KCLKCLKCL 具体分析短路上的电压

                                    # 电路分析方程

                                    # 结点方程

                                    # 快速法

                                    1. 即对结点使用 KCLKCLKCL
                                    2. 通过 ΔU结点⋅R支路\Delta U_{\text{结点}}\cdot R_{\text{支路}}ΔU结点​⋅R支路​ 表示非电流源支路的电流
                                    3. 可以对单个结点使用, 快速分析结点的电压

                                    # 整体法

                                    1. 将电路化为最简, 并且将电压源等效为电流源 (否则两个串联的元件之间有一个结点)
                                    2. 确定结点电导矩阵 GGG
                                      1. 标定电路图上的结点, 并为结点标号, 并选择接地点
                                      2. 以结点相连各支路的电导 (电阻倒数) 之和为自电导 GnnG_{nn}Gnn​
                                      3. 相邻结点之间的电导为互电导 GnmG_{nm}Gnm​, 因此最终可以得到一个对称矩阵
                                    3. 确定电流源列向量 IsI_sIs​, 流入结点的电流源取正
                                    4. 未知数为结点电压向量 U=UiU={U_i}U=Ui​, 得到方程

                                    GU=Is GU=I_s GU=Is​

                                    # 网孔方程

                                    # 整体法

                                    1. 将电路化为最简, 并且将电流源等效为电压源 (否则两个并联的元件之间有一个网点)
                                    2. 确定网孔电阻矩阵 RRR
                                      1. 标定电路图上的网孔, 并为网孔标号, 确定网孔电流方向
                                      2. 以网孔上所有元件的电阻之和为自电阻 RnnR_{nn}Rnn​
                                      3. 相邻网孔之间的支路上的电阻为互电阻 RnmR_{nm}Rnm​, 因此最终可以得到一个对称矩阵
                                    3. 确定电流源列向量 UsU_sUs​, 与网孔电流为非关联的电压源取正
                                    4. 未知数为网孔电流向量 I=IiI={I_i}I=Ii​, 得到方程

                                    RI=Us RI=U_s RI=Us​

                                    # 在非线性电路的应用

                                    1. 根据替代定理, 当知道支路的电压后, 可将其视为电压源处理 (用于处理半导体)
                                    2. 复杂电路通常使用结点法表示电压 VVV, 可用快速法列写部分结点方程 (不适合整体使用)
                                    3. 复杂电路通常使用电压源, 因此适合使用网孔方程对整体分析

                                    # 易错知识点

                                    # 等效电阻

                                    1. 计算等效电阻时要将电源置零
                                    2. 置零后使用节点法时注意检查, 电压源置零后为短路, 节点可能减少

                                    # 开路电压(等效电压)

                                    计算开路电压时, 不能因为两条与端口相连的支路上没有构成回路就忽视. 如果此支路上存在电压/电流源(受控源)要考虑, 并且通过 KVL 计算开路电压

                                    eg. 端口支路存在电压源/电流源后, 与地线的电压不为零
