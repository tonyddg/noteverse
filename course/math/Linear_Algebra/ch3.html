                                                    # 关于向量空间的补充与应用

                                                    参考教程 麻省理工公开课 线性代数open in new window P9~P13

                                                    # 向量空间的属性补充

                                                    向量空间的其他属性与定义

                                                    # 线性无关性 (Linear Independence)

                                                    对于一个向量组 x1⃗,x2⃗,…,xn⃗{\vec{x_1},\vec{x_2},\dots,\vec{x_n}}x1​​,x2​​,…,xn​​, 当除了系数全为零 (零组合) 的情况下, 向量组中的向量不存在任何线性组合能得到零向量 0⃗\vec{0}0, 则该向量组线性无关

                                                    否则称此向量组线性相关, 例如向量组 {v⃗,2v⃗}\{\vec{v},2\vec{v}\}{v,2v}, {0⃗,v⃗}\{\vec{0}, \vec{v}\}{0,v}, {v1⃗,v2⃗,v3⃗}(vn⃗∈R2)\{\vec{v_1},\vec{v_2},\vec{v_3}\}(\vec{v_n}\in R^2){v1​​,v2​​,v3​​}(vn​​∈R2)

                                                    # 与矩阵的联系

                                                    可将向量组作为列向量并组成 m×nm\times nm×n 的矩阵 AAA, 当向量组线性无关时, 矩阵 AAA 将具有以下的特征

                                                    • 当矩阵 AAA 的零空间 N(A)N(A)N(A) 内仅有零向量, 则此向量组线性无关
                                                    • 矩阵 AAA 的秩满足 r=nr=nr=n, 矩阵不存在自由列

                                                    # 生成空间与基 (Basis)

                                                    一个向量组生成一个空间 SSS , 表示这个空间内的向量包含了向量组内向量的所有线性组合.

                                                    定义向量空间的基为一个具有以下两个特性的向量组

                                                    1. 作为基的向量组线性无关
                                                    2. 这些向量组能够生成这个向量空间

                                                    # 向量空间的维数 (Dimension)

                                                    对于任意一个向量空间, 可能有无数个基, 但是这些基中的向量数量一定相同. 例如, 对于向量空间 RnR^nRn, 其基一定有 nnn 个向量. 将这个基所需要的向量个数定义为空间的维数, 记为 dim[C(A)]dim[C(A)]dim[C(A)]

                                                    对于任意向量空间, 其中至少有一个零向量
                                                    在这种情况下, 认为该向量空间的维数为 000

                                                    # 四个基本子空间

                                                    对于一个 m×nm\times nm×n 的矩阵 AAA, 其秩为 rrr
                                                    定义四个基本子空间

                                                    • 列空间 C(A)C(A)C(A)
                                                    • 零空间 N(A)N(A)N(A)
                                                    • 行空间 C(AT)C(A^{T})C(AT)
                                                    • 左零空间 N(AT)N(A^{T})N(AT)

                                                    这四个基本子空间的基能够在一次 AAA 消元得到 RRR 的过程 [A∣I]→[R∣E][A|I]\to [R|E][A∣I]→[R∣E] 中全部得出

                                                    # 列空间

                                                    将来自矩阵 AAA 的列向量线性组合产生的向量空间称为列空间
                                                    表示为 C(A)C(A)C(A)

                                                    列空间中的向量均属于 RmR^mRm, 显然 C(A)C(A)C(A) 为 RmR^mRm 的子空间

                                                    C(A)⊆Rm C(A)\subseteq R^m C(A)⊆Rm

                                                    # 列空间的维数

                                                    由于秩确定了矩阵中独立列向量的个数, 显然列空间的维数即矩阵 AAA 的秩 rrr

                                                    dim⁡[C(A)]=r \dim[C(A)]=r dim[C(A)]=r

                                                    # 生成列空间的基

                                                    矩阵 AAA 经过初等矩阵消元与行交换得到简化行阶梯矩阵 RRR
                                                    现引入左零空间基求解中的矩阵 EEE

                                                    对于同一个线性组合 x⃗\vec{x}x, Ax⃗=b⃗≠EAx⃗=Eb⃗A\vec{x}=\vec{b}\neq EA\vec{x}=E\vec{b}Ax=b=EAx=Eb
                                                    显然有 C(A)≠C(R)C(A)\neq C(R)C(A)=C(R), 但由于 RRR 没有经过列交换
                                                    因此两个矩阵在线性组合中发挥作用的列相同

                                                    对于 RRR 显然其 III 部分的矩阵块中的列向量足以组合得到 C(R)C(R)C(R)
                                                    因此在 AAA 中同样位置的列向量也足以组合得到 C(A)C(A)C(A)
                                                    这些列也即 AAA 的主元列

                                                    综上所述, AAA 中的主元列即构成了列空间 C(A)C(A)C(A) 的一组基

                                                    # 零空间

                                                    将方程 Ax⃗=0⃗A\vec{x}=\vec{0}Ax=0 的解 x⃗\vec{x}x 构成的空间定义为零空间 N(A)N(A)N(A)
                                                    也可以说, 使矩阵 AAA 的列向量线性组合得到 0⃗\vec{0}0 的向量组成零空间

                                                    参与乘法 x⃗\vec{x}x 必定属于 RnR^nRn, 显然 N(A)N(A)N(A) 为 RnR^nRn 的子空间

                                                    N(A)⊆Rn N(A)\subseteq R^n N(A)⊆Rn

                                                    # 零空间的维数

                                                    观察零空间矩阵, 其各列线性无关, 列数为 n−rn-rn−r
                                                    零空间矩阵的列空间与 AAA 零空间相同, 因此零空间维数即矩阵 AAA 的自由变量个数 n−rn-rn−r

                                                    dim⁡[N(A)]=n−r \dim[N(A)]=n-r dim[N(A)]=n−r

                                                    # 生成零空间的基

                                                    对于零空间矩阵, 其各列线性无关
                                                    其列空间 C(N)C(N)C(N) 又与零空间 N(A)N(A)N(A) 相同

                                                    因此, 易得矩阵 AAA 的零空间矩阵中的列向量即零空间 N(A)N(A)N(A) 的一组基

                                                    # 行空间

                                                    将来自矩阵 AAA 的行向量线性组合产生的向量空间称为行空间
                                                    易得, AAA 的行空间即 ATA^TAT 的列空间, 因此表示方法为 C(AT)C(A^T)C(AT)

                                                    行空间中的向量均属于 RnR^nRn, 显然 C(AT)C(A^T)C(AT) 为 RnR^nRn 的子空间

                                                    C(AT)⊆Rn C(A^T)\subseteq R^n C(AT)⊆Rn

                                                    # 行空间的维数

                                                    矩阵 AAA 在转置后的秩依然为 rrr, 因此行空间的维数也为矩阵 AAA 的秩 rrr

                                                    dim⁡[C(AT)]=r \dim[C(A^T)]=r dim[C(AT)]=r

                                                    # 生成行空间的基

                                                    当矩阵 AAA 左乘消元矩阵得到简化行阶梯矩阵 RRR 的过程中
                                                    本质上为 AAA 的各行之间线性组合, RRR 中的各行来自 AAA 的线性组合
                                                    因此, 在这个过程中行空间始终不变 (显然列空间不一定相同)
                                                    A,RA,RA,R 矩阵具有相同的行空间 C(AT)=C(RT)C(A^T)=C(R^T)C(AT)=C(RT)

                                                    结合 RRR 的性质综上可得, 简化行阶梯矩阵 RRR 中的非 000 行向量可以构成行空间 C(AT)C(A^T)C(AT) 的一组基
                                                    也将来自 RRR 行向量的基称为行空间 C(AT)C(A^T)C(AT) 的最佳基

                                                    # 左零空间

                                                    将方程 y⃗TA=0⃗\vec{y}^T A=\vec{0}y​TA=0 的解 y⃗T\vec{y}^Ty​T 构成的空间定义为左零空间
                                                    由 (y⃗TA)T=ATy⃗(\vec{y}^T A)^{T}=A^{T}\vec{y}(y​TA)T=ATy​ 可得, 左零空间与 ATA^TAT 的零空间相同, 因此表示方法为 N(AT)N(A^T)N(AT)
                                                    也可以说, 使矩阵 AAA 的行向量线性组合得到 0⃗\vec{0}0 的向量组成零空间

                                                    参与乘法 y⃗T\vec{y}^Ty​T 必定属于 RmR^mRm, 显然 N(AT)N(A^T)N(AT) 为 RmR^mRm 的子空间

                                                    N(AT)⊆Rm N(A^{T})\subseteq R^m N(AT)⊆Rm

                                                    # 左零空间的维数

                                                    由零空间推广可得, 左零空间维数即矩阵 ATA^TAT 的自由变量个数 m−rm-rm−r

                                                    dim⁡[N(AT)]=m−r \dim[N(A^T)]=m-r dim[N(AT)]=m−r

                                                    # 生成左零空间的基

                                                    参考矩阵求逆的方法, 对于增广矩阵 [A∣I][A|I][A∣I], 在将 AAA 经过换行(初等矩阵左乘)为 RRR 的过程中有 [A∣I]→[R∣E][A|I]\to[R|E][A∣I]→[R∣E]

                                                    其中 EEE 为 m×mm\times mm×m 的矩阵, 类似于矩阵的逆, 有 EA=REA=REA=R
                                                    并且 EEE 来自初等矩阵相乘, 因此必定为可逆矩阵, 其各行线性无关

                                                    观察到, RRR 下方 m−rm-rm−r 行为零向量 0⃗\vec{0}0 (见简化行阶梯矩阵的结构)
                                                    这 m−rm-rm−r 行零向量 0⃗\vec{0}0 又来自 EEE 下方 m−rm-rm−r 行的行向量与 AAA 的行向量线性组合
                                                    因此矩阵 EEE 末尾的 m−rm-rm−r 行向量必定属于左零空间 (E)r=i∈N(AT),(i=r,…m)(E)_{r=i}\in N(A^T),(i=r,\dots m)(E)r=i​∈N(AT),(i=r,…m)

                                                    由于 EEE 自身的线性无关性以及左零空间的维数, 因此矩阵 EEE 的第 rrr 行到第 mmm 行向量 (末尾的 m−rm-rm−r 行) 构成了左零空间 N(AT)N(A^T)N(AT) 的一组基

                                                    # 例题

                                                    对于集合 SSS, 集合中的向量 v⃗\vec{v}v 的元素总是满足 v1+v2+v3+v4=0v_1+v_2+v_3+v_4=0v1​+v2​+v3​+v4​=0
                                                    证明集合 SSS 为一个向量空间, 并求出该向量空间的维数与基

                                                    # 向量空间证明

                                                    首先以零向量检查线性性, 显然零向量满足集合 SSS 的要求

                                                    然后根据向量空间线性性的要求可得

                                                    a(v1+v2+v3+v4)+b(v1′+v2′+v3′+v4′)=a⋅0+b⋅0=0 a(v_1+v_2+v_3+v_4)+b(v_1'+v_2'+v_3'+v_4')=a\cdot 0+b\cdot 0=0 a(v1​+v2​+v3​+v4​)+b(v1′​+v2′​+v3′​+v4′​)=a⋅0+b⋅0=0

                                                    因此集合 AAA 为一个向量空间

                                                    # 题目分析

                                                    根据 v⃗\vec{v}v 的条件可得其满足

                                                    Av⃗=0,A=[1111] A\vec{v}=0,A=\begin{bmatrix}1&1&1&1\end{bmatrix} Av=0,A=[1​1​1​1​]

                                                    虽然 AAA 仅有一行, 但依然可将其作为一个 1×41\times 41×4 的矩阵处理, 因此 SSS 即 AAA 的零空间 S=N(A)S=N(A)S=N(A)

                                                    # 向量空间的维数与基

                                                    根据零空间的维数与基即可求出 SSS 的维数与基

                                                    显然 AAA 的秩 r=1r=1r=1, 因此零空间的维数为 4−r=3=dim(S)4-r=3=dim(S)4−r=3=dim(S)

                                                    根据零空间矩阵的求法
                                                    矩阵 AAA 已经满足简化行阶梯矩阵的要求, 其中自由列 F=[111]F=\begin{bmatrix}1&1&1\end{bmatrix}F=[1​1​1​], I′I'I′ 为一个 3×33\times 33×3 的单位矩阵, 有

                                                    N=N′=[−FI′]=[−1−1−1100010001] N=N'= \begin{bmatrix} -F\\ I'\end{bmatrix}= \begin{bmatrix} -1&-1&-1\\ 1&0&0\\ 0&1&0\\ 0&0&1\end{bmatrix}N=N′=[−FI′​]=​−1100​−1010​−1001​​

                                                    零空间矩阵中的各列即向量空间 SSS 的基

                                                    # 问题扩展

                                                    除了 AAA 的零空间, 还可尝试求出 AAA 的剩余三个基本子空间的维数与基

                                                    • 列空间
                                                      • 维数
                                                        列空间的维数即 AAA 的秩 r=1=dim[C(A)]r=1=dim[C(A)]r=1=dim[C(A)]
                                                      • 基
                                                        由于 A=RA=RA=R 因此取 AAA 中的第一列元素 111 即 C(A)C(A)C(A) 的基
                                                    • 左零空间
                                                      • 维数
                                                        左零空间的维数即 m−r=0=dim[N(AT)]m-r=0=dim[N(A^T)]m−r=0=dim[N(AT)]
                                                      • 基
                                                        由于 AAA 仅有一行, 因此除了数乘 000, 方程 yTA=0y^TA=0yTA=0 不存在任何解
                                                        因此 N(AT)=∅N(A^T)=\varnothingN(AT)=∅ 为空集, 不存在任何基
                                                    • 行空间
                                                      • 维数
                                                        行空间的维数即 r=1=dim[C(AT)]r=1=dim[C(A^T)]r=1=dim[C(AT)]
                                                      • 基
                                                        由于 AAA 只有一行, 因此行空间的基即视为向量的 AAA

                                                    # 矩阵空间

                                                    将向量空间的定义推广至矩阵可得

                                                    矩阵空间是一个由无数个或几个矩阵组成的集合

                                                    在这个集合内, 矩阵需要对线性组合运算封闭 (数乘与加法), 即空间内的任意两个矩阵的线性组合仍在此空间内 (不包含矩阵相乘)

                                                    由此可得, 所有向量空间必须包含 000 矩阵

                                                    # 典型矩阵空间

                                                    定义集合 MMM 包含了所有 3×33\times 33×3 的矩阵, 显然 MMM 为一个矩阵空间

                                                    根据矩阵空间的要求还可已找到如下 MMM 的子空间

                                                    • 包含了所有 3×33\times 33×3 对称矩阵的集合 SSS
                                                    • 包含了所有 3×33\times 33×3 上三角矩阵的集合 UUU
                                                    • 包含了所有 3×33\times 33×3 对角矩阵的集合 DDD

                                                    显然对于一个包含了所有 3×33\times 33×3 且秩 r<2r<2r<2 的矩阵的集合不是矩阵空间, 因为两个秩 r=2r=2r=2 的矩阵相加可能得到秩 r=3r=3r=3 的矩阵

                                                    # 矩阵空间的维数与基

                                                    显然矩阵空间 MMM 的一组基即九个不同位置为 111 其余位置为 000 的矩阵
                                                    因此 MMM 的维数为 999

                                                    [100000000],[010000000],…,[000000001] \begin{bmatrix}1&0&0\\ 0&0&0\\ 0&0&0\end{bmatrix}, \begin{bmatrix}0&1&0\\ 0&0&0\\ 0&0&0\end{bmatrix},\dots, \begin{bmatrix}0&0&0\\ 0&0&0\\ 0&0&1\end{bmatrix}​100​000​000​​,​000​100​000​​,…,​000​000​001​​

                                                    矩阵空间 SSS 的一组基为三个对角上不同位置为 111 其余位置为 000 以及三个对称位置为 111 其余位置为 000 的矩阵
                                                    因此 SSS 的维数为 666

                                                    矩阵空间 UUU 的一组基为六个上三角上不同位置为 111 其余位置为 000 的矩阵
                                                    因此 UUU 的维数为 666

                                                    矩阵空间 DDD 的一组基为三个对角上不同位置为 111 其余位置为 000 的矩阵
                                                    因此 DDD 的维数为 333

                                                    # 矩阵空间的集合运算规律

                                                    对于对称矩阵空间 SSS 与上三角矩阵 UUU 的交集依然为矩阵空间, 并且易得该矩阵空间即对角矩阵空间 DDD

                                                    S∩U=D S\cap U=D S∩U=D

                                                    虽然对称矩阵空间 SSS 与上三角矩阵 UUU 的并集不是矩阵空间, 但可定义一个集合, 其中的元素来自对称矩阵空间 SSS 与上三角空间 UUU 中任意两个矩阵之和, 记为 +++
                                                    易得该集合也是矩阵空间, 且即 MMM

                                                    S+U=M S+U=M S+U=M

                                                    不难发现

                                                    dim(S)+dim(U)=dim(D)+dim(M) dim(S)+dim(U)=dim(D)+dim(M) dim(S)+dim(U)=dim(D)+dim(M)

                                                    推广有, 对于任意两个任意空间 A,BA,BA,B 满足 (证明略)

                                                    dim(A)+dim(B)=dim(A∩B)+dim(A+B) dim(A)+dim(B)=dim(A\cap B)+dim(A + B) dim(A)+dim(B)=dim(A∩B)+dim(A+B)

                                                    # 秩一矩阵

                                                    当一个矩阵的秩为 111 时, 表明其仅有一个主元列 / 行, 矩阵中的其它列 / 行均为主元列 / 行的倍数

                                                    # 秩一矩阵的分解

                                                    由其性质可得, 对于秩一矩阵 AAA, 可将其拆分为两个向量 u⃗,v⃗\vec{u},\vec{v}u,v 的乘积

                                                    A=u⃗⋅v⃗T A=\vec{u}\cdot\vec{v}^T A=u⋅vT

                                                    根据矩阵之和的秩的性质, 可将任意矩阵分解为若干个秩一矩阵, 再将这些秩一矩阵分解为两向量之积

                                                    # 图和网络

                                                    规定图或网络, 为一系列的点与边的集合
                                                    在这个集合中, 各个点之间至少存在一条通路 (不考虑边的方向), 不存在孤立的几个点或网络

                                                    # 图与矩阵的物理含义

                                                    通常可以使用一张图来表示一个电路, 液压, 导热等网络
                                                    图上的每一个结点都对应了一个势能点, 图的边则反映了两条结点之间的关系, 边的方向则对应着人为规定的参考方向

                                                    在这些图中, 结点的值通常体现为势能, 如电势能
                                                    而边的值则体现为两个势能点之间的流动量, 如电流

                                                    图在使用矩阵表示时, 将各个结点的势能表示为向量 x⃗\vec{x}x, 各个边的流动量表示为向量 b⃗\vec{b}b, 因此对于如下表达式

                                                    Ax⃗=b⃗ A\vec{x}=\vec{b} Ax=b

                                                    矩阵 AAA 中各个元素实际上仅反映了每条边上的流动量与边上两点的势能之间的方向关系
                                                    因此将表示图的矩阵 AAA 称为关联矩阵

                                                    # 左零空间

                                                    在左零空间中, 包含了线性方程 ATy⃗=0⃗A^T\vec{y}=\vec{0}ATy​=0 的所有解 y⃗\vec{y}y​

                                                    # 左零空间的物理意义

                                                    注意到, 由于电路网络满足基尔霍夫电流定律, 即各个结点的净流入电流为 000
                                                    该定律下, 电路网络中各边的电流需要满足的条件与左零空间中网络特点相同
                                                    因此基尔霍夫电流定律的另一表述即电路网络中各边的电流组成的向量应来自该网络关联矩阵的左零空间

                                                    # 行空间

                                                    由左零空间的特点可得, 当几条边构成一个回路时, 他们将线性相关 (存在得到 0⃗\vec{0}0 的组合)
                                                    因此任意一个回路的边对应 ATA^TAT 的几个列不可能是主元列

                                                    规定几条边组合而成的不封闭但相互连接的图为树
                                                    因此, 对于图中边最多的子树, 该树的各边对应 ATA^TAT 中的列即主元列, 也是关联矩阵行空间的一组基

                                                    此外, 根据关联矩阵的秩可得, 行空间的维数为 r=n−1r=n-1r=n−1, 因此一个图中边数最多的子树有 n−1n-1n−1 条边

                                                    # 应用数学的平衡方程

                                                    根据关联矩阵, 有应用数学的平衡方程

                                                    ATCAu⃗=f⃗ A^TCA\vec{u}=\vec{f} ATCAu=f​

                                                    其中

                                                    • AAA 为 m×nm\times nm×n 的关联矩阵
                                                    • u⃗\vec{u}u 为 nnn 元素的列向量, 体现了网络中结点的势能
                                                    • f⃗\vec{f}f​ 为 nnn 元素的行向量, 体现了网络中结点的流入流量

                                                    该方程的各个部分具有以下含义

                                                    # 势能差

                                                    Au⃗=e⃗ A\vec{u}=\vec{e} Au=e

                                                    由于关联矩阵 AAA 各行通过 111 与 −1-1−1 的取值表示各条边的参考方向
                                                    Au⃗A\vec{u}Au 的实质即计算出各条边在参考方向上两侧的势差

                                                    对于电路网络即各条边的电势差

                                                    # 电流量

                                                    Ce⃗=i C\vec{e}={i} Ce=i

                                                    然而势差与流量之间往往存在着阻力, 因此还需要矩阵 CCC 将势差转化为流量

                                                    对于电路网络, 通常 CCC 为一个对角矩阵, 对角线上的元素即对应边的电导率

                                                    # 电流源

                                                    ATi⃗=f⃗ A^T\vec{i}=\vec{f} ATi=f​

                                                    关联矩阵的转置 ATA^TAT 则反映了结点上的流量流入与流出情况
                                                    ATi⃗A^T\vec{i}ATi 的实质即各个节点上的净流入状况

                                                    对于电路网络, 由于电源的存在, 结点的电流净流入不一定为 000, 因此还需要流量 fff
