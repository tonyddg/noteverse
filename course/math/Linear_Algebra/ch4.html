                                          # 正交性的定义 Orthogonal

                                          # 向量的正交性

                                          对于向量 u⃗,v⃗\vec{u},\vec{v}u,v, 当两个向量满足

                                          u⃗T⋅v⃗=0⃗ \vec{u}^T\cdot\vec{v}=\vec{0} uT⋅v=0

                                          则称向量 u⃗\vec{u}u 与 v⃗\vec{v}v 正交

                                          显然

                                          • 零向量 0⃗\vec{0}0 与任何其他向量正交
                                          • 向量 u⃗,v⃗\vec{u},\vec{v}u,v 均来自向量空间 RnR^nRn

                                          # 向量空间的正交性

                                          有向量空间 S,TS,TS,T, 其均为 RnR^nRn 的子空间
                                          当任意一个来自 SSS 的向量与任意一个来自 TTT 的向量都满足正交性时
                                          则称向量空间 SSS 与 TTT 正交

                                          显然

                                          • 当向量空间 S,TS,TS,T 正交时 S∩T=0⃗S\cap T=\vec{0}S∩T=0

                                          # 行空间与零空间正交性证明

                                          对于 m×nm\times nm×n 的矩阵 AAA, 其行空间 C(AT)C(A^T)C(AT) 与零空间 N(A)N(A)N(A) 均为向量空间 RnR^nRn 的子空间

                                          根据零空间的定义可得, 对于任意一个来自零空间 N(A)N(A)N(A) 的向量 x⃗\vec{x}x 有

                                          Ax⃗=0⃗[r⃗1Tr⃗2T⋮r⃗mT]x⃗=[00⋮0] \begin{split} A\vec{x}&=\vec{0}\\ \begin{bmatrix} \vec{r}_1^T\\ \vec{r}_2^T\\ \vdots\\ \vec{r}_m^T\end{bmatrix} \vec{x}&=\begin{bmatrix} 0\\0\\\vdots\\0 \end{bmatrix} \end{split}Ax​r1T​r2T​⋮rmT​​​x​=0=​00⋮0​​​

                                          将 AAA 视为一个由行向量 r⃗iT\vec{r}_i^TriT​ 组成的向量, x⃗\vec{x}x 视为一个 n×1n\times 1n×1 的矩阵, 则该等式表明, AAA 中的每一行均与 x⃗\vec{x}x 正交

                                          由于行空间 C(AT)C(A^T)C(AT) 的一组基来自 AAA 的主元行, 因此上式也表明 x⃗\vec{x}x 与行空间的一组基 a⃗k\vec{a}_kak​ 正交, 有 x⃗Ta⃗k=0⃗\vec{x}^T\vec{a}_k=\vec{0}xTak​=0

                                          因此对于来自行空间的任意向量 a⃗\vec{a}a 有

                                          x⃗Ta⃗=x⃗T(c1a⃗1+c2a⃗2+⋯+cra⃗r)=0⃗ \vec{x}^T\vec{a}=\vec{x}^T(c_1\vec{a}_1+c_2\vec{a}_2+\dots +c_r\vec{a}_r)=\vec{0} xTa=xT(c1​a1​+c2​a2​+⋯+cr​ar​)=0

                                          因此对于来自同一矩阵 AAA 的行空间与零空间具有正交性
                                          同理可得, 来自同一矩阵 AAA 的列空间与左零空间具有正交性

                                          # 正交补

                                          注意到, 行空间的维数 dim[C(AT)]=rdim[C(A^T)]=rdim[C(AT)]=r, 零空间的维数 dim[N(A)]=n−rdim[N(A)]=n-rdim[N(A)]=n−r, 这两个空间均为 RnR^nRn 的子空间

                                          由于行空间与零空间正交, 因此行空间与零空间分别完全占据了 RnR^nRn 的两个维度
                                          零空间不仅仅与行空间正交, 而且零空间是与行空间正交的, 维数最大的向量空间
                                          以此定义, 零空间为行空间的正交补

                                          同理, 左零空间为列空间的正交补

                                          由此可得, 对于任意向量 x⃗∈Rn\vec{x}\in R^nx∈Rn, 必定可以分解为 x⃗=a⃗+b⃗\vec{x}=\vec{a}+\vec{b}x=a+b, 其中 a⃗∈C(AT),b⃗∈N(A)\vec{a}\in C(A^T),\vec{b}\in N(A)a∈C(AT),b∈N(A)
                                          x⃗∈Rm\vec{x}\in R^mx∈Rm 同理

                                          2. 将向量 a⃗\vec{a}a 视为矩阵, 则可以知道, 投影矩阵为一个对称矩阵, 有 P=PTP=P^TP=PT

                                          # 一般投影

                                          在 RnR^nRn 空间中有
                                          向量 b⃗\vec{b}b, 在 RnR^nRn 空间中体现为一条直线
                                          向量空间 C(A)C(A)C(A) 在 RnR^nRn 空间中体现为一个多维平面

                                          在 C(A)C(A)C(A) 中一定有一个向量 p⃗=Ax⃗\vec{p}=A\vec{x}p​=Ax, 使 b⃗\vec{b}b 与 p⃗\vec{p}p​ 距离最近
                                          以此定义 b⃗\vec{b}b 在向量空间 C(A)C(A)C(A) 上的投影为 p⃗\vec{p}p​
                                          (注意向量 x⃗\vec{x}x 为线性组合得到 p⃗\vec{p}p​ 的系数, 因此 x⃗\vec{x}x 反映了 b⃗\vec{b}b 投影到 C(A)C(A)C(A) 各个方向的分量)

                                          显然误差向量 e⃗=b⃗−p⃗\vec{e}=\vec{b}-\vec{p}e=b−p​ 与向量空间 C(A)C(A)C(A) 中的任意向量正交 (直线 BPBPBP 垂直于多维平面 C(A)C(A)C(A))

                                          根据左零空间 N(AT)N(A^T)N(AT) 的性质可得 e⃗\vec{e}e 满足

                                          # 一般投影矩阵

                                          根据高维投影向量 p⃗\vec{p}p​ 的推导可得, 一般情况下的投影矩阵满足

                                          P=A(ATA)−1AT P=A(A^TA)^{-1}A^T P=A(ATA)−1AT

                                          对于一般投影矩阵可以得到以下性质

                                          1. 由于 AAA 不一定是可逆矩阵, 因此其中的 (ATA)−1(A^TA)^{-1}(ATA)−1 不能消去
                                            但当 AAA 为可逆矩阵时, 有 P=IP=IP=I, 表明 b⃗\vec{b}b 在 RnR^nRn 的投影即 b⃗\vec{b}b 自身
                                          2. 投影矩阵 PPP 为对称矩阵
                                          3. 投影矩阵 PPP 满足 P2=PP^2=PP2=P

                                          注意

                                          • 矩阵 AAA 的各列必须线性无关, 即矩阵应当列满秩 m=rm=rm=r
                                          • 当 AAA 为可逆矩阵时, 必定有 P=IP=IP=I (证明略)
                                          • 当 AAA 为不可逆矩阵, 则应取其中的主元列计算 PPP, 而不能直接使用公式

                                          • 假设 b⃗\vec{b}b 垂直于空间 C(A)C(A)C(A) (b⃗\vec{b}b 与 C(A)C(A)C(A) 中的每个向量正交), 因此 b⃗∈N(AT)\vec{b}\in N(A^T)b∈N(AT), ATb⃗=0⃗A^T\vec{b}=\vec{0}ATb=0

                                          对于一系列属于 RmR^mRm 空间的向量 {q⃗}\{\vec{q}\}{q​} 满足

                                          {q⃗iT⋅q⃗j=0,i≠jq⃗iT⋅q⃗j=1,i=j \begin{cases} \vec{q}_i^T\cdot\vec{q}_j=0,&i\neq j\\ \vec{q}_i^T\cdot\vec{q}_j=1,&i=j \end{cases}{q​iT​⋅q​j​=0,q​iT​⋅q​j​=1,​i=ji=j​

                                          第一行的条件表明, q⃗i\vec{q}_iq​i​ 与组内的其他向量正交
                                          第二行的条件表明, q⃗i\vec{q}_iq​i​ 的长度为 111
                                          则称 {q⃗}\{\vec{q}\}{q​} 为一组标准正交基

                                          # 标准正交矩阵

                                          以标准正交基为列, 可得到一个 m×nm\times nm×n 的矩阵 Q=[q⃗1q⃗2…q⃗n]Q=\begin{bmatrix}\vec{q}_1&\vec{q}_2&\dots&\vec{q}_n\end{bmatrix}Q=[q​1​​q​2​​…​q​n​​]
