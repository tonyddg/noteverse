                                            # 特征值基础

                                            # 特征值与特征向量的定义

                                            特征值依然以方阵为讨论对象
                                            对于一个 n×nn\times nn×n 的矩阵 AAA, 存在某些方向的向量 x⃗\vec{x}x 在右乘 AAA 后, 得到的结果仍然平行于 x⃗\vec{x}x, 将具有这种特性的向量称为特征向量 x⃗\vec{x}x, 而称相乘结果 Ax⃗A\vec{x}Ax 与原向量 x⃗\vec{x}x 之比 λ\lambdaλ 为特征值, 有

                                            Ax⃗=λx⃗ A\vec{x}=\lambda\vec{x} Ax=λx

                                            对于一个矩阵 AAA 的特征向量 x⃗\vec{x}x, 与其同一方向上所有向量均能作为特征向量, 一般仅使用其中一个非零向量代表这个方向

                                            # 特征值与特征向量的计算

                                            在特征值与特征向量的定义式中, 存在着两个未知变量, 为此需要进行如下变形

                                            Ax⃗=λx⃗(A−λI)x⃗=0⃗ \begin{split} A\vec{x}&=\lambda\vec{x}\\ (A-\lambda I)\vec{x}&=\vec{0} \end{split}Ax(A−λI)x​=λx=0​

                                            此时, 仅当矩阵 A−λIA-\lambda IA−λI 为不可逆矩阵, 方程 (A−λI)x⃗=0⃗(A-\lambda I)\vec{x}=\vec{0}(A−λI)x=0 才会存在非零解
                                            而要当矩阵 A−λIA-\lambda IA−λI 为不可逆矩阵时, 根据行列式的性质, 有

                                            det⁡(A−λI)=0 \det(A-\lambda I)=0 det(A−λI)=0

                                            上式可展开为一个关于 λ\lambdaλ 的 nnn 次方程, 称为特征方程, 因此对于一个 n×nn\times nn×n 方阵, 其共有 nnn 个特征值
                                            并且其中可能存在多重根与复根, 因此也存在多重特征值与复数特征值的情况

                                            得到特征值后可得, 对于任意一个特征值 λ\lambdaλ, 其有特征向量 x⃗\vec{x}x 有无数多个, 并且这些特征值均来自矩阵 A−λIA-\lambda IA−λI 的零空间
                                            通常在计算出特征值后, 选择 N(A−λI)N(A-\lambda I)N(A−λI) 的一组基作为该特征的特定特征向量
                                            注意, N(A−λI)N(A-\lambda I)N(A−λI) 的维数与 λ\lambdaλ 的重数无关, 因此有 nnn 个特征值不代表有 nnn 个不平行的特征向量 x⃗\vec{x}x

                                            # 特征值的特性

                                            • 将方阵 AAA 对角线上的元素之和称为迹 tr(A)tr(A)tr(A)
                                              • 矩阵 AAA 的迹等于其特征值之和, 可用该性质寻找最后一个特征值
                                              • 矩阵的迹的相反数等于其特征方程中的一次项系数
                                            • 对于方阵 AAA 的行列式
                                              • 矩阵 AAA 的行列式等于其特征值之积
                                              • 矩阵的行列式也等于其特征方程中的常数项
                                            • 对于矩阵 AAA 有特征值 λ\lambdaλ, 则矩阵数乘 kAkAkA 有特征值 kλk\lambdakλ
                                            • 对于矩阵 AAA 有特征值 λ\lambdaλ, 则矩阵与单位矩阵运算 A+kIA+kIA+kI 有特征值 λ+k\lambda+kλ+k, 且对应特征值的特征向量相同, 证明如下 (注意对于任意两个矩阵 A,BA,BA,B, 其和与积的特征值不存在特殊性质)

                                            (A+kI)x⃗=λx⃗+kx⃗=(λ+k)x⃗ (A+kI)\vec{x}=\lambda\vec{x}+k\vec{x}=(\lambda+k)\vec{x} (A+kI)x=λx+kx=(λ+k)x

                                            • 矩阵 AAA 以及其转置 ATA^TAT 具有相同的特征值, 该性质由行列式的推论七可得到
                                            • 矩阵 AAA 的逆 A−1A^{-1}A−1 的特征值为矩阵 AAA 特征值 λi\lambda_iλi​ 的倒数 1λi\frac{1}{\lambda_i}λi​1​
                                            • 对于矩阵 A,BA,BA,B, 其乘积 ABABAB 与 BABABA 具有相同的特征值, 证明有 (AB)x⃗=λx⃗→(BA)(Bx⃗)=λ(Bx⃗)(AB)\vec{x}=\lambda\vec{x}\to(BA)(B\vec{x})=\lambda(B\vec{x})(AB)x=λx→(BA)(Bx)=λ(Bx), 更多关于矩阵相乘与特征值的关系间参考资料open in new window

                                            # 投影矩阵

                                            对于投影矩阵 PPP, 根据投影矩阵的性质

                                            • 对于 x⃗∈C(P)\vec{x}\in C(P)x∈C(P), 此时有 Px⃗=x⃗P\vec{x}=\vec{x}Px=x, 因此认为投影矩阵列空间 C(P)C(P)C(P) 中的任意向量均为 PPP 的特征向量, 且有特征值 λ=1\lambda=1λ=1
                                            • 对于 x⃗⊥C(P)\vec{x}\perp C(P)x⊥C(P) (与 C(P)C(P)C(P) 中的所有向量正交), 此时有 Px⃗=0⃗P\vec{x}=\vec{0}Px=0, 即类似不可逆矩阵 SSS 的情况, 对于这些向量 x⃗∈N(P)\vec{x}\in N(P)x∈N(P), 也是 PPP 的特征向量, 有特征值 λ=0\lambda=0λ=0
                                            • 因此投影矩阵仅有特征值 111 与 000

                                            # 标准正交矩阵

                                            注意到对于任意向量 x⃗\vec{x}x 与标准正交矩阵 QQQ

                                            ∥Qx⃗∥=∑i=1n∣xi∣∥q⃗i∥=∥x⃗∥ \begin{Vmatrix}Q\vec{x}\end{Vmatrix}=\sum_{i=1}^n |x_i|\begin{Vmatrix}\vec{q}_i\end{Vmatrix}=\begin{Vmatrix}\vec{x}\end{Vmatrix} ​Qx​​=i=1∑n​∣xi​∣​q​i​​​=​x​​

                                            因此标准正交矩阵不会改变向量的长度, 在求特征值时有 ∥Qx⃗∥=∣λ∣∥x⃗∥→∥λ∥=1\begin{Vmatrix}Q\vec{x}\end{Vmatrix}=|\lambda|\begin{Vmatrix}\vec{x}\end{Vmatrix}\to \begin{Vmatrix}\lambda\end{Vmatrix}=1​Qx​​=∣λ∣​x​​→​λ​​=1

                                            正交矩阵的特征值 λ\lambdaλ 也可能为复数, 但绝对值必定为 111

                                            # 对称矩阵

                                            对于对称矩阵, 其特征值全为实数, 且不同特征值的特征向量相互垂直

                                            例如对称矩阵 S=[3113]S=\begin{bmatrix}3&1\\1&3\end{bmatrix}S=[31​13​], 其有特征方程

                                            ∣3−λ113−λ∣=0(3−λ)2−1=0λ2−6λ+8=0 \begin{split} \begin{vmatrix}3-\lambda&1\\1&3-\lambda\end{vmatrix}&=0\\ (3-\lambda)^2-1&=0\\ \lambda^2-6\lambda+8&=0 \end{split}​3−λ1​13−λ​​(3−λ)2−1λ2−6λ+8​=0=0=0​

                                            解得其有特征值

                                            • λ1=2\lambda_{1}=2λ1​=2, 对应特征向量 x⃗1=[1−1]T\vec{x}_1=\begin{bmatrix}1&-1\end{bmatrix}^Tx1​=[1​−1​]T
                                            • λ2=4\lambda_{2}=4λ2​=4, 对应特征向量 x⃗2=[11]T\vec{x}_2=\begin{bmatrix}1&1\end{bmatrix}^Tx2​=[1​1​]T

                                            # 反对称矩阵

                                            对于反对称矩阵 (A=−ATA=-A^TA=−AT), 其特征值全为纯虚数

                                            例如对称矩阵 A=[0−110]A=\begin{bmatrix}0&-1\\1&0\end{bmatrix}A=[01​−10​] (同时该矩阵也是 90°90\degree90° 的旋转矩阵), 其有特征方程

                                            ∣−λ−11−λ∣=0λ2+1=0λ=±i \begin{split} \begin{vmatrix}-\lambda&-1\\1&-\lambda\end{vmatrix}&=0\\ \lambda^2+1&=0\\ \lambda&=\pm i \end{split}​−λ1​−1−λ​​λ2+1λ​=0=0=±i​

                                            对于一般矩阵, 其特征值介于二者之间, 可能为实数或复数, 且通常越对称, 实数越多

                                            # 上三角矩阵

                                            对于上三角矩阵, 其对角线上的值即其特征值 (根据行列式的性质易得)

                                            例如上三角矩阵 U=[3103]U=\begin{bmatrix}3&1\\0&3\end{bmatrix}U=[30​13​], 其有特征方程

                                            ∣3−λ103−λ∣=0(3−λ)2=0λ=3 \begin{split} \begin{vmatrix}3-\lambda&1\\0&3-\lambda\end{vmatrix}&=0\\ (3-\lambda)^2&=0\\ \lambda&=3 \end{split}​3−λ0​13−λ​​(3−λ)2λ​=0=0=3​

                                            解得该矩阵有特征根 λ=3\lambda=3λ=3, 且为二重特征根
                                            此外, 对于 U−3I=[0100]U-3I=\begin{bmatrix}0&1\\0&0\end{bmatrix}U−3I=[00​10​], 该矩阵的零空间维数 dim[N(U−3I)]=2−r=1dim[N(U-3I)]=2-r=1dim[N(U−3I)]=2−r=1, 因此零空间为一维
                                            这表明该矩阵仅有一个方向的特征向量 x⃗=[10]T\vec{x}=\begin{bmatrix}1&0\end{bmatrix}^Tx=[1​0​]T

                                            # 对角化分解

                                            对于一个 n×nn\times nn×n 的矩阵 AAA, 将其任意 nnn 个特征向量 x⃗k\vec{x}_kxk​ 组成特征矩阵 SSS
                                            根据特征值与特征向量的定义, 矩阵 AAA 与其特征矩阵 SSS 相乘时有

                                            AS=A[x⃗1x⃗2…x⃗n]=[λ1x⃗1λ2x⃗2…λnx⃗n]=[x⃗1x⃗2…x⃗n][λ10000λ200⋮⋮000λn]=SΛ \begin{split}AS&=A\begin{bmatrix}\vec{x}_1&\vec{x}_2&\dots&\vec{x}_n\end{bmatrix}\\ &=\begin{bmatrix}\lambda_1\vec{x}_1&\lambda_2\vec{x}_2&\dots&\lambda_n\vec{x}_n\end{bmatrix}\\ &=\begin{bmatrix}\vec{x}_1&\vec{x}_2&\dots&\vec{x}_n\end{bmatrix}\begin{bmatrix}\lambda_1&0&0&0\\0&\lambda_2&0&0\\\vdots&&&\vdots\\0&0&0&\lambda_n\end{bmatrix}\\ &=S\Lambda \end{split}AS​=A[x1​​x2​​…​xn​​]=[λ1​x1​​λ2​x2​​…​λn​xn​​]=[x1​​x2​​…​xn​​]​λ1​0⋮0​0λ2​0​000​00⋮λn​​​=SΛ​

                                            其中矩阵 Λ\LambdaΛ 为一个以 AAA 的特征值组成的对角矩阵

                                            当这 nnn 个特征向量不平行且非 000 时, 矩阵 SSS 可逆, 因此有对角化分解

                                            A=SΛS−1 A=S\Lambda S^{-1} A=SΛS−1

                                            # 对角化矩阵的条件

                                            易得, 当一个矩阵具有 nnn 个不同的特征值 λ\lambdaλ 时, 各个特征值都将对应一个方向的特征向量, 且这些特征向量互相平行, 因此该矩阵能够对角化

                                            但是当一个矩阵的特征值中有重根时, 其重根不一定对应与根的重数相同的, 互不平行的特征向量, 因此有可能无法对角化, 如前文的例子 (注意没有重根仅是可对角化的充分条件, 存在重根时仍有可能可以对角化)

                                            但在一般情况下, 对于绝大部分的矩阵均能对角化, 对于不可对角化的矩阵可参见若尔当定理

                                            # 矩阵的幂

                                            当矩阵满足对角化条件并对角化后, 对于矩阵的 kkk 次幂有

                                            Ak=SΛkS−1 A^k=S\Lambda^k S^{-1} Ak=SΛkS−1

                                            根据对角矩阵的特性, 对于矩阵的任意次幂, 其特征向量不变, 特征值变为原矩阵的 kkk 次 λ′=λk\lambda'=\lambda^kλ′=λk

                                            当矩阵所有特征值满足 ∥λ∥<1\begin{Vmatrix}\lambda\end{Vmatrix}<1​λ​​<1, 有 lim⁡k→∞Ak=0\lim_{k\to\infty}A^k=0limk→∞​Ak=0

                                            # 连续微分状态方程

                                            同样以向量 u⃗\vec{u}u 表示系统特定状态下的一组参数
                                            此时系统关于时间连续, 系统各个参数的变化量满足线性微分方程组

                                            # 马尔可夫矩阵

                                            马尔科夫矩阵是离散差分状态方程中一类特殊的系数矩阵

                                            定义当 n×nn\times nn×n 的方阵 AAA 满足以下条件时, 称之为马尔可夫矩阵

                                            • 矩阵的各列元素之和等于 111
                                            • 矩阵的各个元素大于等于 000

                                            马尔可夫矩阵的特征值将具有以下特性

                                            1. 马尔可夫矩阵的任意特征值满足 ∥λi∥≥0\begin{Vmatrix}\lambda_i\end{Vmatrix}\ge 0​λi​​​≥0
                                            2. 马尔可夫矩阵存在至少一个特征值满足 λk=1\lambda_k=1λk​=1
                                            3. 马尔科夫矩阵的任意次幂依然是马尔科夫矩阵

                                            由马尔可夫矩阵的特性与离散差分状态方程稳态与 λ\lambdaλ 的关系可得, 当系统的系数矩阵为马尔科夫矩阵时, 当 kkk 足够大, 系统参数将仅与 λi=1\lambda_i=1λi​=1 对应的特征向量有关
                                            并且马尔科夫矩阵中, 状态参数之和始终不变

                                            现就特性二给出证明
                                            由于矩阵各列元素之和为 111, 因此矩阵 A−IA-IA−I 的各列元素之和为 000
                                            这表明矩阵行向量之和为零, 即左零空间 N[(A−I)T]N[(A-I)^T]N[(A−I)T] 中存在向量 [11…1]\begin{bmatrix}1&1&\dots&1\end{bmatrix}[1​1​…​1​]
                                            因此矩阵的秩 r<nr<nr<n, 不可逆, 有 det⁡(A−I)=0\det(A-I)=0det(A−I)=0
