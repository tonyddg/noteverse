                                # 代数精度

                                当求积公式对次数不超过 mmm 的多项式能够准确成立, 但对 m+1m+1m+1 次多项式不成立, 则称求积公式具有 mmm 次代数精度

                                # 确定代数精度/积分公式的系数

                                对于

                                I=∫abf(x)dx I=\int_a^bf(x)dx I=∫ab​f(x)dx

                                可以令 f(x)=xif(x)=x^if(x)=xi, 从 000 开始不断提高 iii, 直到公式不成立, 注意积分上下限 a,ba,ba,b

                                # 牛顿-柯斯特公式

                                即等距插值公式的积分形式, nnn 阶公式至少具有 nnn 次代数精度, 其中偶数阶的公式具有 n+1n+1n+1 次代数精度 (nnn 阶对应 nnn 次插值, 等距离的取区间内的 n+1n+1n+1 个点)

                                # 梯形公式

                                取 n=1n=1n=1, 具有 n=1n=1n=1 次代数精度 将积分区间的两个端点相连, 计算围成的梯形的面积, 近似为积分结果 (积分属于计算面积, 因此 b−ab-ab−a 项不能忘)

                                ∫abf(x)dx≈b−a2[f(a)+f(b)] \int_a^bf(x)dx\approx\frac{b-a}{2}[f(a)+f(b)] ∫ab​f(x)dx≈2b−a​[f(a)+f(b)]

                                # 辛普森公式

                                取 n=2n=2n=2, 具有 n+1=3n+1=3n+1=3 次代数精度

                                ∫abf(x)dx≈b−a6[f(a)+4f(a+b2)+f(b)] \int_a^bf(x)dx\approx\frac{b-a}{6}[f(a)+4f(\frac{a+b}{2})+f(b)] ∫ab​f(x)dx≈6b−a​[f(a)+4f(2a+b​)+f(b)]

                                当 n≥8n\ge8n≥8 时, 公式出现负数系数, 不具有稳定性, 可分段使用低阶公式
